## ShapeBase class

الفئة الأساسية للكائنات الموجودة في طبقة الرسم، مثل الشكل التلقائي أو الشكل الحر أو كائن OLE أو عنصر تحكم ActiveX أو الصورة.

لمعرفة المزيد، قم بزيارة[العمل مع الأشكال](https://docs.aspose.com/words/net/working-with-shapes/) مقالة توثيقية.

```csharp
public abstract class ShapeBase : CompositeNode
```

### ملاحظات

هذه فئة مجردة. الفئتان المشتقتان اللتان يمكنك إنشاء مثيل لهما هما[`Shape`](../shape/) و[`GroupShape`](../groupshape/).

الشكل عبارة عن عقدة في شجرة المستندات.

إذا كان الشكل طفلا[`Paragraph`](../../aspose.words/paragraph/) كائن، ثم يقال أن الشكل هو "المستوى الأعلى". يتم قياس الأشكال ذات المستوى الأعلى ووضعها في نقاط.

يمكن أن يحدث الشكل أيضًا عندما يكون طفلاً لـ[`GroupShape`](../groupshape/) كائن عند تجميع عدة أشكال . يتم وضع الأشكال الفرعية لشكل المجموعة في مساحة الإحداثيات ويتم تعريف الوحدات بواسطة[`CoordSize`](./coordsize/) و[`CoordOrigin`](./coordorigin/) خصائص شكل المجموعةparent .

يمكن وضع الشكل سطريًا مع النص أو عائمًا. يتم التحكم في طريقة تحديد المواقع باستخدام[`WrapType`](./wraptype/) ملكية.

عندما يكون الشكل عائمًا، يتم وضعه بالنسبة لشيء ما (على سبيل المثال، الفقرة الحالية، الهامش أو الصفحة). يتم تحديد الموضع النسبي للشكل باستخدام [`RelativeHorizontalPosition`](./relativehorizontalposition/) و[`RelativeVerticalPosition`](./relativeverticalposition/) ملكيات.

يتم وضع الشكل العائم بشكل واضح باستخدام[`Left`](./left/) و[`Top`](./top/) خصائص أو محاذاتها بالنسبة لبعض الكائنات الأخرى باستخدام[`HorizontalAlignment`](./horizontalalignment/) و[`VerticalAlignment`](./verticalalignment/) ملكيات.

### أمثلة

يوضح كيفية إدراج صورة عائمة في وسط الصفحة.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// أدخل صورة عائمة ستظهر خلف النص المتداخل وقم بمحاذاتها مع منتصف الصفحة.
Shape shape = builder.InsertImage(ImageDir + "Logo.jpg");
shape.WrapType = WrapType.None;
shape.BehindText = true;
shape.RelativeHorizontalPosition = RelativeHorizontalPosition.Page;
shape.RelativeVerticalPosition = RelativeVerticalPosition.Page;
shape.HorizontalAlignment = HorizontalAlignment.Center;
shape.VerticalAlignment = VerticalAlignment.Center;

doc.Save(ArtifactsDir + "Image.CreateFloatingPageCenter.docx");
```
