## LoadOptions.UseSystemLcid property

يحصل على أو يحدد ما إذا كان سيتم استخدام قيمة LCID التي تم الحصول عليها من سجل Windows لتحديد هوامش إعداد الصفحة الافتراضية.

```csharp
public bool UseSystemLcid { get; set; }
```

## ملاحظات

إذا تم ضبطه على`حقيقي`، ثم يتم محاكاة سلوك MS Word والذي يأخذ قيمة LCID من سجل Windows.

القيمة الافتراضية هي`خطأ شنيع`.
