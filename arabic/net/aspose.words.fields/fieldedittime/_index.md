## ملاحظات

يسترجع إجمالي وقت التحرير، بالدقائق، منذ إنشاء المستند.

## أمثلة

يوضح كيفية استخدام الحقل EDITTIME.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// سيظهر حقل EDITTIME بالدقائق،
//الوقت المستغرق مع فتح المستند في نافذة Microsoft Word.
builder.MoveToHeaderFooter(HeaderFooterType.HeaderPrimary);
builder.Write("You've been editing this document for ");
FieldEditTime field = (FieldEditTime)builder.InsertField(FieldType.FieldEditTime, true);
builder.Writeln(" minutes.");

// هذه الخاصية المُدمجة في المستند تُتبّع الدقائق. يستخدمها Microsoft Word.
// لتتبع الوقت المستغرق لفتح المستند. يمكننا أيضًا تعديله بأنفسنا.
doc.BuiltInDocumentProperties.TotalEditingTime = 10;
field.Update();

Assert.AreEqual(" EDITTIME ", field.GetFieldCode());
Assert.AreEqual("10", field.Result);

// لا يقوم الحقل بتحديث نفسه في الوقت الفعلي، وسيتعين عليه أيضًا
// يتم التحديث يدويًا في Microsoft Word في أي وقت نحتاج فيه إلى قيمة دقيقة.
doc.UpdateFields();
doc.Save(ArtifactsDir + "Field.EDITTIME.docx");
```
