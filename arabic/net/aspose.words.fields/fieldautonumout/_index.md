### ملاحظات

إدراج رقم تلقائي بتنسيق مخطط تفصيلي.

### أمثلة

يوضح كيفية ترقيم الفقرات باستخدام حقول AUTONUMOUT.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// تعرض حقول AUTONUMOUT رقمًا يتزايد في كل حقل AUTONUMOUT.
// على عكس حقول AUTONUM، تستخدم حقول AUTONUMOUT نظام ترقيم المخطط التفصيلي،
// والذي يمكننا تعريفه في Microsoft Word عبر التنسيق -> الرصاص وأمبير. الترقيم -> “المخطط التفصيلي مرقمة”.
// هذا يسمح لنا بترقيم العناصر تلقائيًا مثل قائمة مرقمة.
// حقول LISTNUM هي بديل أحدث لحقول AUTONUMOUT.
// سيعرض هذا الحقل "1.".
builder.InsertField(FieldType.FieldAutoNumOutline, true);
builder.Writeln("\tParagraph 1.");

// سيعرض هذا الحقل "2."
builder.InsertField(FieldType.FieldAutoNumOutline, true);
builder.Writeln("\tParagraph 2.");

foreach (FieldAutoNumOut field in doc.Range.Fields.Where(f => f.Type == FieldType.FieldAutoNumOutline))
    Assert.AreEqual(" AUTONUMOUT ", field.GetFieldCode());

doc.Save(ArtifactsDir + "Field.AUTONUMOUT.docx");
```
