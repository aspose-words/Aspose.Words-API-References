## ReportingEngine.GetRestrictedTypes method

Returnerar typer, vilka medlemmar samt vilka härledda typers medlemmar ska vara oåtkomliga för motorn. via mallsyntaxen.

```csharp
public static Type[] GetRestrictedTypes()
```

### Returvärde

Typer, vilka medlemmar samt vilka härledda typers medlemmar ska vara oåtkomliga för motorn via mallsyntaxen .

## Anmärkningar

Den returnerade arrayen innehåller objekt som tidigare angetts med[`SetRestrictedTypes`](../setrestrictedtypes/) .

Att ändra element i den returnerade arrayen har ingen effekt på begränsade typer. För att ändra begränsade typer, använd [`SetRestrictedTypes`](../setrestrictedtypes/) istället.
