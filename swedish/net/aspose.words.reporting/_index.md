De**Aspose.Words.Rapportering** Namnrymden innehåller klasser av Aspose.Words nya rapporteringsmotor som stöder rapportmallar markerade med ett språk baserat på LINQ-metodsyntaxen.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [CsvDataLoadOptions](./csvdataloadoptions/) | Representerar alternativ för att analysera CSV-data. |
| [CsvDataSource](./csvdatasource/) | Ger åtkomst till data från en CSV-fil eller ström som ska användas i en rapport. |
| [JsonDataLoadOptions](./jsondataloadoptions/) | Representerar alternativ för att analysera JSON-data. |
| [JsonDataSource](./jsondatasource/) | Ger åtkomst till data från en JSON-fil eller ström som ska användas i en rapport. |
| [KnownTypeSet](./knowntypeset/) | Representerar en oordnad mängd (dvs. en samling unika objekt) som innehållerType objekt vars helt eller delvis kvalificerade namn kan användas i rapportmallar för att anropa motsvarande typers statiska medlemmar, utföra typomvandlingar etc. |
| [ReportingEngine](./reportingengine/) | Tillhandahåller rutiner för att fylla malldokument med data och en uppsättning inställningar för att styra dessa rutiner. |
| [XmlDataLoadOptions](./xmldataloadoptions/) | Representerar alternativ för inläsning av XML-data. |
| [XmlDataSource](./xmldatasource/) | Ger åtkomst till data från en XML-fil eller ström som ska användas i en rapport. |
## Uppräkning

| Uppräkning | Beskrivning |
| --- | --- |
| [JsonSimpleValueParseMode](./jsonsimplevalueparsemode/) | Anger ett läge för att analysera enkla JSON-värden (null, booleska, tal, heltal och sträng) vid laddning av JSON. Ett sådant läge påverkar inte analysering av datum- och tidsvärden. |
| [ReportBuildOptions](./reportbuildoptions/) | Anger alternativ som styr beteendet för[`ReportingEngine`](../aspose.words.reporting/reportingengine/) medan man skapar en rapport. |
