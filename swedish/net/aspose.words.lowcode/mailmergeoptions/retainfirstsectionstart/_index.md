## MailMergeOptions.RetainFirstSectionStart property

Hämtar eller anger ett värde som anger om sektionsstarten för det första dokumentavsnittet och dess kopior för efterföljande datakällrader behålls under dokumentkoppling eller uppdateras enligt MS Words beteende.

```csharp
public bool RetainFirstSectionStart { get; set; }
```

## Anmärkningar

Standardvärdet är`sann` .
