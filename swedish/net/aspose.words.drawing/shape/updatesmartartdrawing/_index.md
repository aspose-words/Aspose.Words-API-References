## Shape.UpdateSmartArtDrawing method

Uppdaterar SmartArt förrenderad ritning genom att använda Aspose.Words SmartArt kall renderingsmotor.

```csharp
public void UpdateSmartArtDrawing()
```

### Anmärkningar

Microsoft Word genererar och sparar den förrenderade ritningen tillsammans med SmartArt-objektet. Men om dokumentet sparas av andra program kan den förrenderade SmartArt-ritningen saknas eller vara felaktig. Om förrenderad ritning är tillgänglig använder Aspose.Words den för att rendera SmartArt-objektet. Om förrenderad ritning saknas då Aspose.Words använder sin egen SmartArt kall renderingsmotor för att rendera SmartArt-objektet. Om den förrenderade ritningen är felaktig måste den här metoden anropas för att anropa SmartArt cold renderingsmotorn.
