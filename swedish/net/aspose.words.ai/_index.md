De**Aspose.Words.AI** Namnrymden möjliggör sömlös integration med stora språkmodeller (LLM), såsom OpenAI och Gemini, , vilket ger robust stöd för AI-driven dokumentbehandling.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [AiModel](./aimodel/) | Representerar information om en generativ språkmodell. |
| [AnthropicAiModel](./anthropicaimodel/) | En abstrakt klass som representerar integrationen med Anthropics AI-modeller inom Aspose.Words. |
| [CheckGrammarOptions](./checkgrammaroptions/) | Gör det möjligt att ange olika alternativ vid grammatikkontroll i ett dokument med hjälp av AI. |
| [GoogleAiModel](./googleaimodel/) | En abstrakt klass som representerar integrationen med Googles AI-modeller inom Aspose.Words. |
| [OpenAiModel](./openaimodel/) | En abstrakt klass som representerar integrationen med OpenAI:s stora språkmodeller inom Aspose.Words. |
| [SummarizeOptions](./summarizeoptions/) | Gör det möjligt att ange olika alternativ för att sammanfatta dokumentinnehåll. |
