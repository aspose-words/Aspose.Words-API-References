## Exempel

Visar hur man arbetar med fältet 'FieldNone' i ett dokument.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// Infoga ett fält som inte anger en objektivfälttyp i sin fältkod.
Field field = builder.InsertField(" NOTAREALFIELD //en");

// Fälttypen "FieldNone" är reserverad för fält som dessa.
Assert.AreEqual(FieldType.FieldNone, field.Type);

// Vi kan också fortfarande arbeta med dessa fält och tilldela dem som instanser av FieldUnknown-klassen.
FieldUnknown fieldUnknown = (FieldUnknown)field;
Assert.AreEqual(" NOTAREALFIELD //a", fältOkänd.HämtaFältkod());
```
