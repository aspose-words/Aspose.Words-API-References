De**Aspose.Words.Loading**namnområdet tillhandahåller klasser och uppräkningar som tillåter att ange ytterligare alternativ när dokument laddas.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [ChmLoadOptions](./chmloadoptions/) | Gör det möjligt att ange ytterligare alternativ när CHM-dokument laddas i en[`Document`](../aspose.words/document/) objekt. |
| [DocumentLoadingArgs](./documentloadingargs/) | Ett argument passerat in[`Notify`](../aspose.words.loading/idocumentloadingcallback/notify/) . |
| [HtmlLoadOptions](./htmlloadoptions/) | Gör det möjligt att ange ytterligare alternativ när HTML-dokument läses in i en[`Document`](../aspose.words/document/) objekt. |
| [LanguagePreferences](./languagepreferences/) | Gör det möjligt att ställa in språkinställningar. |
| [LoadOptions](./loadoptions/) | Gör det möjligt att ange ytterligare alternativ (som lösenord eller bas-URI) när laddar ett dokument i en[`Document`](../aspose.words/document/) objekt. |
| [PdfLoadOptions](./pdfloadoptions/) | Gör det möjligt att ange ytterligare alternativ när Pdf-dokument laddas in i en[`Document`](../aspose.words/document/) objekt. |
| [ResourceLoadingArgs](./resourceloadingargs/) | Tillhandahåller data för[`ResourceLoading`](../aspose.words.loading/iresourceloadingcallback/resourceloading/) metod. |
| [RtfLoadOptions](./rtfloadoptions/) | Tillåter att ange ytterligare alternativ vid laddningRtf dokument till en[`Document`](../aspose.words/document/) objekt. |
| [TxtLoadOptions](./txtloadoptions/) | Tillåter att ange ytterligare alternativ vid laddningText dokument till en[`Document`](../aspose.words/document/) objekt. |
## Gränssnitt

| Gränssnitt | Beskrivning |
| --- | --- |
| [IDocumentLoadingCallback](./idocumentloadingcallback/) | Implementera det här gränssnittet om du vill att din egen anpassade metod ska anropas när ett dokument laddas. |
