De**Aspose.Words.Laddar** namnrymden tillhandahåller klasser och uppräkningar som tillåter att ange ytterligare alternativ vid laddning av dokument.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [ChmLoadOptions](./chmloadoptions/) | Gör det möjligt att ange ytterligare alternativ när CHM-dokument laddas till en[`Document`](../aspose.words/document/) objekt. |
| [DocumentLoadingArgs](./documentloadingargs/) | Ett argument skickades in[`Notify`](../aspose.words.loading/idocumentloadingcallback/notify/) . |
| [HtmlLoadOptions](./htmlloadoptions/) | Gör det möjligt att ange ytterligare alternativ när HTML-dokument laddas in i en[`Document`](../aspose.words/document/) objekt. |
| [LanguagePreferences](./languagepreferences/) | Gör det möjligt att ställa in språkinställningar. |
| [LoadOptions](./loadoptions/) | Gör det möjligt att ange ytterligare alternativ (som lösenord eller bas-URI) när ett dokument laddas in i en[`Document`](../aspose.words/document/) objekt. |
| [MarkdownLoadOptions](./markdownloadoptions/) | Gör det möjligt att ange ytterligare alternativ vid laddningMarkdown dokument till ett[`Document`](../aspose.words/document/) objekt. |
| [PdfLoadOptions](./pdfloadoptions/) | Gör det möjligt att ange ytterligare alternativ när PDF-dokument laddas till en[`Document`](../aspose.words/document/) objekt. |
| [ResourceLoadingArgs](./resourceloadingargs/) | Tillhandahåller data för[`ResourceLoading`](../aspose.words.loading/iresourceloadingcallback/resourceloading/) metod. |
| [RtfLoadOptions](./rtfloadoptions/) | Gör det möjligt att ange ytterligare alternativ vid laddningRtf dokument till ett[`Document`](../aspose.words/document/) objekt. |
| [TxtLoadOptions](./txtloadoptions/) | Gör det möjligt att ange ytterligare alternativ vid laddningText dokument till ett[`Document`](../aspose.words/document/) objekt. |
## Gränssnitt

| Gränssnitt | Beskrivning |
| --- | --- |
| [IDocumentLoadingCallback](./idocumentloadingcallback/) | Implementera detta gränssnitt om du vill att din egen anpassade metod ska anropas när ett dokument laddas. |
| [IResourceLoadingCallback](./iresourceloadingcallback/) | Implementera detta gränssnitt om du vill styra hur Aspose.Words laddar externa resurser när importerar ett dokument och infogar bilder med hjälp av[`DocumentBuilder`](../aspose.words/documentbuilder/) . |
