## LoadOptions.UseSystemLcid property

Hämtar eller anger om LCID-värde från Windows-registret ska användas för att bestämma standardmarginaler för sidinställningar.

```csharp
public bool UseSystemLcid { get; set; }
```

## Anmärkningar

Om inställd på`sann`, sedan emuleras MS Word-beteendet vilket tar LCID-värdet från Windows-registret.

Standardvärdet är`falsk`.
