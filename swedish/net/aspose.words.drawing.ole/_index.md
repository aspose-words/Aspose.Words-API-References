De**Aspose.Words.Drawing.Ole** Namnrymden innehåller klasser för att arbeta med OLE/ActiveX-kontroller och formulär i Microsoft Word-dokument.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [CheckBoxControl](./checkboxcontrol/) | Kryssrutekontrollen växlar mellan ett värde. |
| [CommandButtonControl](./commandbuttoncontrol/) | CommandButton-kontrollen kör ett makro som utför en åtgärd när en användare klickar på det. |
| [Forms2OleControl](./forms2olecontrol/) | Representerar Microsoft Forms 2.0 OLE-kontroll. |
| [Forms2OleControlCollection](./forms2olecontrolcollection/) | Representerar en samling av[`Forms2OleControl`](../aspose.words.drawing.ole/forms2olecontrol/) objekt. |
| [MorphDataControl](./morphdatacontrol/) | Strukturen MorphDataControl är en samling av sex kontroller: CheckBox, ComboBox, ListBox, OptionButton, TextBox och ToggleButton. |
| [OleControl](./olecontrol/) | Representerar OLE ActiveX-kontroll. |
| [OptionButtonControl](./optionbuttoncontrol/) | Alternativknappskontrollen möjliggör ett enda val i en begränsad uppsättning ömsesidigt uteslutande val. |
| [TextBoxControl](./textboxcontrol/) | TextBox-kontrollen visar text från en organiserad datamängd eller användarinmatning. |
