De**Aspose.Words.Rendering** namnrymden tillhandahåller klasser som gör det möjligt att anpassa utskrift eller rendering av dokument.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [AsposeWordsPrintDocument](./asposewordsprintdocument/) | Tillhandahåller en standardimplementering för utskrift av en[`Document`](../aspose.words/document/)inom .NET-utskriftsramverket. |
| [NodeRendererBase](./noderendererbase/) | Basklass för[`ShapeRenderer`](../aspose.words.rendering/shaperenderer/) och[`OfficeMathRenderer`](../aspose.words.rendering/officemathrenderer/) . |
| [OfficeMathRenderer](./officemathrenderer/) | Tillhandahåller metoder för att rendera en individ[`OfficeMath`](../aspose.words.math/officemath/) till en raster- eller vektorbild eller till ett grafikobjekt. |
| [PageInfo](./pageinfo/) | Representerar information om en viss dokumentsida. |
| [PrinterSettingsContainer](./printersettingscontainer/) | Representerar ett minne för vissa parametrar avPrinterSettings objekt. |
| [ShapeRenderer](./shaperenderer/) | Tillhandahåller metoder för att rendera en individ[`Shape`](../aspose.words.drawing/shape/) eller[`GroupShape`](../aspose.words.drawing/groupshape/) till en raster- eller vektorbild eller till ett grafikobjekt. |
| [ThumbnailGeneratingOptions](./thumbnailgeneratingoptions/) | Kan användas för att ange ytterligare alternativ när miniatyrbilder genereras för ett dokument. |
## Uppräkning

| Uppräkning | Beskrivning |
| --- | --- |
| [ColorPrintMode](./colorprintmode/) | Anger hur sidor utan färg skrivs ut om enheten stöder färgutskrift. |
