## SaveOptions.CustomTimeZoneInfo property

Hämtar eller ställer in en anpassad lokal tidszon som används för datum-/tidsfält.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

## Anmärkningar

Det här alternativet är tillgängligt i antingen .Net Framework från och med version 3.5 eller .Net Standard.

Som standard använder Aspose.Words systemets lokala tidszon när datum-/tidsfält skrivs. Det här alternativet gör det möjligt att ange ett anpassat värde.
