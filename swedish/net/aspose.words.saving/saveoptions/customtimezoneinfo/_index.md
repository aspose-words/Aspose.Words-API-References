## SaveOptions.CustomTimeZoneInfo property

Hämtar eller ställer in anpassad lokal tidszon som används för datum-/tidsfält.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

## Anmärkningar

Det här alternativet är tillgängligt i antingen .Net framework från 3.5 version eller .Net Standard.

Som standard använder Aspose.Words systemets lokala tidszon vid skrivning av datum/tidsfält, detta alternativ gör det möjligt att ställa in anpassat värde.
