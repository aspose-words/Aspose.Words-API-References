De**Aspose.Words.Fonts** namnrymden tillhandahåller klasser och uppräkningar för att komma åt information om -teckensnitt som används i ett dokument.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [DefaultFontSubstitutionRule](./defaultfontsubstitutionrule/) | Standardregel för teckensnittsersättning. |
| [FileFontSource](./filefontsource/) | Representerar den enda TrueType-teckensnittsfilen som lagras i filsystemet. |
| [FolderFontSource](./folderfontsource/) | Representerar mappen som innehåller TrueType-teckensnittsfiler. |
| [FontConfigSubstitutionRule](./fontconfigsubstitutionrule/) | Regel för ersättning av teckensnittskonfiguration. |
| [FontEmbeddingLicensingRights](./fontembeddinglicensingrights/) | Representerar inbäddningslicensrättigheter för teckensnittet. |
| [FontFallbackSettings](./fontfallbacksettings/) | Anger inställningar för alternativa teckensnitt. |
| [FontInfo](./fontinfo/) | Anger information om ett teckensnitt som används i dokumentet. |
| [FontInfoCollection](./fontinfocollection/) | Representerar en samling teckensnitt som används i ett dokument. |
| [FontInfoSubstitutionRule](./fontinfosubstitutionrule/) | Regel för ersättning av teckensnittsinformation. |
| [FontNameSubstitutionRule](./fontnamesubstitutionrule/) | Regel för teckensnittsersättning för bearbetning av teckensnittsnamn. |
| [FontSettings](./fontsettings/) | Anger teckensnittsinställningar för ett dokument. |
| [FontSourceBase](./fontsourcebase/) | Detta är en abstrakt basklass för de klasser som låter användaren ange olika teckensnittskällor. |
| [FontSubstitutionRule](./fontsubstitutionrule/) | Detta är en abstrakt basklass för teckensnittsersättningsregeln. |
| [FontSubstitutionSettings](./fontsubstitutionsettings/) | Anger inställningar för mekanismen för teckensnittsersättning. |
| [MemoryFontSource](./memoryfontsource/) | Representerar den enda TrueType-teckensnittsfilen som lagras i minnet. |
| [PhysicalFontInfo](./physicalfontinfo/) | Anger information om fysiskt teckensnitt som är tillgängligt för Aspose.Words teckensnittsmotor. |
| [StreamFontSource](./streamfontsource/) | Basklass för användardefinierad strömfontkälla. |
| [SystemFontSource](./systemfontsource/) | Representerar alla TrueType-teckensnitt som är installerade i systemet. |
| [TableSubstitutionRule](./tablesubstitutionrule/) | Regel för ersättning av tabellfonter. |
