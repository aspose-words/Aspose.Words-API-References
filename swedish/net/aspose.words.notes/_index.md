De**Aspose.Words.Notes** namnrymden tillhandahåller klasser för att arbeta med slutnoter och fotnoter i Microsoft Word.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [EndnoteOptions](./endnoteoptions/) | Representerar alternativen för slutnotsnumrering för ett dokument eller avsnitt. |
| [Footnote](./footnote/) | Representerar en behållare för texten i en fotnot eller slutnot. |
| [FootnoteOptions](./footnoteoptions/) | Representerar fotnotsnumreringsalternativen för ett dokument eller avsnitt. |
| [FootnoteSeparator](./footnoteseparator/) | Representerar en behållare för fotnots-/slutnotsavgränsare och fortsättningsinnehåll i ett dokument. |
| [FootnoteSeparatorCollection](./footnoteseparatorcollection/) | Ger maskinskriven åtkomst till[`FootnoteSeparator`](../aspose.words.notes/footnoteseparator/) noder i ett dokument. |
## Uppräkning

| Uppräkning | Beskrivning |
| --- | --- |
| [EndnotePosition](./endnoteposition/) | Definierar slutnotens position. |
| [FootnoteNumberingRule](./footnotenumberingrule/) | Avgör när automatisk numrering av fotnoter eller slutnoter återupptas. |
| [FootnotePosition](./footnoteposition/) | Definierar fotnotens position. |
| [FootnoteSeparatorType](./footnoteseparatortype/) | Anger typen av fotnots-/slutnotsavgränsare. |
| [FootnoteType](./footnotetype/) | Anger om detta är en fotnot eller en slutnot. |
