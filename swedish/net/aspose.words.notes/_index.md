De **Aspose.Words.Notes** namnutrymme tillhandahåller klasser för att arbeta med Microsoft Word-slutnoter och fotnoter.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [EndnoteOptions](./endnoteoptions/) | Representerar slutnotens numreringsalternativ för ett dokument eller avsnitt. |
| [Footnote](./footnote/) | Representerar en behållare för text i en fotnot eller slutnot. |
| [FootnoteOptions](./footnoteoptions/) | Representerar alternativen för fotnotsnumrering för ett dokument eller avsnitt. |
## Uppräkning

| Uppräkning | Beskrivning |
| --- | --- |
| [EndnotePosition](./endnoteposition/) | Definierar slutnotens position. |
| [FootnoteNumberingRule](./footnotenumberingrule/) | Bestämmer när automatisk fotnots- eller slutnotsnumrering startar om. |
| [FootnotePosition](./footnoteposition/) | Definierar fotnotspositionen. |
| [FootnoteType](./footnotetype/) | Anger om detta är en fotnot eller en slutnot. |
