## CustomXmlPartCollection class

Represents a collection of Custom XML Parts. The items are [CustomXmlPart](../customxmlpart/) objects.
To learn more, visit the [Structured Document Tags or Content Control](https://docs.aspose.com/words/python-net/working-with-content-control-sdt/) documentation article.




### Remarks

You do not normally need to create instances of this class. You can access custom XML data
stored in a document via the [Document.custom_xml_parts](../../aspose.words/document/custom_xml_parts/) property.
