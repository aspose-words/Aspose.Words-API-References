## FileFontSource.cache_key property

The key of this source in the cache.

This key is used to identify cache item when saving/loading font search cache with
[FontSettings.save_search_cache()](../../fontsettings/save_search_cache/#bytesio) and
[FontSettings.set_fonts_sources()](../../fontsettings/set_fonts_sources/#fontsourcebaselist_bytesio) methods.

If key is not specified then [FileFontSource.file_path](../file_path/) will be used as a key instead.
