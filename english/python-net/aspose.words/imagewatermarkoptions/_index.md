## ImageWatermarkOptions class

Contains options that can be specified when adding a watermark with image.
To learn more, visit the [Working with Watermark](https://docs.aspose.com/words/python-net/working-with-watermark/) documentation article.




### Constructors
| Name | Description |
| --- | --- |
| [ImageWatermarkOptions()](./__init__/#default) | The default constructor. |

### Properties

| Name | Description |
| --- | --- |
| [is_washout](./is_washout/) | Gets or sets a boolean value which is responsible for washout effect of the watermark. The default value is ``True``. |
| [scale](./scale/) | Gets or sets the scale factor expressed as a fraction of the image. The default value is 0 - auto. |
