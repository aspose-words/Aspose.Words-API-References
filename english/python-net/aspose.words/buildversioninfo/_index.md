## BuildVersionInfo class

Provides information about the current product name and version.
To learn more, visit the [Generator or Producer Name Included in Output Documents](https://docs.aspose.com/words/python-net/generator-or-producer-name-included-in-output-documents/) documentation article.




### Properties

| Name | Description |
| --- | --- |
| [product](./product/) | Gets the full name of the product. |
| [version](./version/) | Gets the product version. |

### Examples

Shows how to display information about your installed version of Aspose.Words.

```python
print(f'I am currently using {aw.BuildVersionInfo.product}, version number {aw.BuildVersionInfo.version}!')
```
