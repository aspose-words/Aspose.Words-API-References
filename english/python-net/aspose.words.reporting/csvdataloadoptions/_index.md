## CsvDataLoadOptions class

Represents options for parsing CSV data.
To learn more, visit the [LINQ Reporting Engine](https://docs.aspose.com/words/python-net/linq-reporting-engine/) documentation article.




An instance of this class can be passed into constructors of [CsvDataSource](../csvdatasource/).



### Constructors
| Name | Description |
| --- | --- |
| [CsvDataLoadOptions()](./__init__/#default) | Initializes a new instance of this class with default options. |
| [CsvDataLoadOptions(has_headers)](./__init__/#bool) | Initializes a new instance of this class with specifying whether CSV data contains column names at the first line. |

### Properties

| Name | Description |
| --- | --- |
| [comment_char](./comment_char/) | Gets or sets the character that is used to comment lines of CSV data. |
| [delimiter](./delimiter/) | Gets or sets the character to be used as a column delimiter. |
| [has_headers](./has_headers/) | Gets or sets a value indicating whether the first record of CSV data contains column names. |
| [quote_char](./quote_char/) | Gets or sets the character that is used to quote field values. |
