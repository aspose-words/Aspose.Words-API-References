## VariationAxis enumeration

Represents OpenType Design-Variation Axis Tag.
https://learn.microsoft.com/en-us/typography/opentype/spec/dvaraxisreg


### Members

| Name | Description |
| --- | --- |
| ITALIC | Registered tag for the roman/italic axis. |
| OPTICAL_SIZE | Registered tag for the optical-size axis. Note: The optical-size axis supersedes the OpenType `size` feature. |
| SLANT | Registered tag for the slant axis. |
| WEIGHT | Registered tag for the weight axis. |
| WIDTH | Registered tag for the width axis. |
