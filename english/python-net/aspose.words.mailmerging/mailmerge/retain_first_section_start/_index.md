## MailMerge.retain_first_section_start property

Gets or sets a value indicating whether the [PageSetup.section_start](../../../aspose.words/pagesetup/section_start/) of the first document section and its copies for subsequent data source rows
are retained during mail merge or updated according to MS Word behaviour.


The default value is ``True``.
