## FieldMergingArgsBase class

Base class for [FieldMergingArgs](../fieldmergingargs/) and [ImageFieldMergingArgs](../imagefieldmergingargs/).
To learn more, visit the [Mail Merge and Reporting](https://docs.aspose.com/words/python-net/mail-merge-and-reporting/) documentation article.

### Properties

| Name | Description |
| --- | --- |
| [document](./document/) | Returns the [FieldMergingArgsBase.document](./document/) object for which the mail merge is performed. |
| [document_field_name](./document_field_name/) | Gets the name of the merge field as specified in the document. |
| [field](./field/) | Gets the object that represents the current merge field. |
| [field_name](./field_name/) | Gets the name of the merge field in the data source. |
| [field_value](./field_value/) | Gets or sets the value of the field from the data source. |
| [record_index](./record_index/) | Gets the zero based index of the record that is being merged. |
| [table_name](./table_name/) | Gets the name of the data table for the current merge operation or empty string if the name is not available. |
