## NodeRendererBase.RenderToSize method

Renders the shape into a Graphics object to a specified size.

```csharp
public float RenderToSize(Graphics graphics, float x, float y, float width, float height)
```

| Parameter | Type | Description |
| --- | --- | --- |
| graphics | Graphics | The object where to render to. |
| x | Single | The X coordinate (in world units) of the top left corner of the rendered shape. |
| y | Single | The Y coordinate (in world units) of the top left corner of the rendered shape. |
| width | Single | The maximum width (in world units) that can be occupied by the rendered shape. |
| height | Single | The maximum height (in world units) that can be occupied by the rendered shape. |
