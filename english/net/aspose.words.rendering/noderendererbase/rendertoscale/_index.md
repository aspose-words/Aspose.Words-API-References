## NodeRendererBase.RenderToScale method

Renders the shape into a Graphics object to a specified scale.

```csharp
public SizeF RenderToScale(Graphics graphics, float x, float y, float scale)
```

| Parameter | Type | Description |
| --- | --- | --- |
| graphics | Graphics | The object where to render to. |
| x | Single | The X coordinate (in world units) of the top left corner of the rendered shape. |
| y | Single | The Y coordinate (in world units) of the top left corner of the rendered shape. |
| scale | Single | The scale for rendering the shape (1.0 is 100%). |

### Return Value

The width and height (in world units) of the rendered shape.
