## ImageSaveOptions.TiffBinarizationMethod property

Gets or sets method used while converting images to 1 bpp format when [`SaveFormat`](../saveformat) is SaveFormat.Tiff and [`TiffCompression`](../tiffcompression) is equal to TiffCompression.Ccitt3 or TiffCompression.Ccitt4.

```csharp
public ImageBinarizationMethod TiffBinarizationMethod { get; set; }
```

### Remarks

The default value is ImageBinarizationMethod.Threshold.
