## Row.GetText method

Gets the text of all cells in this row including the end of row character.

```csharp
public override string GetText()
```

## Remarks

Returns concatenated text of all child nodes with the end of row character [`ControlChar.Cell`](../../../aspose.words/controlchar/cell) appended at the end.The returned string includes all control and special characters as described in [`ControlChar`](../../../aspose.words/controlchar).
