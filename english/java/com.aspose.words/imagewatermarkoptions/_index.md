### getScale() {#getScale--}
```
public double getScale()
```


获取表示为图像分数的比例因子。默认值为 0 - 自动。

有效值范围从 0 到 65.5（含）。

自动缩放意味着水印将缩放到其相对于页边距的最大宽度和最大高度。

**退货:**
double - 表示为图像分数的比例因子。
