java.lang.Object
```
public class FrameFormat
```

表示段落的框架相关格式。

始终创建此对象。如果一个段落是一个框架，那么所有属性都将包含各自的值，否则所有属性都设置为其默认值。

利用[isFrame()](../../com.aspose.words/frameformat\#isFrame--)检查段落是否为框架。
