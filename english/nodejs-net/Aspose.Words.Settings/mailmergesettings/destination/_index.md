## MailMergeSettings.destination property

Specifies how Microsoft Word will output the results of a mail merge.
The default value is [MailMergeDestination.Default](../../mailmergedestination/#Default).



```js
get destination(): Aspose.Words.Settings.MailMergeDestination
```
