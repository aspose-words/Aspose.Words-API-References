## MailMergeSettings.checkErrors property

Specifies the type of error reporting which shall be conducted by Microsoft Word when performing a mail merge. 
The default value is [MailMergeCheckErrors.Default](../../mailmergecheckerrors/#Default).



```js
get checkErrors(): Aspose.Words.Settings.MailMergeCheckErrors
```
