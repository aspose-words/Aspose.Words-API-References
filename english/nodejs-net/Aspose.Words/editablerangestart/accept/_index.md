## accept(visitor) {#documentvisitor}

Accepts a visitor.


```js
accept(visitor: Aspose.Words.DocumentVisitor)
```

| Parameter | Type | Description |
| --- | --- | --- |
| visitor | [DocumentVisitor](../../documentvisitor/) | The visitor that will visit the node. |

### Remarks

Calls [DocumentVisitor.visitEditableRangeStart()](../../documentvisitor/visitEditableRangeStart/#editablerangestart).

For more info see the Visitor design pattern.




### Returns

``false`` if the visitor requested the enumeration to stop.
