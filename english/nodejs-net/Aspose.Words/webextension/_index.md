## WebExtension class

Represents a web extension object.
To learn more, visit the [Work with Office Add-ins](https://docs.aspose.com/words/nodejs-net/work-with-office-add-ins/) documentation article.




### Properties

| Name | Description |
| --- | --- |
| [alternateReferences](./alternateReferences/) | Specifies alternate references to a web extension. |
| [bindings](./bindings/) | Specifies a list of web extension bindings. |
| [id](./id/) | Uniquely identifies the web extension instance in the current document. |
| [isFrozen](./isFrozen/) | Specifies whether the user can interact with the web extension or not. |
| [properties](./properties/) | Represents a set of web extension custom properties. |
| [reference](./reference/) | Specifies the primary reference to an web extension. |
