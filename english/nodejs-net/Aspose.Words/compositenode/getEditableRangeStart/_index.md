## getEditableRangeStart(index, isDeep) {#number_boolean}

Returns an Nth child [EditableRangeStart](../../editablerangestart/) node.



```js
getEditableRangeStart(index: number, isDeep: boolean)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | number | Zero based index of the child [EditableRangeStart](../../editablerangestart/) node to select. Negative indexes are also allowed and indicate access from the end, that is -1 means the last node. |
| isDeep | boolean | ``true`` to select from all child nodes recursively; ``false`` to select only among immediate children. |

### Remarks

If index is out of range, a ``null`` is returned.




### Returns

The child [EditableRangeStart](../../editablerangestart/) node or ``null`` if no matching node is found.
