## getSdtRangeEnd(index, isDeep) {#number_boolean}

Returns an Nth child [StructuredDocumentTagRangeEnd](../../../aspose.words.markup/structureddocumenttagrangeend/) node.



```js
getSdtRangeEnd(index: number, isDeep: boolean)
```

| Parameter | Type | Description |
| --- | --- | --- |
| index | number | Zero based index of the child [StructuredDocumentTagRangeEnd](../../../aspose.words.markup/structureddocumenttagrangeend/) node to select. Negative indexes are also allowed and indicate access from the end, that is -1 means the last node. |
| isDeep | boolean | ``true`` to select from all child nodes recursively; ``false`` to select only among immediate children. |

### Remarks

If index is out of range, a ``null`` is returned.




### Returns

The child [StructuredDocumentTagRangeEnd](../../../aspose.words.markup/structureddocumenttagrangeend/) node or ``null`` if no matching node is found.
