## Frameset class

Represents a frames page or a single frame on a frames page.
To learn more, visit the [Programming with Documents](https://docs.aspose.com/words/nodejs-net/programming-with-documents/) documentation article.




### Remarks

If the [Frameset.childFramesets](./childFramesets/) property contains items, this instance is a frames page, otherwise it is
a single frame.
