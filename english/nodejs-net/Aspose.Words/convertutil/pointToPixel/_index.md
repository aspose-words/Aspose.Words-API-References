## pointToPixel(points) {#number}

Converts points to pixels at 96 dpi.


```js
pointToPixel(points: number)
```

| Parameter | Type | Description |
| --- | --- | --- |
| points | number | The value to convert. |

### Remarks

1 inch equals 72 points.
