## ChartDataLabel.topMode property

Gets or sets the interpretation mode of the [ChartDataLabel.top](../top/) property value: whether it sets the location
of the data label from the top edge of the chart of from the position specified by its [ChartDataLabel.position](../position/)
property.



```js
get topMode(): Aspose.Words.Drawing.Charts.ChartDataLabelLocationMode
```

### Remarks

The property cannot be set in a Word 2016 chart.
