## FieldOptions.builtInTemplatesPaths property

Gets or sets paths of MS Word built-in templates.


```js
get builtInTemplatesPaths(): string[]
```

### Remarks

This property is used by the [FieldAutoText](../../fieldautotext/) and [FieldGlossary](../../fieldglossary/) fields, if referenced auto text entry is not found in the [Document.attachedTemplate](../../../aspose.words/document/attachedTemplate/) template.

By default MS Word stores built-in templates in c:\\Users\\\<username\>\\AppData\\Roaming\\Microsoft\\Document Building Blocks\\1033\\16\\Built-In Building Blocks.dotx and
C:\\Users\\\<username\>\\AppData\\Roaming\\Microsoft\\Templates\\Normal.dotm files.
