## FieldTemplate class

Implements the TEMPLATE field.
To learn more, visit the [Working with Fields](https://docs.aspose.com/words/nodejs-net/working-with-fields/) documentation article.




### Remarks

Retrieves the file name of the template used by the current document.


**Inheritance:** [FieldTemplate](./) → [Field](../../aspose.words/field/)
