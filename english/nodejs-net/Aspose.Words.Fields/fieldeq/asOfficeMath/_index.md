## asOfficeMath() {#default}

Returns Office Math object corresponded to the EQ field.


```js
asOfficeMath()
```

### Returns

Returns ``null`` if field code is empty or invalid, otherwise an [OfficeMath](../../../aspose.words.math/officemath/) instance.
