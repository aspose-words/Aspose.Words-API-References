## setTextInputValue(newValue) {#object}

Applies the text format specified in [FormField.textInputFormat](../textInputFormat/) and stores the value in [FormField.result](../result/).



```js
setTextInputValue(newValue: object)
```

| Parameter | Type | Description |
| --- | --- | --- |
| newValue | object | Can be a string, number or a **DateTime** object. |

### Remarks

The [FormField.textInputDefault](../textInputDefault/) value is applied if *newValue* is``null``.
