## VbaProject class

Provides access to VBA project information.
A VBA project inside the document is defined as a collection of VBA modules.
To learn more, visit the [Working with VBA Macros](https://docs.aspose.com/words/nodejs-net/working-with-vba-macros/) documentation article.




### Constructors
| Name | Description |
| --- | --- |
| [VbaProject()](./constructor/#default) | Creates a blank [VbaProject](./). |

| Name | Description |
| --- | --- |

| [isProtected](./isProtected/) | Shows whether the [VbaProject](./) is password protected. |
| [isSigned](./isSigned/) | Shows whether the [VbaProject](./) is signed or not. |
| [modules](./modules/) | Returns collection of VBA project modules. |
