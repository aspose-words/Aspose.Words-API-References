## SaveOutputParameters class

This object is returned to the caller after a document is saved and contains additional information that 
has been generated or calculated during the save operation. The caller can use or ignore this object.
To learn more, visit the [Save a Document](https://docs.aspose.com/words/nodejs-net/save-a-document/) documentation article.




### Properties

| Name | Description |
| --- | --- |
| [contentType](./contentType/) | Returns the Content-Type string (Internet Media Type) that identifies the type of the saved document. |
