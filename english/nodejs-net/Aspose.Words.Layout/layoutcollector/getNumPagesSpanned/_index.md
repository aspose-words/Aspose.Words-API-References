## getNumPagesSpanned(node) {#node}

Gets number of pages the specified node spans. 0 if node is within a single page.
This is the same as [LayoutCollector.getEndPageIndex()](../getEndPageIndex/#node) - [LayoutCollector.getStartPageIndex()](../getStartPageIndex/#node).



```js
getNumPagesSpanned(node: Aspose.Words.Node)
```
