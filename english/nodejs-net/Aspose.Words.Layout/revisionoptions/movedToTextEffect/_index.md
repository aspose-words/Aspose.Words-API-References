## RevisionOptions.movedToTextEffect property

Allows to specify the effect to be applied to the areas where content was moved to [RevisionType.Moving](../../../aspose.words/revisiontype/#Moving).
Default value is [RevisionTextEffect.DoubleUnderline](../../revisiontexteffect/#DoubleUnderline)



```js
get movedToTextEffect(): Aspose.Words.Layout.RevisionTextEffect
```

### Exceptions

| exception | condition |
| --- | --- |
| RuntimeError (Proxy error(ArgumentOutOfRangeException)) | Values of [RevisionTextEffect.Hidden](../../revisiontexteffect/#Hidden) and [RevisionTextEffect.DoubleStrikeThrough](../../revisiontexteffect/#DoubleStrikeThrough)  are not allowed. |
