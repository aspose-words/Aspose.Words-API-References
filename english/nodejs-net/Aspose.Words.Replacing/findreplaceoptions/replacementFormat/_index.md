## FindReplaceOptions.replacementFormat property

Specifies format of the replacement. Default is [ReplacementFormat.Text](../../replacementformat/#Text).



```js
get replacementFormat(): Aspose.Words.Replacing.ReplacementFormat
```

### Remarks

Has effect only when using in Aspose.Words.LowCode.Replacer
