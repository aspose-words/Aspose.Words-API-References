## FontEmbeddingLicensingRights class

Represents embedding licensing rights for the font.


### Remarks

To learn more, visit the
[OpenType specification section](https://learn.microsoft.com/en-us/typography/opentype/spec/os2#fstype)
on Microsoft Typography portal.



### Properties

| Name | Description |
| --- | --- |
| [bitmapEmbeddingOnly](./bitmapEmbeddingOnly/) | Indicates the "Bitmap embedding only" restriction. |
| [embeddingUsagePermissions](./embeddingUsagePermissions/) | Usage permissions. |
| [noSubsetting](./noSubsetting/) | Indicates the "No subsetting" restriction. |
