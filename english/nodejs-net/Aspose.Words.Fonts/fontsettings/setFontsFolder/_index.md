## setFontsFolder(fontFolder, recursive) {#string_boolean}

Sets the folder where Aspose.Words looks for TrueType fonts when rendering documents or embedding fonts.
This is a shortcut to [FontSettings.setFontsFolders()](../setFontsFolders/#string[]_boolean) for setting only one font directory.



```js
setFontsFolder(fontFolder: stringrecursive: boolean)
```

| Parameter | Type | Description |
| --- | --- | --- |
| fontFolder | string | The folder that contains TrueType fonts. |
| recursive | boolean | True to scan the specified folders for fonts recursively. |
