The **Aspose.Words.Math** module contains classes that represent Office Math elements.





## Classes

| Class | Description |
| --- | --- |
| [OfficeMath](./officemath/) | Represents an Office Math object such as function, equation, matrix or alike. Can contain child elements including runs of mathematical text, bookmarks, comments, other [OfficeMath](./officemath/) instances and some other nodes. To learn more, visit the [Working with OfficeMath](https://docs.aspose.com/words/nodejs-net/working-with-officemath/) documentation article. |

## Enumerations

| Enumeration | Description |
| --- | --- |
| [MathObjectType](./mathobjecttype/) | Specifies type of an Office Math object. |
| [OfficeMathDisplayType](./officemathdisplaytype/) | Specifies the display format type of the equation. |
| [OfficeMathJustification](./officemathjustification/) | Specifies the justification of the equation. |
