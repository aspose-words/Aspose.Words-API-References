## ImageSize(widthPixels, heightPixels) {#number_number}

Initializes width and height to the given values in pixels. Initializes resolution to 96 dpi.


```js
ImageSize(widthPixels: number, heightPixels: number)
```

| Parameter | Type | Description |
| --- | --- | --- |
| widthPixels | number | Width in pixels. |
| heightPixels | number | Height in pixels. |

## ImageSize(widthPixels, heightPixels, horizontalResolution, verticalResolution) {#number_number_number_number}

Initializes width, height and resolution to the given values.


```js
ImageSize(widthPixels: number, heightPixels: number, horizontalResolution: number, verticalResolution: number)
```

| Parameter | Type | Description |
| --- | --- | --- |
| widthPixels | number | Width in pixels. |
| heightPixels | number | Height in pixels. |
| horizontalResolution | number | Horizontal resolution in DPI. |
| verticalResolution | number | Vertical resolution in DPI. |
