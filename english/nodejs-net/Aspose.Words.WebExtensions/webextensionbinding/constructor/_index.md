## WebExtensionBinding(id, bindingType, appRef) {#string_webextensionbindingtype_string}

Creates web extension binding with specified parameters.


```js
WebExtensionBinding(id: stringbindingType: Aspose.Words.WebExtensions.WebExtensionBindingTypeappRef: string)
```

| Parameter | Type | Description |
| --- | --- | --- |
| id | string | Binding identifier. |
| bindingType | [WebExtensionBindingType](../../webextensionbindingtype/) | Binding type. |
| appRef | string | Binding key used to map the binding entry in this list with the bound data in the document. |
