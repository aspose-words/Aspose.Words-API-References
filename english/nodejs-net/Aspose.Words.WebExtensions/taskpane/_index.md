## TaskPane class

Represents an add-in task pane object.
To learn more, visit the [Work with Office Add-ins](https://docs.aspose.com/words/nodejs-net/work-with-office-add-ins/) documentation article.




### Constructors
| Name | Description |
| --- | --- |
| [TaskPane()](./TaskPane/#default) | The default constructor. |

### Properties

| Name | Description |
| --- | --- |
| [dockState](./dockState/) | Specifies the last-docked location of this task pane object. |
| [isLocked](./isLocked/) | Specifies whether the task pane is locked to the document in the UI and cannot be closed by the user. |
| [isVisible](./isVisible/) | Specifies whether the task pane shows as visible by default when the document opens. |
| [row](./row/) | Specifies the index, enumerating from the outside to the inside, of this task pane among other persisted task panes docked in the same default location. |
| [webExtension](./webExtension/) | Represents an web extension object. |
| [width](./width/) | Specifies the default width value for this task pane instance. |
