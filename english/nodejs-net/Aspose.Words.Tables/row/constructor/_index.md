## Row(doc) {#documentbase}

Initializes a new instance of the [Row](../) class.



```js
Row(doc: Aspose.Words.DocumentBase)
```

| Parameter | Type | Description |
| --- | --- | --- |
| doc | [DocumentBase](../../../Aspose.Words/documentbase/) | The owner document. |

### Remarks

When [Row](../) is created, it belongs to the specified document, but is not
yet part of the document and [Node.parentNode](../../../Aspose.Words/node/parentNode/) is ``None``.

To append [Row](../) to the document use [CompositeNode.insertAfter()](../../../Aspose.Words/compositenode/insertAfter/#node_node) or [CompositeNode.insertBefore()](../../../Aspose.Words/compositenode/insertBefore/#node_node)
on the table where you want the row inserted.
