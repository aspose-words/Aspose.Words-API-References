## IStructuredDocumentTag.IsShowingPlaceholderText property

Bunun içeriğinin olup olmadığını belirtir.**SDT** metin yer tutucusunu içerecek şekilde yorumlanacaktır (SDT içindeki normal metin içeriklerinin aksine).

true olarak ayarlanırsa, bu belge açıldığında bu durum devam ettirilecektir (yer tutucu metni göstererek).

public bool IsShowingPlaceholderText { get; set; }
