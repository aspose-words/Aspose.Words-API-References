## FileFormatInfo.Encoding property

Geçerli belge biçimi için geçerliyse algılanan kodlamayı alır. Şu anda yalnızca HTML belgeleri için kodlamayı algılar.

```csharp
public Encoding Encoding { get; }
```

### Örnekler

Bir html dosyasındaki kodlamanın nasıl algılanacağını gösterir.

```csharp
FileFormatInfo info = FileFormatUtil.DetectFileFormat(MyDir + "Document.html");

Assert.AreEqual(LoadFormat.Html, info.LoadFormat);

// Encoding özelliği yalnızca bir html belgesi için bir FileFormatInfo nesnesi oluşturduğumuzda kullanılır.
Assert.AreEqual("Western European (Windows)", info.Encoding.EncodingName);
Assert.AreEqual(1252, info.Encoding.CodePage);
```
