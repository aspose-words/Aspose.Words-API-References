**Aspose.Words.Settings** ad alanı, bir belgede saklanan çeşitli ayarları ve seçenekleri belirtirken kullanılan sınıfları ve enums 'yi içerir. Bunlar, adres-mektup birleştirme, yazma koruması, uyumluluk ve diğer ayarlardır.

## sınıflar

| Sınıf | Tanım |
| --- | --- |
| [CompatibilityOptions](./compatibilityoptions/) | Uyumluluk seçeneklerini (yani, **Uyumluluk** sekmesi **Seçenekler** Microsoft Word'deki iletişim kutusu). |
| [HyphenationOptions](./hyphenationoptions/) | Belge tireleme seçeneklerini yapılandırmaya izin verir. |
| [MailMergeSettings](./mailmergesettings/) | Bir belgeye ilişkin tüm adres-mektup birleştirme bilgilerini belirtir. |
| [Odso](./odso/) | Adres-mektup birleştirme veri kaynağı için Office Veri Kaynağı Nesnesi (ODSO) ayarlarını belirtir. |
| [OdsoFieldMapData](./odsofieldmapdata/) | Harici veri kaynağındaki bir sütunun, belge içindeki önceden tanımlanmış birleştirme alanlarıyla nasıl eşleneceğini belirtir. |
| [OdsoFieldMapDataCollection](./odsofieldmapdatacollection/) | Yazılı bir koleksiyon[`OdsoFieldMapData`](../aspose.words.settings/odsofieldmapdata/) nesneler. |
| [OdsoRecipientData](./odsorecipientdata/) | Dış veri kaynağı içindeki adres-mektup birleştirmenin dışında tutulacak tek bir kayıt hakkındaki bilgileri temsil eder. |
| [OdsoRecipientDataCollection](./odsorecipientdatacollection/) | Yazılı bir koleksiyon[`OdsoRecipientData`](../aspose.words.settings/odsorecipientdata/) |
| [ViewOptions](./viewoptions/) | Bir belgenin Microsoft Word'de nasıl gösterileceğini kontrol eden çeşitli seçenekler sağlar. |
| [WriteProtection](./writeprotection/) | Bir belgenin yazma koruması ayarlarını belirtir. |

| [Compatibility](./compatibility/) | Uyumluluk seçeneklerinin adlarını belirtir. |

| [MailMergeDataType](./mailmergedatatype/) | Harici adres-mektup birleştirme veri kaynağının türünü belirtir. |
| [MailMergeDestination](./mailmergedestination/) | Bir belge üzerinde adres-mektup birleştirme gerçekleştirildiğinde oluşturulabilecek olası sonuçları belirtir. |
| [MailMergeMainDocumentType](./mailmergemaindocumenttype/) | Adres-mektup birleştirme kaynak belgesinin olası türlerini belirtir. |
| [MsWordVersion](./mswordversion/) | Aspose.Wods'un MS Word sürümüne özel uygulama davranışını taklit etmesini sağlar. |
| [MultiplePagesType](./multiplepagestype/) | Belgenin nasıl yazdırılacağını belirtir. |

| [ViewType](./viewtype/) | Microsoft Word'deki görüntüleme modu için olası değerler. |
