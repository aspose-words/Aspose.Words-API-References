## LoadOptions.UseSystemLcid property

Sayfa düzeni varsayılan kenar boşluklarını belirlemek için Windows kayıt defterinden alınan LCID değerinin kullanılıp kullanılmayacağını alır veya ayarlar.

```csharp
public bool UseSystemLcid { get; set; }
```

## Notlar

Eğer ayarlanırsa`doğru`, daha sonra Windows kayıt defterinden LCID değerini alan MS Word davranışı taklit edilir.

Varsayılan değer:`YANLIŞ`.
