## JsonSimpleValueParseMode enumeration

JSON yüklenirken JSON basit değerlerinin (boş, boole, sayı, tamsayı ve dize) ayrıştırılması için bir mod belirtir. Böyle bir mod, tarih-saat değerlerinin ayrıştırılmasını etkilemez.

```csharp
public enum JsonSimpleValueParseMode
```

### değerler

| İsim | Değer | Tanım |
| --- | --- | --- |
| Loose | `0` | JSON basit değer türlerinin, dize temsillerinin ayrıştırılması üzerine belirlendiği modu belirtir. Örneğin, JSON parçacığından '{ prop: "123" }' 'prop' türü bu modda tamsayı olarak belirlenir. |
| Strict | `1` | JSON basit değer türlerinin JSON gösteriminin kendisinden belirlendiği modu belirtir. Örneğin, JSON snippet'inden '{ prop: "123" }' 'prop' türü bu modda dize olarak belirlenir. |
