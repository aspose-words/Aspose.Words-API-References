### getAlwaysGenerateRootObject() {#getAlwaysGenerateRootObject--}
```
public boolean getAlwaysGenerateRootObject()
```


获取一个标志，该标志指示生成的数据源是否将始终包含 XML 根元素的对象。如果一个 XML 根元素没有属性并且它的所有子元素都具有相同的名称，则默认情况下不会创建这样的对象。默认值为**false**.

**退货:**
