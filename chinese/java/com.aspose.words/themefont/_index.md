java.lang.Object
```
public class ThemeFont
```

指定文档主题的主题字体名称类型。指定可以在父对象属性中作为主题字体引用的主题字体类型。此主题字体是对位于文档的主题部分中的预定义主题字体之一的引用，它允许在文档中集中设置字体信息。
## 字段

| 字段 | 描述 |
| --- | --- |
| [MAJOR](#MAJOR) | 主要主题字体。 |
| [MINOR](#MINOR) | 次要主题字体。 |
| [NONE](#NONE) | 没有主题字体。 |
