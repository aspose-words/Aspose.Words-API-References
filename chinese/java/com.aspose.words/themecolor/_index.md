**遗产:**
java.lang.Object
```
public class ThemeColor
```

指定文档主题的主题颜色。

要了解更多信息，请访问**Working with Styles and Themes**文档文章。

指定的主题颜色是对位于文档的主题部分中的预定义主题颜色之一的引用，它允许在文档中集中设置颜色信息。
