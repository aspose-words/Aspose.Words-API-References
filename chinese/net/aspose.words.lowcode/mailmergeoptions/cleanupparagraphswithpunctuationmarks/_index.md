## MailMergeOptions.CleanupParagraphsWithPunctuationMarks property

获取或设置一个值，指示带有标点符号的段落是否被视为空 ，并且如果RemoveEmptyParagraphs指定了选项。

```csharp
public bool CleanupParagraphsWithPunctuationMarks { get; set; }
```

## 评论

默认值为`真的`.

* ！
* ，
* 。
* ：
* ；
* ？
* ¡
