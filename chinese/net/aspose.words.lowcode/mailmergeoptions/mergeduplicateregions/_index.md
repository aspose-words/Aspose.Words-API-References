## MailMergeOptions.MergeDuplicateRegions property

获取或设置一个值，该值指示在执行针对数据源的区域邮件合并时，是否应合并所有具有数据源名称的文档邮件合并区域 ，还是仅合并第一个。

```csharp
public bool MergeDuplicateRegions { get; set; }
```

## 评论

默认值为`错误的`.
