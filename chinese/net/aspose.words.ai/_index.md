这**Aspose.Words.AI**命名空间可以与大型语言模型 (LLM)（例如 OpenAI 和 Gemini）无缝集成，为 AI 驱动的文档处理提供强大的支持。

## 课程

| 班级 | 描述 |
| --- | --- |
| [AiModel](./aimodel/) | 表示有关生成语言模型的信息。 |
| [AnthropicAiModel](./anthropicaimodel/) | 一个抽象类，代表与 Aspose.Words 中的 Anthropic AI 模型的集成。 |
| [CheckGrammarOptions](./checkgrammaroptions/) | 允许在使用 AI 检查文档语法时指定各种选项。 |
| [GoogleAiModel](./googleaimodel/) | 一个抽象类，代表与 Aspose.Words 中的 Google AI 模型的集成。 |
| [OpenAiModel](./openaimodel/) | 一个抽象类，表示与 Aspose.Words 中的 OpenAI 大型语言模型的集成。 |
| [SummarizeOptions](./summarizeoptions/) | 允许指定用于总结文档内容的各种选项。 |
## 接口

| 界面 | 描述 |
| --- | --- |
| [IAiModelText](./iaimodeltext/) | 旨在生成各种基于文本的内容的 AI 模型的通用接口。 |
