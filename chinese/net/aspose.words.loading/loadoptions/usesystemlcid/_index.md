## LoadOptions.UseSystemLcid property

获取或设置是否使用从 Windows 注册表获取的 LCID 值来确定页面设置默认边距。

```csharp
public bool UseSystemLcid { get; set; }
```

## 评论

如果设置为`真的`，然后模拟 MS Word 行为，从 Windows 注册表中获取 LCID 值。

默认值为`错误的`。
