## HtmlLoadOptions.IgnoreNoscriptElements property

获取或设置是否忽略&lt;noscript&gt;的值HTML 元素。 默认值为` false` 。

```csharp
public bool IgnoreNoscriptElements { get; set; }
```

### 评论

与 MS Word 一样，Aspose.Words 不支持脚本，默认加载 &lt;noscript&gt; 的内容元素 到生成的文档中。但是，在大多数浏览器中，都支持脚本，并且内容来自 &lt;noscript&gt; 不可见。将此属性设置为` true` 强制 Aspose.Words 忽略所有 &lt;noscript&gt;元素 并有助于生成看起来更接近浏览器中所见内容的文档。

### 例子

显示如何忽略 &lt;noscript&gt; HTML 元素。

```csharp
const string html = @"
    <html>
      <head>
        <title>NOSCRIPT</title>
          <meta http-equiv=""Content-Type"" content=""text/html; charset=utf-8"">
          <script type=""text/javascript"">
            alert(""Hello, world!"");
          </script>
      </head>
    <body>
      <noscript><p>Your browser does not support JavaScript!</p></noscript>
    </body>
    </html>";

HtmlLoadOptions htmlLoadOptions = new HtmlLoadOptions();
htmlLoadOptions.IgnoreNoscriptElements = ignoreNoscriptElements;

Document doc = new Document(new MemoryStream(Encoding.UTF8.GetBytes(html)), htmlLoadOptions);
doc.Save(ArtifactsDir + "HtmlLoadOptions.IgnoreNoscriptElements.pdf");
```
