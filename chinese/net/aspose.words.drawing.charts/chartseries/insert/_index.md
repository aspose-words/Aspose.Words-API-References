## Insert(int, ChartXValue) {#insert}

将指定的 X 值插入到图表系列的指定索引处。如果系列支持 Y 值 和气泡大小，则 X 值它们将为空。

```csharp
public void Insert(int index, ChartXValue xValue)
```

### 评论

具有默认格式的相应数据点将被插入到数据点集合中。并且, 如果显示数据标签，也会插入相应的具有默认格式的数据标签。
