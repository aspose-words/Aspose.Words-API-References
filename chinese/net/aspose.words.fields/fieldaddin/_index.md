## 例子

展示如何处理 ADDIN 字段。

```csharp
Document doc = new Document(MyDir + "Field sample - ADDIN.docx");

// Aspose.Words 不支持插入 ADDIN 字段，但我们仍然可以加载和读取它们。
FieldAddIn field = (FieldAddIn)doc.Range.Fields[0];

Assert.AreEqual(" ADDIN \"My value\" ", field.GetFieldCode());
```
