## FieldOptions.PreProcessCulture property

获取或设置区域性以预处理字段值。

```csharp
public CultureInfo PreProcessCulture { get; set; }
```

### 评论

目前此属性仅影响FieldDocProperty字段。

默认值为 **null** 。当此属性设置为 **null** 时，会预处理[`FieldDocProperty`](../../fielddocproperty)字段的值 文化由[`FieldUpdateCultureSource`](../fieldupdateculturesource)属性控制。

### 例子

显示如何设置预处理区域性。

```csharp
Document doc = new Document(MyDir + "Document.docx");
DocumentBuilder builder = new DocumentBuilder(doc);

 // 设置某些字段将根据其设置显示值的文化。
doc.FieldOptions.PreProcessCulture = new CultureInfo("de-DE");

Field field = builder.InsertField(" DOCPROPERTY CreateTime");

 // DOCPROPERTY 字段将根据预处理culture
 显示其结果
// 我们已经设置为德语。该字段将使用“dd.mm.yyyy hh:mm”格式显示日期/时间。
Assert.IsTrue(Regex.Match(field.Result, @"\d{2}[.]\d{2}[.]\d{4} \d{2}[:]\d{2}").Success);

doc.FieldOptions.PreProcessCulture = CultureInfo.InvariantCulture;
field.Update();

 // 切换到不变区域性后，DOCPROPERTY 字段将使用“mm/dd/yyyy hh:mm”格式。
Assert.IsTrue(Regex.Match(field.Result, @"\d{2}[/]\d{2}[/]\d{4} \d{2}[:]\d{2}").Success);
```
