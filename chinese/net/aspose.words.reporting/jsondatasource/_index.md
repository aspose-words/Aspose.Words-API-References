### 评论

访问相应文件或流的数据，同时生成一个报告，将此类的实例作为 数据源传递给[`ReportingEngine`](../reportingengine).BuildReport 重载之一。

在模板文档中，如果顶级 JSON 元素是数组，则[`JsonDataSource`](../jsondatasource)实例应该是 以与DataTable相同的方式处理实例。如果顶级 JSON 元素 是对象，则[`JsonDataSource`](../jsondatasource)实例应以与:相同的方式处理一个DataRow 实例。有关详细信息，请参阅模板语法参考 (https://docs.aspose.com/display/wordsnet/Template+Syntax)。

在模板文档中，您可以使用 JSON 元素的类型值。为方便起见，引擎将 JSON 简单类型的集合 替换为以下类型:

* Nullable
* Nullable
* Nullable
* Nullable
* String

引擎会根据 JSON 表示形式自动识别额外类型的值。

要覆盖 JSON 数据加载的默认行为，初始化并传递[`JsonDataLoadOptions`](../jsondataloadoptions)instance 到此类的构造函数。
