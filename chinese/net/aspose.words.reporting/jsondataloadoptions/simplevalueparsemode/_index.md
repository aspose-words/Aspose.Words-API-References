## JsonDataLoadOptions.SimpleValueParseMode property

获取或设置在加载 JSON 时解析 JSON 简单值（空、布尔、数字、整数和字符串） 的模式。这种模式不会影响日期时间值的解析。默认值为 Loose.

```csharp
public JsonSimpleValueParseMode SimpleValueParseMode { get; set; }
```
