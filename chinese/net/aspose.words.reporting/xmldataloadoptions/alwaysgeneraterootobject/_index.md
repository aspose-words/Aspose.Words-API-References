## XmlDataLoadOptions.AlwaysGenerateRootObject property

获取或设置一个标志，该标志指示生成的数据源是否始终包含 XML 根 元素的对象。如果 XML 根元素没有属性并且其所有子元素都具有相同的名称，则默认情况下不会创建这样的对象 。

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

### 评论

默认值为`错误的`.
