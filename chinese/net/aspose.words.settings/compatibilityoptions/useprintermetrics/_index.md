## CompatibilityOptions.UsePrinterMetrics property

使用打印机指标显示文档。

```csharp
public bool UsePrinterMetrics { get; set; }
```

## 评论

打印机指标可能因使用的驱动程序而异。 例如，Windows“Microsoft OpenXPS 类驱动程序 2”和“Microsoft Print to PDF”提供的指标略有不同。 因此，如果启用此选项，最终文档的布局可能会发生变化。
