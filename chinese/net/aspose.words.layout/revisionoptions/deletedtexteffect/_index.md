## RevisionOptions.DeletedTextEffect property

允许指定要应用于已删除内容的效果Deletion。 默认值为StrikeThrough

```csharp
public RevisionTextEffect DeletedTextEffect { get; set; }
```

### 例子

显示如何修改修订的外观。

```csharp
Document doc = new Document(MyDir + "Revisions.docx");

 // 获取控制修订外观的 RevisionOptions 对象。
RevisionOptions revisionOptions = doc.LayoutOptions.RevisionOptions;

 // 以绿色和斜体呈现插入修订。
revisionOptions.InsertedTextColor = RevisionColor.Green;
revisionOptions.InsertedTextEffect = RevisionTextEffect.Italic;

 // 以红色和粗体显示删除修订。
revisionOptions.DeletedTextColor = RevisionColor.Red;
revisionOptions.DeletedTextEffect = RevisionTextEffect.Bold;

 // 相同的文本会在一次移动修订中出现两次：
 // 一次在出发点，一次在到达目的地。
 // 使用双删除线将移出的修订版处的文本渲染为黄色 through
 // 并在移动到的修订版处加双下划线蓝色。
revisionOptions.MovedFromTextColor = RevisionColor.Yellow;
revisionOptions.MovedFromTextEffect = RevisionTextEffect.DoubleStrikeThrough;
revisionOptions.MovedToTextColor = RevisionColor.Blue;
revisionOptions.MovedFromTextEffect = RevisionTextEffect.DoubleUnderline;

// 以深红色和粗体呈现格式修订。
revisionOptions.RevisedPropertiesColor = RevisionColor.DarkRed;
revisionOptions.RevisedPropertiesEffect = RevisionTextEffect.Bold;

 // 在页面左侧受修订影响的行旁边放置一个深蓝色粗条。
revisionOptions.RevisionBarsColor = RevisionColor.DarkBlue;
revisionOptions.RevisionBarsWidth = 15.0f;

 // 显示修订标记和原文。
revisionOptions.ShowOriginalRevision = true;
revisionOptions.ShowRevisionMarks = true;

 // 获取移动、删除、格式化修订和注释以显示在绿色气球
 // 在页面的右侧。
revisionOptions.ShowInBalloons = ShowInBalloons.Format;
revisionOptions.CommentColor = RevisionColor.BrightGreen;

 // 这些特性只适用于.pdf或.jpg等格式。
doc.Save(ArtifactsDir + "Revision.RevisionOptions.pdf");
```
