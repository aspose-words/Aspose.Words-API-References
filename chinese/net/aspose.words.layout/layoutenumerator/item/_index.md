## LayoutEnumerator indexer

获取实体的命名属性。

```csharp
public object this[string key] { get; }
```

| 范围 | 描述 |
| --- | --- |
| key | 属性的名称（区分大小写）。 |

### 返回值

如果属性不可用则为 Null，否则为属性的值。

## 评论

这目前用于获取跨度的字体属性。 参见[`Font`](../../../aspose.words/font/)可能的属性名称的类。并非所有属性都受支持。
