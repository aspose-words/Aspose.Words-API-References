## VerticalAlignment enumeration

指定浮动形状、文本框或浮动表格的垂直对齐方式。

```csharp
public enum VerticalAlignment
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| None | `0` | 对象被显式定位，通常使用它的 **Top** 属性。 |
| Top | `1` | 指定对象应位于垂直对齐基础的顶部。 |
| Center | `2` | 指定对象应相对于垂直对齐基准居中。 |
| Bottom | `3` | 指定对象应位于垂直对齐基准的底部。 |
| Inside | `4` | 指定对象应位于水平对齐基础内。 |
| Outside | `5` | 指定对象应在垂直对齐基准之外。 |
| Inline | `-1` | 未记录。似乎是浮动段落和表格的可能值。 |
| Default | `0` | 与None相同。 |

### 例子

显示如何将浮动图像插入页面中心。

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// 插入将出现在重叠文本后面的浮动图像，并将其与页面中心对齐。
Shape shape = builder.InsertImage(ImageDir + "Logo.jpg");
shape.WrapType = WrapType.None;
shape.BehindText = true;
shape.RelativeHorizontalPosition = RelativeHorizontalPosition.Page;
shape.RelativeVerticalPosition = RelativeVerticalPosition.Page;
shape.HorizontalAlignment = HorizontalAlignment.Center;
shape.VerticalAlignment = VerticalAlignment.Center;

doc.Save(ArtifactsDir + "Image.CreateFloatingPageCenter.docx");
```
