## CompositeNode.IndexOf method

返回子节点数组中指定子节点的索引。

```csharp
public int IndexOf(Node child)
```

### 评论

如果在子节点中找不到该节点，则返回 -1。

Body body = doc.FirstSection.Body;

// 检索第一节正文中最后一段的索引。
