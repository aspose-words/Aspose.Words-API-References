## PageSetup.HeadingLevelForChapter property

获取或设置应用于文档中章节标题的标题级别样式。

```csharp
public int HeadingLevelForChapter { get; set; }
```

## 评论

可以是从 0 到 9 的数字。如果应用于页码，0 表示没有章节编号。

在创建包含章节编号的页码之前，文档标题必须应用编号大纲格式。
