## SectionLayoutMode enumeration

指定允许定义文档网格行为的部分的布局模式。

```csharp
public enum SectionLayoutMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Default | `0` | 指定不应将文档网格应用于文档中相应部分的内容。 |
| Grid | `1` | 指定相应的部分应将附加行间距和字符间距 添加到其中的每一行和每个字符中，以保持特定数量 每页行数和每行字符数。 输入时字符不会自动与网格线对齐。 |
| LineGrid | `2` | 指定相应部分应在其中的每一行中添加额外的行距 以保持每页指定的行数。 |
| SnapToChars | `3` | 指定相应的部分应将附加行间距和字符间距 添加到其中的每一行和每个字符中，以保持特定数量 每页行数和每行字符数。 键入时字符将自动与网格线对齐。 |

### 例子

显示如何为每行可能包含的字符数指定 a。

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// 启用pitching，然后用它来设置本节每页的行数。
// 足够大的字体大小会将一些行向下推到下一页以避免字符重叠。
builder.PageSetup.LayoutMode = SectionLayoutMode.LineGrid;
builder.PageSetup.LinesPerPage = 15;

builder.ParagraphFormat.SnapToGrid = true;

for (int i = 0; i < 30; i++)
    builder.Write("Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. ");

doc.Save(ArtifactsDir + "PageSetup.LinesPerPage.docx");
```
