## SaveOptions.CustomTimeZoneInfo property

获取或设置用于日期/时间字段的自定义本地时区。

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

## 评论

此选项在 3.5 版及以上的 .Net 框架或 .Net Standard 中均可用。

默认情况下，Aspose.Words 在写入日期/时间字段时使用系统本地时区，此选项允许设置自定义值。
