## MarkdownLinkExportMode enumeration

导出目标文档链接的模式。

```csharp
public enum MarkdownLinkExportMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Auto | `0` | 如果链接具有往返信息或在文档中多次提及 ，则该链接将导出为参考块。在所有其他情况下，链接将导出为内联块。 |
| Inline | `1` | 链接导出为内联块。 |
| Reference | `2` | 链接导出为参考块。 |
