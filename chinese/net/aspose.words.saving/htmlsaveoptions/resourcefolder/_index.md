## HtmlSaveOptions.ResourceFolder property

指定在将文档 导出为 HTML 时保存所有资源（如图像、字体和外部 CSS）的物理文件夹。默认为空字符串。

```csharp
public string ResourceFolder { get; set; }
```

### 评论

`ResourceFolder`是指定应写入所有资源的文件夹的最简单方法。 另一种方法是使用单个属性[`FontsFolder`](../fontsfolder),[`ImagesFolder`](../imagesfolder), 和[`CssStyleSheetFileName`](../cssstylesheetfilename)。

`ResourceFolder`的优先级低于通过P指定的文件夹:Aspose.Words.Saving.HtmlSaveOptions.FontsFolder, [`ImagesFolder`](../imagesfolder)和CssStyleSheetFileName。例如，如果 `ResourceFolder`和[`FontsFolder`](../fontsfolder)指定，字体将保存 到[`FontsFolder`](../fontsfolder)，而图像和 CSS 将保存到`ResourceFolder`。

如果`ResourceFolder`指定的文件夹不存在，则会自动创建.

### 例子

显示如何为外部保存的资源设置文件夹和文件夹别名Aspose.Words 在将文档保存为 HTML 时将创建。

```csharp
Document doc = new Document(MyDir + "Rendering.docx");

HtmlSaveOptions options = new HtmlSaveOptions
{
    CssStyleSheetType = CssStyleSheetType.External,
    ExportFontResources = true,
    ImageResolution = 72,
    FontResourcesSubsettingSizeThreshold = 0,
    FontsFolder = ArtifactsDir + "Fonts",
    ImagesFolder = ArtifactsDir + "Images",
    ResourceFolder = ArtifactsDir + "Resources",
    FontsFolderAlias = "http://example.com/fonts",
    ImagesFolderAlias = "http://example.com/images",
    ResourceFolderAlias = "http://example.com/resources",
    ExportOriginalUrlForLinkedImages = true
};

doc.Save(ArtifactsDir + "HtmlSaveOptions.FolderAlias.html", options);
```
