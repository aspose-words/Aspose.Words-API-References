## HtmlSaveOptions.NavigationMapLevel property

指定导出为 EPUB、MOBI 或 AZW3 格式时填充到导航地图的标题的最大级别。默认值为`3`.

```csharp
public int NavigationMapLevel { get; set; }
```

## 评论

导航图允许用户代理提供一种简单的文档结构导航方式。 通常导航点对应于文档中的标题。为了将标题填充到级别**氮** 将此值分配给`NavigationMapLevel`。

默认情况下，填充三个级别的标题：样式段落**标题 1**,**标题 2** 和**标题 3**. 您可以将此属性设置为 1 到 9 之间的值，以请求相应的最大级别。 将其设置为零会将导航图缩小为仅文档根或文档部分的根。
