## RtfSaveOptions.ExportImagesForOldReaders property

指定“老读者”的关键字是否写入 RTF。 这会显着影响 RTF 文档的大小。 默认值为` true` 。

```csharp
public bool ExportImagesForOldReaders { get; set; }
```

### 评论

“老读者”是 Microsoft Word 97 之前的应用程序以及写字板。 当此选项为` true` Aspose.Words 写入额外的 RTF 关键字。 这些关键字允许在 "old reader" 应用程序中打开文档时正确显示文档，但会显着增加文档的大小。

如果将此选项设置为` false` ，则只有 WMF、EMF 和 BMP 格式的图像 将显示在“老读者”中。

### 例子

显示如何使用自定义选项将文档保存为 .rtf。

```csharp
Document doc = new Document(MyDir + "Rendering.docx");

 // 创建一个“RtfSaveOptions”对象以传递给文档的“Save”方法来修改我们如何将其保存到 RTF.
RtfSaveOptions options = new RtfSaveOptions();

Assert.AreEqual(SaveFormat.Rtf, options.SaveFormat);

// 将“ExportCompactSize”属性设置为“true” to
 // 以从右到左的文本兼容性为代价减小保存文档的大小。
options.ExportCompactSize = true;

 // 将“ExportImagesFotOldReaders”属性设置为“true”以使用额外的关键字来确保我们的文档是
 // 与 Microsoft Word 97 之前的阅读器和写字板兼容。
 // 将“ExportImagesFotOldReaders”属性设置为“false”以减小文档的大小，
 // 但要防止老读者能够读取文档可能包含的任何非元文件或 BMP 图像。
options.ExportImagesForOldReaders = exportImagesForOldReaders;

doc.Save(ArtifactsDir + "RtfSaveOptions.ExportImages.rtf", options);
```
