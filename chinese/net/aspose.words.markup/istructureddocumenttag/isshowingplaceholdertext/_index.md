## IStructuredDocumentTag.IsShowingPlaceholderText property

指定此内容是否**特殊测试**应被解释为包含占位符 text （与 SDT 中的常规文本内容相反）。

如果设置为 true，则打开此文档时应恢复此状态（显示占位符文本）。

```csharp
public bool IsShowingPlaceholderText { get; set; }
```
