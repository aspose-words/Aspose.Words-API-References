## CellFormat.HideMark property

返回或设置单元格标记的可见性。

```csharp
public bool HideMark { get; set; }
```

## 评论

指定如果行中的所有单元格都为空，则表格单元格内容呈现为无高度； 但是，如果单元格边框、单元格边距或单元格间距非零，则单元格具有可见高度。
