Das **Aspose.Words.BuildingBlocks** namespace stellt Klassen bereit, die den Zugriff auf und die Verwendung von AutoText, AutoCorrect-Einträgen und Bausteinen in einem Dokument ermöglichen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [BuildingBlock](./buildingblock/) | Stellt einen Glossardokumenteintrag dar, z. B. einen Baustein-, AutoText- oder AutoKorrektur-Eintrag. |
| [BuildingBlockCollection](./buildingblockcollection/) | Eine Sammlung von[`BuildingBlock`](../aspose.words.buildingblocks/buildingblock/) Objekte im Dokument. |
| [GlossaryDocument](./glossarydocument/) | Stellt das Stammelement für ein Glossardokument innerhalb eines Word-Dokuments dar. Ein Glossardokument ist ein Speicher für AutoText, AutoKorrektur-Einträge und Bausteine. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [BuildingBlockBehavior](./buildingblockbehavior/) | Gibt das Verhalten an, das auf den Inhalt des Bausteins angewendet werden soll, wenn er in das Hauptdokument eingefügt wird. |
| [BuildingBlockGallery](./buildingblockgallery/) | Gibt die vordefinierte Galerie an, in die ein Baustein eingeordnet wird. |
| [BuildingBlockType](./buildingblocktype/) | Gibt einen Bausteintyp an. Der Typ kann die Sichtbarkeit und das Verhalten des Bausteins in Microsoft Word beeinflussen. |
