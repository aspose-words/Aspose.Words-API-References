Der **Aspose.Words.Fonts** Der Namespace stellt Klassen und Aufzählungen bereit, um auf Informationen zu in einem Dokument verwendeten x000d-Schriftarten zuzugreifen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [DefaultFontSubstitutionRule](./defaultfontsubstitutionrule/) | Standardregel zum Ersetzen von Schriftarten. |
| [FileFontSource](./filefontsource/) | Stellt die einzelne TrueType-Schriftartdatei dar, die im Dateisystem gespeichert ist. |
| [FolderFontSource](./folderfontsource/) | Stellt den Ordner dar, der TrueType-Schriftartendateien enthält. |
| [FontConfigSubstitutionRule](./fontconfigsubstitutionrule/) | Schriftartkonfigurations-Ersetzungsregel. |
| [FontFallbackSettings](./fontfallbacksettings/) | Gibt die Einstellungen für den Schriftart-Fallback-Mechanismus an. |
| [FontInfo](./fontinfo/) | Gibt Informationen zu einer im Dokument verwendeten Schriftart an. |
| [FontInfoCollection](./fontinfocollection/) | Stellt eine Sammlung von Schriftarten dar, die in einem Dokument verwendet werden. |
| [FontInfoSubstitutionRule](./fontinfosubstitutionrule/) | Schriftart-Info-Ersetzungsregel. |
| [FontNameSubstitutionRule](./fontnamesubstitutionrule/) | Schriftartersetzungsregel für die Verarbeitung des Schriftartnamens. |
| [FontSettings](./fontsettings/) | Gibt Schriftarteinstellungen für ein Dokument an. |
| [FontSourceBase](./fontsourcebase/) | Dies ist eine abstrakte Basisklasse für die Klassen, die es dem Benutzer ermöglichen, verschiedene Schriftartquellen anzugeben. |
| [FontSubstitutionRule](./fontsubstitutionrule/) | Dies ist eine abstrakte Basisklasse für die Schriftartersetzungsregel. |
| [FontSubstitutionSettings](./fontsubstitutionsettings/) | Gibt die Einstellungen für den Schriftartersetzungsmechanismus an. |
| [MemoryFontSource](./memoryfontsource/) | Stellt die einzelne im Speicher gespeicherte TrueType-Schriftartdatei dar. |
| [PhysicalFontInfo](./physicalfontinfo/) | Gibt Informationen über die physische Schriftart an, die für die Aspose.Words-Schriftart-Engine verfügbar ist. |
| [StreamFontSource](./streamfontsource/) | Basisklasse für benutzerdefinierte Stream-Schriftartquelle. |
| [SystemFontSource](./systemfontsource/) | Stellt alle auf dem System installierten TrueType-Schriftarten dar. |
| [TableSubstitutionRule](./tablesubstitutionrule/) | Regel zum Ersetzen von Tabellenschriftarten. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [EmbeddedFontFormat](./embeddedfontformat/) | Gibt das Format einer bestimmten darin eingebetteten Schriftart an[`FontInfo`](../aspose.words.fonts/fontinfo/) Objekt. |
| [EmbeddedFontStyle](./embeddedfontstyle/) | Gibt den Stil einer eingebetteten Schriftart in a an[`FontInfo`](../aspose.words.fonts/fontinfo/) Objekt. |
| [FontFamily](./fontfamily/) | Stellt die Schriftfamilie dar. |
| [FontPitch](./fontpitch/) | Stellt den Schriftabstand dar. |
| [FontSourceType](./fontsourcetype/) | Gibt den Typ einer Schriftartquelle an. |
