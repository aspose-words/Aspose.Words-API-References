Der**Aspose.Words.Fonts** Der Namespace bietet Klassen und Aufzählungen für den Zugriff auf Informationen zu den in einem Dokument verwendeten x000d-Schriftarten.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [DefaultFontSubstitutionRule](./defaultfontsubstitutionrule/) | Standardregel zur Schriftartersetzung. |
| [FileFontSource](./filefontsource/) | Stellt die einzelne im Dateisystem gespeicherte TrueType-Schriftdatei dar. |
| [FolderFontSource](./folderfontsource/) | Stellt den Ordner dar, der TrueType-Schriftdateien enthält. |
| [FontConfigSubstitutionRule](./fontconfigsubstitutionrule/) | Regel zur Ersetzung der Schriftartkonfiguration. |
| [FontEmbeddingLicensingRights](./fontembeddinglicensingrights/) | Stellt die Einbettungslizenzrechte für die Schriftart dar. |
| [FontFallbackSettings](./fontfallbacksettings/) | Gibt die Einstellungen für den Fallback-Mechanismus für Schriftarten an. |
| [FontInfo](./fontinfo/) | Gibt Informationen zu einer im Dokument verwendeten Schriftart an. |
| [FontInfoCollection](./fontinfocollection/) | Stellt eine Sammlung von Schriftarten dar, die in einem Dokument verwendet werden. |
| [FontInfoSubstitutionRule](./fontinfosubstitutionrule/) | Regel zur Ersetzung von Schriftinformationen. |
| [FontNameSubstitutionRule](./fontnamesubstitutionrule/) | Schriftartersetzungsregel zur Verarbeitung des Schriftartnamens. |
| [FontSettings](./fontsettings/) | Gibt die Schriftarteinstellungen für ein Dokument an. |
| [FontSourceBase](./fontsourcebase/) | Dies ist eine abstrakte Basisklasse für die Klassen, die es dem Benutzer ermöglichen, verschiedene Schriftartquellen anzugeben. |
| [FontSubstitutionRule](./fontsubstitutionrule/) | Dies ist eine abstrakte Basisklasse für die Schriftartersetzungsregel. |
| [FontSubstitutionSettings](./fontsubstitutionsettings/) | Gibt die Einstellungen für den Mechanismus zur Schriftartersetzung an. |
| [MemoryFontSource](./memoryfontsource/) | Stellt die einzelne im Speicher gespeicherte TrueType-Schriftdatei dar. |
| [PhysicalFontInfo](./physicalfontinfo/) | Gibt Informationen über die physische Schriftart an, die der Aspose.Words-Schriftarten-Engine zur Verfügung steht. |
| [StreamFontSource](./streamfontsource/) | Basisklasse für benutzerdefinierte Stream-Font-Quelle. |
| [SystemFontSource](./systemfontsource/) | Stellt alle im System installierten TrueType-Schriftarten dar. |
| [TableSubstitutionRule](./tablesubstitutionrule/) | Regel zur Ersetzung von Tabellenschriftarten. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [EmbeddedFontFormat](./embeddedfontformat/) | Gibt das Format einer bestimmten eingebetteten Schriftart an[`FontInfo`](../aspose.words.fonts/fontinfo/) Objekt. |
| [EmbeddedFontStyle](./embeddedfontstyle/) | Gibt den Stil einer eingebetteten Schriftart in einem[`FontInfo`](../aspose.words.fonts/fontinfo/) Objekt. |
| [FontEmbeddingUsagePermissions](./fontembeddingusagepermissions/) | Stellt die Nutzungsberechtigungen für die Schriftarteinbettung dar. |
| [FontFamily](./fontfamily/) | Stellt die Schriftfamilie dar. |
| [FontPitch](./fontpitch/) | Stellt die Schriftgröße dar. |
| [FontSourceType](./fontsourcetype/) | Gibt den Typ der Schriftartquelle an. |
