## StructuredDocumentTagCollection.GetByTag method

Gibt das erste strukturierte Dokument-Tag zurück, das in der Sammlung mit dem angegebenen Tag gefunden wurde.

```csharp
public IStructuredDocumentTag GetByTag(string tag)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| tag | String | Das Tag des strukturierten Dokument-Tags. |
