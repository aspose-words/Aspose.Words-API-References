## SdtCalendarType enumeration

Gibt die möglichen Kalendertypen an, die zur Angabe verwendet werden können[`CalendarType`](../structureddocumenttag/calendartype/) in einem Office Open XML-Dokument.

```csharp
public enum SdtCalendarType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | `0` | Wird als Standardwert in OOXML verwendet. GleichGregorian . |
| Gregorian | `0` | Gibt an, dass der gregorianische Kalender gemäß ISO 8601 verwendet werden soll. Dieser Kalender sollte in die entsprechende Sprache lokalisiert werden. |
| GregorianArabic | `1` | Gibt an, dass der Gregorianische Kalender gemäß ISO 8601 verwendet werden soll. Die Werte für diesen Kalender sollten auf Arabisch dargestellt werden. |
| GregorianMeFrench | `2` | Gibt an, dass der gregorianische Kalender gemäß ISO 8601 verwendet werden soll. Die Werte für diesen Kalender sollten in Französisch für den Nahen Osten dargestellt werden. |
| GregorianUs | `3` | Gibt an, dass der gregorianische Kalender gemäß ISO 8601 verwendet werden soll. Die Werte für diesen Kalender sollten auf Englisch dargestellt werden. |
| GregorianXlitEnglish | `4` | Gibt an, dass der Gregorianische Kalender gemäß ISO 8601 verwendet werden soll. Die Werte für diesen Kalender sollten die Darstellung der englischen Zeichenfolgen in den entsprechenden arabischen Zeichen sein (die arabische Transliteration des Englischen für den Gregorianischen Kalender). |
| GregorianXlitFrench | `5` | Gibt an, dass der Gregorianische Kalender gemäß ISO 8601 verwendet werden soll. Die Werte für diesen Kalender sollten die Darstellung der französischen Zeichenfolgen in den entsprechenden arabischen Zeichen sein (die arabische Transliteration des Französischen für den Gregorianischen Kalender). |
| Hebrew | `6` | Gibt an, dass der hebräische Mondkalender, wie er durch die Gauß-Formel für Pessach [CITATION] und die vollständige Neuformulierung des mündlichen Gesetzes (Mischne Tora) beschrieben wird, verwendet werden soll. |
| Hijri | `7` | Gibt an, dass der Hijri-Mondkalender, wie vom Königreich Saudi-Arabien, Ministerium für Islamische Angelegenheiten, Stiftungen, Da'wa und Führung beschrieben, verwendet werden soll. |
| Japan | `8` | Gibt an, dass der Kalender der japanischen Kaiserzeit, wie im japanischen Industriestandard JIS X 0301 beschrieben, verwendet werden soll. |
| Korea | `9` | Gibt an, dass der Kalender der koreanischen Tangun-Ära, , wie in der koreanischen Gesetzesverordnung Nr. 4 beschrieben, verwendet werden soll. |
| None | `10` | Gibt an, dass kein Kalender verwendet werden soll. |
| Saka | `11` | Gibt an, dass der Kalender der Saka-Ära, wie vom Calendar Reform Committee of India beschrieben, als Teil der indischen Ephemeride und des Nautischen Almanachs verwendet werden soll. |
| Taiwan | `12` | Gibt an, dass der taiwanesische Kalender gemäß der Definition des chinesischen Nationalstandards CNS 7648 verwendet werden soll. |
| Thai | `13` | Gibt an, dass der thailändische Kalender, wie durch den königlichen Erlass seiner Majestät König Vajiravudh (Rama VI) in Royal Gazette BE 2456 (1913 n. Chr.) und durch den Erlass von Premierminister Phibunsongkhram (1941 n. Chr.) definiert, das Jahr am beginnen soll Der gregorianische 1. Januar und die Zuordnung des Jahres Null zum gregorianischen Jahr 543 v. Chr. werden verwendet. |
