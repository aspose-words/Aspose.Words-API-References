Der**Aspose.Words.Markup** Der Namespace enthält Klassen, die vom Kunden definierte Semantik in einem Dokument darstellen: Smarttags, benutzerdefiniertes XML und strukturierte Dokument-Tags (Inhaltssteuerelemente).

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [CustomPart](./custompart/) | Stellt einen benutzerdefinierten Teil (mit beliebigem Inhalt) dar, der nicht durch den ISO/IEC 29500-Standard definiert ist. |
| [CustomPartCollection](./custompartcollection/) | Stellt eine Sammlung von[`CustomPart`](../aspose.words.markup/custompart/) Objekte. |
| [CustomXmlPart](./customxmlpart/) | Stellt einen benutzerdefinierten XML-Datenspeicherteil dar (benutzerdefinierte XML-Daten innerhalb eines Pakets). |
| [CustomXmlPartCollection](./customxmlpartcollection/) | Stellt eine Sammlung von benutzerdefinierten XML-Teilen dar. Die Elemente sind[`CustomXmlPart`](../aspose.words.markup/customxmlpart/) Objekte. |
| [CustomXmlProperty](./customxmlproperty/) | Stellt ein einzelnes benutzerdefiniertes XML-Attribut oder eine Smarttag-Eigenschaft dar. |
| [CustomXmlPropertyCollection](./customxmlpropertycollection/) | Stellt eine Sammlung benutzerdefinierter XML-Attribute oder Smarttag-Eigenschaften dar. |
| [CustomXmlSchemaCollection](./customxmlschemacollection/) | Eine Sammlung von Zeichenfolgen, die XML-Schemas darstellen, die einem benutzerdefinierten XML-Teil zugeordnet sind. |
| [SdtListItem](./sdtlistitem/) | Dieses Element gibt ein einzelnes Listenelement innerhalb eines übergeordnetenComboBox oderDropDownList strukturiertes Dokument-Tag. |
| [SdtListItemCollection](./sdtlistitemcollection/) | Bietet Zugriff auf[`SdtListItem`](../aspose.words.markup/sdtlistitem/) Elemente eines strukturierten Dokument-Tags. |
| [SmartTag](./smarttag/) | Dieses Element gibt das Vorhandensein eines Smarttags um eine oder mehrere Inline-Strukturen (Läufe, Bilder, Felder usw.) innerhalb eines Absatzes an. |
| [StructuredDocumentTag](./structureddocumenttag/) | Stellt ein strukturiertes Dokument-Tag (SDT oder Inhaltssteuerelement) in einem Dokument dar. |
| [StructuredDocumentTagCollection](./structureddocumenttagcollection/) | Eine Sammlung von[`IStructuredDocumentTag`](../aspose.words.markup/istructureddocumenttag/) Instanzen, die die strukturierten Dokument-Tags im angegebenen Bereich darstellen. |
| [StructuredDocumentTagRangeEnd](./structureddocumenttagrangeend/) | Stellt ein Ende von**reichte** strukturiertes Dokument-Tag, das Inhalte mit mehreren Abschnitten akzeptiert. Siehe auch[`StructuredDocumentTagRangeStart`](../aspose.words.markup/structureddocumenttagrangestart/) Knoten. |
| [StructuredDocumentTagRangeStart](./structureddocumenttagrangestart/) | Stellt einen Anfang von**reichte** strukturiertes Dokument-Tag, das Inhalte mit mehreren Abschnitten akzeptiert. Siehe auch[`StructuredDocumentTagRangeEnd`](../aspose.words.markup/structureddocumenttagrangeend/) . |
| [XmlMapping](./xmlmapping/) | Gibt die Informationen an, die zum Erstellen einer Zuordnung zwischen dem übergeordneten strukturierten Dokumenttag und einem XML-Element verwendet werden, das in einem benutzerdefinierten XML-Datenteil im Dokument gespeichert ist. |
## Schnittstellen

| Schnittstelle | Beschreibung |
| --- | --- |
| [IStructuredDocumentTag](./istructureddocumenttag/) | Schnittstelle zur Definition gemeinsamer Daten für[`StructuredDocumentTag`](../aspose.words.markup/structureddocumenttag/) Und[`StructuredDocumentTagRangeStart`](../aspose.words.markup/structureddocumenttagrangestart/) . |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [MarkupLevel](./markuplevel/) | Gibt die Ebene im Dokumentbaum an, auf der ein bestimmter[`StructuredDocumentTag`](../aspose.words.markup/structureddocumenttag/) kann vorkommen. |
| [SdtAppearance](./sdtappearance/) | Gibt das Erscheinungsbild eines strukturierten Dokument-Tags an. |
| [SdtCalendarType](./sdtcalendartype/) | Gibt die möglichen Kalendertypen an, die verwendet werden können, um[`CalendarType`](../aspose.words.markup/structureddocumenttag/calendartype/) in einem Office Open XML-Dokument. |
| [SdtDateStorageFormat](./sdtdatestorageformat/) | Gibt an, wie das Datum für ein Datums-SDT gespeichert/abgerufen wird, wenn das SDT an einen XML-Knoten im Datenspeicher des Dokuments gebunden ist. |
| [SdtType](./sdttype/) | Gibt den Typ eines SDT-Knotens (Structured Document Tag) an. |
