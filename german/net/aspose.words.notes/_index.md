Der**Aspose.Wörter.Notizen** Der Namespace bietet Klassen zum Arbeiten mit Endnoten und Fußnoten von Microsoft Word.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [EndnoteOptions](./endnoteoptions/) | Stellt die Endnotennummerierungsoptionen für ein Dokument oder einen Abschnitt dar. |
| [Footnote](./footnote/) | Stellt einen Container für Text einer Fußnote oder Endnote dar. |
| [FootnoteOptions](./footnoteoptions/) | Stellt die Optionen zur Fußnotennummerierung für ein Dokument oder einen Abschnitt dar. |
| [FootnoteSeparator](./footnoteseparator/) | Stellt einen Container für das Fußnoten-/Endnotentrennzeichen und den Fortsetzungsinhalt eines Dokuments dar. |
| [FootnoteSeparatorCollection](./footnoteseparatorcollection/) | Bietet typisierten Zugriff auf[`FootnoteSeparator`](../aspose.words.notes/footnoteseparator/) Knoten eines Dokuments. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [EndnotePosition](./endnoteposition/) | Definiert die Endnotenposition. |
| [FootnoteNumberingRule](./footnotenumberingrule/) | Bestimmt, wann die automatische Fußnoten- oder Endnotennummerierung neu gestartet wird. |
| [FootnotePosition](./footnoteposition/) | Definiert die Fußnotenposition. |
| [FootnoteSeparatorType](./footnoteseparatortype/) | Gibt den Typ des Fußnoten-/Endnotentrennzeichens an. |
| [FootnoteType](./footnotetype/) | Gibt an, ob es sich um eine Fußnote oder eine Endnote handelt. |
