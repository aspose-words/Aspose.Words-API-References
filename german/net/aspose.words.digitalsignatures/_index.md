Der**Aspose.Words.DigitalSignatures** Der Namespace bietet Klassen zum Signieren von Dokumenten und Überprüfen von Signaturen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [CertificateHolder](./certificateholder/) | Stellt einen Inhaber von**X509Zertifikat2** Instanz. |
| [DigitalSignature](./digitalsignature/) | Stellt eine digitale Signatur auf einem Dokument und das Ergebnis ihrer Überprüfung dar. |
| [DigitalSignatureCollection](./digitalsignaturecollection/) | Bietet eine schreibgeschützte Sammlung digitaler Signaturen, die an ein Dokument angehängt sind. |
| [DigitalSignatureUtil](./digitalsignatureutil/) | Bietet Methoden zum Signieren von Dokumenten. |
| [SignOptions](./signoptions/) | Ermöglicht die Angabe von Optionen für die Dokumentsignatur. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [DigitalSignatureType](./digitalsignaturetype/) | Gibt den Typ einer digitalen Signatur an. |
| [XmlDsigLevel](./xmldsiglevel/) | Gibt die Ebene einer digitalen Signatur basierend auf dem XML-DSig-Standard an. |
