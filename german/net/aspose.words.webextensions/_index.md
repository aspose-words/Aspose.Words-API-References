Der**Aspose.Words.WebExtensions**Der Namespace stellt Klassen bereit, die das Anpassen von Elementen und Attributen ermöglichen, die das XML-Vokabular zur Darstellung von Office-Add-Ins erweitern.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [BaseWebExtensionCollection&lt;T&gt;](./basewebextensioncollection-1/) | Basisklasse für[`TaskPaneCollection`](../aspose.words.webextensions/taskpanecollection/) ,[`WebExtensionBindingCollection`](../aspose.words.webextensions/webextensionbindingcollection/) , [`WebExtensionPropertyCollection`](../aspose.words.webextensions/webextensionpropertycollection/) Und[`WebExtensionReferenceCollection`](../aspose.words.webextensions/webextensionreferencecollection/) Sammlungen. |
| [TaskPane](./taskpane/) | Stellt ein Add-In-Aufgabenbereichsobjekt dar. |
| [TaskPaneCollection](./taskpanecollection/) | Gibt eine Liste persistenter Aufgabenbereichsobjekte an. |
| [WebExtension](./webextension/) | Stellt ein Web-Erweiterungsobjekt dar. |
| [WebExtensionBinding](./webextensionbinding/) | Gibt eine Bindungsbeziehung zwischen einer Weberweiterung und den Daten im Dokument an. |
| [WebExtensionBindingCollection](./webextensionbindingcollection/) | Gibt eine Liste von Web-Erweiterungsbindungen an. |
| [WebExtensionProperty](./webextensionproperty/) | Gibt eine benutzerdefinierte Eigenschaft einer Web-Erweiterung an. |
| [WebExtensionPropertyCollection](./webextensionpropertycollection/) | Gibt eine Reihe benutzerdefinierter Eigenschaften für Web-Erweiterungen an. |
| [WebExtensionReference](./webextensionreference/) | Stellt den Verweis auf eine Web-Erweiterung dar. Der Verweis dient zur Identifizierung des Anbieterstandorts und der Version der Erweiterung. |
| [WebExtensionReferenceCollection](./webextensionreferencecollection/) | Gibt eine Liste mit Web-Erweiterungsreferenzen an. |
