## ImageFieldMergingArgs.Shape property

Gibt die Form an, die das Serienbriefmodul in das Dokument einfügen muss.

```csharp
public Shape Shape { get; set; }
```

### Bemerkungen

Wenn diese Eigenschaft angegeben ist, ignoriert die Serienbrief-Engine alle anderen Eigenschaften wie z[`ImageFileName`](../imagefilename) oder[`ImageStream`](../imagestream) und fügt die Form einfach in das Dokument ein.

Verwenden Sie diese Eigenschaft, um den Vorgang des Zusammenführens eines Bildzusammenführungsfelds vollständig zu steuern. Sie können beispielsweise angeben[`WrapType`](../../../aspose.words.drawing/shapebase/wraptype)oder jede andere Formeigenschaft, um den resultierenden Knoten fein abzustimmen. Beachten Sie jedoch, dass Sie für die Bereitstellung des Inhalts der Form verantwortlich sind.
