## CompatibilityOptions.UsePrinterMetrics property

Verwenden Sie Druckermetriken zum Anzeigen von Dokumenten.

```csharp
public bool UsePrinterMetrics { get; set; }
```

### Bemerkungen

Druckermetriken können je nach verwendeten Treibern unterschiedlich sein. Beispielsweise bieten Windows „Microsoft OpenXPS Class Driver 2“ und „Microsoft Print to PDF“ leicht unterschiedliche Metriken. Daher kann sich das Layout des endgültigen Dokuments ändern, wenn diese Option aktiviert ist.
