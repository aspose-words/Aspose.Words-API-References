## CompatibilityOptions.AutofitToFirstFixedWidthCell property

Tabellenspalten dürfen die bevorzugte Breite der einzelnen Zellen überschreiten.

```csharp
public bool AutofitToFirstFixedWidthCell { get; set; }
```

## Bemerkungen

Die Option heißt in der Benutzeroberfläche von MS Word 2013 „Word 2003-Tabellen-Autofit-Regeln verwenden“. Sie wirkt sich tatsächlich auch darauf aus, wie das Raster für Tabellen mit festem Layout berechnet wird (in einigen Fällen).
