## SaveOptions.CustomTimeZoneInfo property

Ruft eine benutzerdefinierte lokale Zeitzone ab oder legt sie fest, die für Datums-/Uhrzeitfelder verwendet wird.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

### Bemerkungen

Diese Option ist entweder in .Net Framework ab Version 3.5 oder in .Net Standard verfügbar.

Standardmäßig verwendet Aspose.Words die lokale Zeitzone des Systems, wenn Datums-/Uhrzeitfelder geschrieben werden. Diese Option ermöglicht das Festlegen eines benutzerdefinierten Werts.
