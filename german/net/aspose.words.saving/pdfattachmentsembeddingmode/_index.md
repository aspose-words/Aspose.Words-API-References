## PdfAttachmentsEmbeddingMode enumeration

Gibt an, wie Anhänge in PDF-Dokumente eingebettet werden.

```csharp
public enum PdfAttachmentsEmbeddingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Anhänge sind nicht eingebettet. |
| Annotations | `1` | Anhänge werden als Anmerkungen mit Position auf der Seite eingebettet. |
| DocumentEmbeddedFiles | `2` | Anhänge werden als in das Dokument eingebettete Dateien ohne Anmerkungen eingebettet. |
