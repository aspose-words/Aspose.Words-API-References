## TableContentAlignment enumeration

Ermöglicht die Festlegung der Ausrichtung des Inhalts der Tabelle, die beim Export in das Markdown-Format verwendet werden soll.

```csharp
public enum TableContentAlignment
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Auto | `0` | Die Ausrichtung wird vom ersten Absatz in der entsprechenden Tabellenspalte genommen. |
| Left | `1` | Der Inhalt von Tabellen wird linksbündig ausgerichtet. |
| Center | `2` | Der Inhalt von Tabellen wird an der Mitte ausgerichtet. |
| Right | `3` | Der Inhalt von Tabellen wird rechtsbündig ausgerichtet. |
