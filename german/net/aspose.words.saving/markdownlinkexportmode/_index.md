## MarkdownLinkExportMode enumeration

Der Modus zum Exportieren von Links zu einem Zieldokument.

```csharp
public enum MarkdownLinkExportMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Auto | `0` | Ein Link wird als Referenzblock exportiert, wenn er über Roundtrip-Informationen verfügt oder mehr als einmal in einem Dokument erwähnt wird. In allen anderen Fällen wird ein Link als Inline-Block exportiert. |
| Inline | `1` | Links werden als Inline-Blöcke exportiert. |
| Reference | `2` | Links werden als Referenzblöcke exportiert. |
