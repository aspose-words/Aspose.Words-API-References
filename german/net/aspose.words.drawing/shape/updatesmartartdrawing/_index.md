## Shape.UpdateSmartArtDrawing method

Aktualisiert vorgerenderte SmartArt-Zeichnungen mithilfe der SmartArt-Cold-Rendering-Engine von Aspose.Words.

```csharp
public void UpdateSmartArtDrawing()
```

### Bemerkungen

Microsoft Word generiert und speichert die vorgerenderte Zeichnung zusammen mit dem SmartArt-Objekt. Wenn das Dokument jedoch von anderen Anwendungen gespeichert wird, kann die vorgerenderte SmartArt-Zeichnung fehlen oder falsch sein. Wenn eine vorgerenderte Zeichnung verfügbar ist, verwendet Aspose.Words sie zum Rendern des SmartArt-Objekts. Wenn eine vorgerenderte Zeichnung vorhanden ist fehlt, dann verwendet Aspose.Words seine eigene SmartArt-Engine für kaltes Rendering, um das SmartArt-Objekt zu rendern. Wenn die vorgerenderte Zeichnung nicht korrekt ist, muss diese Methode aufgerufen werden, um die SmartArt-Engine für kaltes -Rendering aufzurufen.
