## CellFormat.HideMark property

Gibt die Sichtbarkeit der Zellenmarkierung zurück oder legt sie fest.

```csharp
public bool HideMark { get; set; }
```

## Bemerkungen

Gibt an, dass der Inhalt einer Tabellenzelle ohne Höhe dargestellt wird, wenn alle Zellen in der Zeile leer sind; Zellen haben jedoch eine sichtbare Höhe, wenn sie Zellränder, Zellabstände oder Zellabstände ungleich Null haben.
