Der**Aspose.Wörter.Tabellen** Der Namespace enthält Klassen, die Tabellen, Zeilen, Zellen und deren Formatierung darstellen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Cell](./cell/) | Stellt eine Tabellenzelle dar. |
| [CellCollection](./cellcollection/) | Bietet typisierten Zugriff auf eine Sammlung von[`Cell`](../aspose.words.tables/cell/) Knoten. |
| [CellFormat](./cellformat/) | Stellt die gesamte Formatierung für eine Tabellenzelle dar. |
| [PreferredWidth](./preferredwidth/) | Stellt einen Wert und seine Maßeinheit dar, der verwendet wird, um die bevorzugte Breite einer Tabelle oder Zelle anzugeben. |
| [Row](./row/) | Stellt eine Tabellenzeile dar. |
| [RowCollection](./rowcollection/) | Bietet typisierten Zugriff auf eine Sammlung von[`Row`](../aspose.words.tables/row/) Knoten. |
| [RowFormat](./rowformat/) | Stellt die gesamte Formatierung für eine Tabellenzeile dar. |
| [Table](./table/) | Stellt eine Tabelle in einem Word-Dokument dar. |
| [TableCollection](./tablecollection/) | Bietet typisierten Zugriff auf eine Sammlung von[`Table`](../aspose.words.tables/table/) Knoten. |
