Der**Aspose.Words.Themes** Der Namespace stellt Klassen bereit, die den Zugriff auf Designinformationen in Microsoft Word-Dokumenten ermöglichen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Theme](./theme/) | Stellt das Dokumentthema dar und bietet Zugriff auf die Hauptthementeile, einschließlich[`MajorFonts`](../aspose.words.themes/theme/majorfonts/) ,[`MinorFonts`](../aspose.words.themes/theme/minorfonts/) Und[`Colors`](../aspose.words.themes/theme/colors/) |
| [ThemeColors](./themecolors/) | Stellt das Farbschema des Dokumentdesigns dar, das zwölf Farben enthält. |
| [ThemeFonts](./themefonts/) | Stellt eine Sammlung von Schriftarten im Schriftartenschema dar und ermöglicht die Angabe verschiedener Schriftarten für verschiedene Sprachen[`Latin`](../aspose.words.themes/themefonts/latin/) ,[`EastAsian`](../aspose.words.themes/themefonts/eastasian/) Und[`ComplexScript`](../aspose.words.themes/themefonts/complexscript/) . |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [ThemeColor](./themecolor/) | Gibt die Designfarben für Dokumentdesigns an. |
| [ThemeFont](./themefont/) | Gibt die Arten von Designschriftartennamen für Dokumentdesigns an. |
