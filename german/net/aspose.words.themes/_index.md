Der**Aspose.Wörter.Themen** Der Namespace stellt Klassen bereit, die den Zugriff auf Designinformationen in Microsoft Word-Dokumenten ermöglichen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Theme](./theme/) | Stellt das Dokumentdesign dar und bietet Zugriff auf die wichtigsten Designteile, einschließlich[`MajorFonts`](../aspose.words.themes/theme/majorfonts/) ,[`MinorFonts`](../aspose.words.themes/theme/minorfonts/) Und[`Colors`](../aspose.words.themes/theme/colors/) |
| [ThemeColors](./themecolors/) | Stellt das Farbschema des Dokumentdesigns dar, das zwölf Farben enthält. |
| [ThemeFonts](./themefonts/) | Stellt eine Sammlung von Schriftarten im Schriftartenschema dar, die es ermöglicht, unterschiedliche Schriftarten für unterschiedliche Sprachen anzugeben[`Latin`](../aspose.words.themes/themefonts/latin/) ,[`EastAsian`](../aspose.words.themes/themefonts/eastasian/) Und[`ComplexScript`](../aspose.words.themes/themefonts/complexscript/) . |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [ThemeColor](./themecolor/) | Gibt die Designfarben für Dokumentdesigns an. |
| [ThemeFont](./themefont/) | Gibt die Typen von Designschriftartennamen für Dokumentdesigns an. |
