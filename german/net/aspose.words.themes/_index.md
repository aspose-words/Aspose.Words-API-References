Das **Aspose.Words.Themes** namespace stellt Klassen bereit, die den Zugriff auf Themeninformationen in Microsoft Word-Dokumenten ermöglichen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Theme](./theme) | Stellt das Dokumentdesign dar und bietet Zugriff auf Hauptdesignteile, einschließlich[`MajorFonts`](../aspose.words.themes/theme/majorfonts) ,[`MinorFonts`](../aspose.words.themes/theme/minorfonts) und[`Colors`](../aspose.words.themes/theme/colors) |
| [ThemeColors](./themecolors) | Stellt das Farbschema des Dokumentdesigns dar, das zwölf Farben enthält. |
| [ThemeFonts](./themefonts) | Stellt eine Sammlung von Schriftarten im Schriftartenschema dar, wodurch unterschiedliche Schriftarten für verschiedene Sprachen angegeben werden können[`Latin`](../aspose.words.themes/themefonts/latin) ,[`EastAsian`](../aspose.words.themes/themefonts/eastasian) und[`ComplexScript`](../aspose.words.themes/themefonts/complexscript) . |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [ThemeColor](./themecolor) | Gibt die Designfarben für Dokumentdesigns an. |
| [ThemeFont](./themefont) | Gibt die Arten von Designschriftnamen für Dokumentdesigns an. |
