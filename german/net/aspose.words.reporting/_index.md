Der**Aspose.Words.Reporting** Der Namespace enthält Klassen der neuen Berichts-Engine von Aspose.Words, die Berichtsvorlagen unterstützt, die mit einer auf der LINQ-Methodensyntax basierenden Sprache markiert sind.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [CsvDataLoadOptions](./csvdataloadoptions/) | Stellt Optionen zum Parsen von CSV-Daten dar. |
| [CsvDataSource](./csvdatasource/) | Bietet Zugriff auf Daten einer CSV-Datei oder eines Streams zur Verwendung in einem Bericht. |
| [JsonDataLoadOptions](./jsondataloadoptions/) | Stellt Optionen zum Parsen von JSON-Daten dar. |
| [JsonDataSource](./jsondatasource/) | Bietet Zugriff auf Daten einer JSON-Datei oder eines JSON-Streams zur Verwendung in einem Bericht. |
| [KnownTypeSet](./knowntypeset/) | Stellt eine ungeordnete Menge dar (d. h. eine Sammlung eindeutiger Elemente), die Folgendes enthält:Type Objekte , deren vollständig oder teilweise qualifizierte Namen in Berichtsvorlagen verwendet werden können, um die entsprechenden statischen Member der Typen aufzurufen, Typumwandlungen durchzuführen usw. |
| [ReportingEngine](./reportingengine/) | Bietet Routinen zum Füllen von Vorlagendokumenten mit Daten und eine Reihe von Einstellungen zum Steuern dieser Routinen. |
| [XmlDataLoadOptions](./xmldataloadoptions/) | Stellt Optionen zum Laden von XML-Daten dar. |
| [XmlDataSource](./xmldatasource/) | Bietet Zugriff auf Daten einer XML-Datei oder eines XML-Streams zur Verwendung in einem Bericht. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [JsonSimpleValueParseMode](./jsonsimplevalueparsemode/) | Gibt einen Modus zum Parsen einfacher JSON-Werte (Null, Boolean, Zahl, Integer und String) beim Laden von JSON an. Dieser Modus hat keinen Einfluss auf das Parsen von Datums-/Uhrzeitwerten. |
| [ReportBuildOptions](./reportbuildoptions/) | Gibt Optionen an, die das Verhalten von[`ReportingEngine`](../aspose.words.reporting/reportingengine/) während der Erstellung eines Berichts. |
