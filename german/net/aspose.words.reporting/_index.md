Dieser Namespace enthält Klassen der neuen Berichts-Engine von Aspose.Words, die Berichtsvorlagen unterstützt, die mit einer Sprache gekennzeichnet sind, die auf der LINQ-Methodensyntax basiert.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [CsvDataLoadOptions](./csvdataloadoptions/) | Repräsentiert Optionen zum Analysieren von CSV-Daten. |
| [CsvDataSource](./csvdatasource/) | Bietet Zugriff auf Daten einer CSV-Datei oder eines Streams zur Verwendung in einem Bericht. |
| [JsonDataLoadOptions](./jsondataloadoptions/) | Repräsentiert Optionen zum Analysieren von JSON-Daten. |
| [JsonDataSource](./jsondatasource/) | Bietet Zugriff auf Daten einer JSON-Datei oder eines Streams zur Verwendung in einem Bericht. |
| [KnownTypeSet](./knowntypeset/) | Repräsentiert eine ungeordnete Menge (dh eine Sammlung einzigartiger Elemente), die Folgendes enthältType Objekte , deren vollständig oder teilweise qualifizierte Namen in Berichtsvorlagen verwendet werden können, um die statischen Member der entsprechenden -Typen aufzurufen, Typumwandlungen durchzuführen usw. |
| [ReportingEngine](./reportingengine/) | Bietet Routinen zum Füllen von Vorlagendokumenten mit Daten und eine Reihe von Einstellungen zum Steuern dieser Routinen. |
| [XmlDataLoadOptions](./xmldataloadoptions/) | Repräsentiert Optionen für das Laden von XML-Daten. |
| [XmlDataSource](./xmldatasource/) | Bietet Zugriff auf Daten einer XML-Datei oder eines Streams zur Verwendung in einem Bericht. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [JsonSimpleValueParseMode](./jsonsimplevalueparsemode/) | Gibt einen Modus zum Analysieren einfacher JSON-Werte (Null, Boolean, Zahl, Ganzzahl und Zeichenfolge) beim Laden von JSON an. Ein solcher Modus wirkt sich nicht auf die Analyse von Datums- und Uhrzeitwerten aus. |
| [ReportBuildOptions](./reportbuildoptions/) | Gibt Optionen an, die das Verhalten von steuern[`ReportingEngine`](../aspose.words.reporting/reportingengine/) beim Erstellen eines Berichts. |
