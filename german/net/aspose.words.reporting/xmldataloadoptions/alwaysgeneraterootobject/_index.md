## XmlDataLoadOptions.AlwaysGenerateRootObject property

Ruft ein Flag ab oder legt es fest, das angibt, ob eine generierte Datenquelle immer ein Objekt für ein XML-Stammelement enthält. Wenn ein XML-Stammelement keine Attribute hat und alle seine untergeordneten Elemente denselben Namen haben, wird ein solches Objekt standardmäßig nicht erstellt.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

## Bemerkungen

Der Standardwert ist`FALSCH` .
