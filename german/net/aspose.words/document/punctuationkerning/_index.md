## Document.PunctuationKerning property

Gibt an, ob das Kerning sowohl auf lateinischen Text als auch auf die Zeichensetzung angewendet wird.

```csharp
public bool PunctuationKerning { get; set; }
```

## Beispiele

Zeigt, wie mit Unterschneidung gearbeitet wird, die sowohl für lateinischen Text als auch für Zeichensetzung gilt.

```csharp
Document doc = new Document(MyDir + "Document.docx");
Assert.True(doc.PunctuationKerning);
```
