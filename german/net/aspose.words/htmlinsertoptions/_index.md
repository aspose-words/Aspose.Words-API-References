## HtmlInsertOptions enumeration

Gibt Optionen für die an[`InsertHtml`](../documentbuilder/inserthtml/) Methode.

```csharp
[Flags]
public enum HtmlInsertOptions
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Verwenden Sie beim Einfügen von HTML die Standardoptionen. |
| UseBuilderFormatting | `1` | Schriftart und Absatzformatierung verwenden, die in angegeben sind[`DocumentBuilder`](../documentbuilder/) als Basisformatierung für text eingefügt aus HTML. |
| RemoveLastEmptyParagraph | `2` | Entfernen Sie den leeren Absatz, der normalerweise nach HTML eingefügt wird, das mit einem Element auf Blockebene endet. |
| PreserveBlocks | `4` | Eigenschaften von Elementen auf Blockebene beibehalten. |
