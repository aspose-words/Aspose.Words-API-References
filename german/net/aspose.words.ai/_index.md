Der**Aspose.Words.AI** Der Namespace ermöglicht eine nahtlose Integration mit großen Sprachmodellen (LLMs) wie OpenAI und Gemini und bietet robuste Unterstützung für die KI-gestützte Dokumentenverarbeitung.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [AiModel](./aimodel/) | Stellt Informationen zu einem generativen Sprachmodell dar. |
| [AnthropicAiModel](./anthropicaimodel/) | Eine abstrakte Klasse, die die Integration mit den KI-Modellen von Anthropic innerhalb von Aspose.Words darstellt. |
| [CheckGrammarOptions](./checkgrammaroptions/) | Ermöglicht die Angabe verschiedener Optionen bei der Grammatikprüfung eines Dokuments mithilfe von KI. |
| [GoogleAiModel](./googleaimodel/) | Eine abstrakte Klasse, die die Integration mit den KI-Modellen von Google innerhalb von Aspose.Words darstellt. |
| [OpenAiModel](./openaimodel/) | Eine abstrakte Klasse, die die Integration mit den großen Sprachmodellen von OpenAI innerhalb von Aspose.Words darstellt. |
| [SummarizeOptions](./summarizeoptions/) | Ermöglicht die Angabe verschiedener Optionen zum Zusammenfassen von Dokumentinhalten. |
## Schnittstellen

| Schnittstelle | Beschreibung |
| --- | --- |
| [IAiModelText](./iaimodeltext/) | Die gemeinsame Schnittstelle für KI-Modelle zur Generierung einer Vielzahl textbasierter Inhalte. |
