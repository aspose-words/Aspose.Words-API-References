## OfficeMath class

Stellt ein Office Math-Objekt wie eine Funktion, Gleichung, Matrix oder ähnliches dar. Kann untergeordnete Elemente enthalten, einschließlich mathematischer Textzeilen, Lesezeichen, Kommentare usw`OfficeMath`Instanzen und einige andere Knoten.

Um mehr zu erfahren, besuchen Sie die[Arbeiten mit OfficeMath](https://docs.aspose.com/words/net/working-with-officemath/) Dokumentationsartikel.

```csharp
public class OfficeMath : CompositeNode
```

## Bemerkungen

In dieser Version von Aspose.Words,`OfficeMath` Knoten stellen keine öffentlichen Methoden und Eigenschaften zum Erstellen oder Ändern von Knoten bereit`OfficeMath` Objekt. In dieser Version können Sie nicht instanziieren.Math Knoten ändern oder bestehende ändern, außer sie zu löschen.

`OfficeMath` kann nur ein Kind von sein[`Paragraph`](../../aspose.words/paragraph/).

## Beispiele

Zeigt, wie die Anzeigeformatierung für Office-Mathematik festgelegt wird.

```csharp
Document doc = new Document(MyDir + "Office math.docx");

OfficeMath officeMath = (OfficeMath) doc.GetChild(NodeType.OfficeMath, 0, true);

// OfficeMath-Knoten, die anderen OfficeMath-Knoten untergeordnet sind, sind immer inline.
// Der Knoten, mit dem wir arbeiten, ist der Basisknoten, um seinen Standort und Anzeigetyp zu ändern.
Assert.AreEqual(MathObjectType.OMathPara, officeMath.MathObjectType);
Assert.AreEqual(NodeType.OfficeMath, officeMath.NodeType);
Assert.AreEqual(officeMath.ParentNode, officeMath.ParentParagraph);

// Ändern Sie den Speicherort und den Anzeigetyp des OfficeMath-Knotens.
officeMath.DisplayType = OfficeMathDisplayType.Display;
officeMath.Justification = OfficeMathJustification.Left;

doc.Save(ArtifactsDir + "Shape.OfficeMath.docx");
```
