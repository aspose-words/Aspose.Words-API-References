Der**Aspose.Wörter.Listen** Der Namespace enthält Klassen zum Arbeiten mit in einem Dokument definierten Aufzählungs- und Nummerierungslisten.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [List](./list/) | Stellt die Formatierung einer Liste dar. |
| [ListCollection](./listcollection/) | Speichert und verwaltet die Formatierung der in einem Dokument verwendeten Aufzählungs- und Nummerierungslisten. |
| [ListFormat](./listformat/) | Ermöglicht die Steuerung, welche Listenformatierung auf einen Absatz angewendet wird. |
| [ListLabel](./listlabel/) | Definiert Eigenschaften, die für eine Listenbezeichnung spezifisch sind. |
| [ListLevel](./listlevel/) | Definiert die Formatierung für eine Listenebene. |
| [ListLevelCollection](./listlevelcollection/) | Eine Sammlung von Listenformatierungen für jede Ebene in einer Liste. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [ListLevelAlignment](./listlevelalignment/) | Gibt die Ausrichtung für die Listennummer oder das Aufzählungszeichen an. |
| [ListTemplate](./listtemplate/) | Gibt eines der in Microsoft Word verfügbaren vordefinierten Listenformate an. |
| [ListTrailingCharacter](./listtrailingcharacter/) | Gibt das Zeichen an, das die Listenbezeichnung vom Text des Absatzes trennt. |
