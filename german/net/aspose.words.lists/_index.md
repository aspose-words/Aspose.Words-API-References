Das **Stellen.Wörter.Listen**Namespace enthält Klassen für working mit Aufzählungszeichen und nummerierten Listen, die in einem Dokument definiert sind.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [List](./list/) | Stellt die Formatierung einer Liste dar. |
| [ListCollection](./listcollection/) | Speichert und verwaltet die Formatierung von Aufzählungs- und Nummerierungslisten, die in einem Dokument verwendet werden. |
| [ListFormat](./listformat/) | Ermöglicht die Steuerung, welche Listenformatierung auf einen Absatz angewendet wird. |
| [ListLabel](./listlabel/) | Definiert Eigenschaften, die für eine Listenbezeichnung spezifisch sind. |
| [ListLevel](./listlevel/) | Definiert die Formatierung für eine Listenebene. |
| [ListLevelCollection](./listlevelcollection/) | Eine Sammlung von Listenformatierungen für jede Ebene in einer Liste. |
## Aufzählung

| [ListTrailingCharacter](./listtrailingcharacter/) | Gibt das Zeichen an, das die Listenbezeichnung vom Absatztext trennt. |
