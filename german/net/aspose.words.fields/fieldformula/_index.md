## Bemerkungen

Berechnet das Ergebnis eines Ausdrucks.

## Beispiele

Zeigt, wie das Formelfeld verwendet wird, um das Ergebnis einer Gleichung anzuzeigen.

```csharp
Document doc = new Document();

// Verwenden Sie einen Feldgenerator, um eine mathematische Gleichung zu konstruieren,
// Erstellen Sie dann ein Formelfeld, um das Ergebnis der Gleichung im Dokument anzuzeigen.
FieldBuilder fieldBuilder = new FieldBuilder(FieldType.FieldFormula);
fieldBuilder.AddArgument(2);
fieldBuilder.AddArgument("*");
fieldBuilder.AddArgument(5);

FieldFormula field = (FieldFormula)fieldBuilder.BuildAndInsert(doc.FirstSection.Body.FirstParagraph);
field.Update();

Assert.AreEqual(" = 2 * 5 ", field.GetFieldCode());
Assert.AreEqual("10", field.Result);

doc.UpdateFields();
doc.Save(ArtifactsDir + "Field.FORMULA.docx");
```
