### Beispiele

Zeigt, wie mit dem Feld „FieldNone“ in einem Dokument gearbeitet wird.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// Fügen Sie ein Feld in seinen Feldcode ein, das keinen Zielfeldtyp angibt.
Field field = builder.InsertField(" NOTAREALFIELD //A");

// Für solche Felder ist der Feldtyp „FieldNone“ reserviert.
Assert.AreEqual(FieldType.FieldNone, field.Type);

// Wir können auch weiterhin mit diesen Feldern arbeiten und sie als Instanzen der FieldUnknown-Klasse zuweisen.
FieldUnknown fieldUnknown = (FieldUnknown)field;
Assert.AreEqual(" NOTAREALFIELD //a", fieldUnknown.GetFieldCode());
```
