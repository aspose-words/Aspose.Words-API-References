Der**Aspose.Wörter.Zeichnung.Ole** Der Namespace enthält Klassen zum Arbeiten mit OLE-/ActiveX-Steuerelementen und Formularen in Microsoft Word-Dokumenten.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [CheckBoxControl](./checkboxcontrol/) | Das CheckBox-Steuerelement schaltet einen Wert um. |
| [CommandButtonControl](./commandbuttoncontrol/) | Das CommandButton-Steuerelement führt ein Makro aus, das eine Aktion ausführt, wenn ein Benutzer darauf klickt. |
| [Forms2OleControl](./forms2olecontrol/) | Stellt ein Microsoft Forms 2.0 OLE-Steuerelement dar. |
| [Forms2OleControlCollection](./forms2olecontrolcollection/) | Stellt eine Sammlung von[`Forms2OleControl`](../aspose.words.drawing.ole/forms2olecontrol/) Objekte. |
| [MorphDataControl](./morphdatacontrol/) | Die MorphDataControl-Struktur ist eine Ansammlung von sechs Steuerelementen: CheckBox, ComboBox, ListBox, OptionButton, TextBox und ToggleButton. |
| [OleControl](./olecontrol/) | Stellt ein OLE ActiveX-Steuerelement dar. |
| [OptionButtonControl](./optionbuttoncontrol/) | Das OptionButton-Steuerelement ermöglicht eine einzelne Auswahl aus einer begrenzten Menge sich gegenseitig ausschließender Auswahlmöglichkeiten. |
| [TextBoxControl](./textboxcontrol/) | Das TextBox-Steuerelement zeigt Text aus einem organisierten Datensatz oder einer Benutzereingabe an. |
