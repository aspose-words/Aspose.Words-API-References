Der**Aspose.Words.Bibliography**Der Namespace stellt Klassen bereit, die den Zugriff auf eine Liste der in einem Dokument verfügbaren Bibliografiequellen ermöglichen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Bibliography](./bibliography/) | Stellt die Liste der im Dokument verfügbaren Bibliografiequellen dar. |
| [Contributor](./contributor/) | Stellt einen Beitragenden zur Bibliografiequelle dar. Kann entweder ein Unternehmen (eine Organisation) oder eine Liste von Personen sein. |
| [ContributorCollection](./contributorcollection/) | Stellt die Mitwirkenden der Bibliografiequelle dar. |
| [Corporate](./corporate/) | Stellt einen Beitragenden zu einer Unternehmensbibliographiequelle (einer Organisation) dar. |
| [Person](./person/) | Stellt einen einzelnen (eine Person) Beitragenden zur Bibliografiequelle dar. |
| [PersonCollection](./personcollection/) | Stellt eine Liste von Personen dar, die zu den Quellen der Bibliografie beigetragen haben. |
| [Source](./source/) | Stellt eine einzelne Quelle dar, beispielsweise ein Buch, einen Zeitschriftenartikel oder ein Interview. |
