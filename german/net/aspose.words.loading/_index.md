Das **Aspose.Words.Loading** namespace stellt Klassen und Aufzählungen bereit, die erlauben, zusätzliche Optionen beim Laden von Dokumenten anzugeben.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [ChmLoadOptions](./chmloadoptions/) | Ermöglicht die Angabe zusätzlicher Optionen beim Laden eines CHM-Dokuments in eine[`Document`](../aspose.words/document/) Objekt. |
| [DocumentLoadingArgs](./documentloadingargs/) | Ein übergebenes Argument[`Notify`](../aspose.words.loading/idocumentloadingcallback/notify/) . |
| [HtmlLoadOptions](./htmlloadoptions/) | Ermöglicht die Angabe zusätzlicher Optionen beim Laden eines HTML-Dokuments in a[`Document`](../aspose.words/document/) Objekt. |
| [LanguagePreferences](./languagepreferences/) | Ermöglicht die Einrichtung von Spracheinstellungen. |
| [LoadOptions](./loadoptions/) | Ermöglicht die Angabe zusätzlicher Optionen (z. B. Kennwort oder Basis-URI) beim Laden eines Dokuments in einen[`Document`](../aspose.words/document/) Objekt. |
| [PdfLoadOptions](./pdfloadoptions/) | Ermöglicht das Festlegen zusätzlicher Optionen beim Laden eines PDF-Dokuments in eine[`Document`](../aspose.words/document/) Objekt. |
| [ResourceLoadingArgs](./resourceloadingargs/) | Liefert Daten für die[`ResourceLoading`](../aspose.words.loading/iresourceloadingcallback/resourceloading/) Methode. |
| [RtfLoadOptions](./rtfloadoptions/) | Ermöglicht die Angabe zusätzlicher Optionen beim LadenRtf Dokument in ein[`Document`](../aspose.words/document/) Objekt. |
| [TxtLoadOptions](./txtloadoptions/) | Ermöglicht die Angabe zusätzlicher Optionen beim LadenText Dokument in ein[`Document`](../aspose.words/document/) Objekt. |
## Schnittstellen

| Schnittstelle | Beschreibung |
| --- | --- |
| [IDocumentLoadingCallback](./idocumentloadingcallback/) | Implementieren Sie diese Schnittstelle, wenn Sie möchten, dass Ihre eigene benutzerdefinierte Methode beim Laden eines Dokuments aufgerufen wird. |
| [IResourceLoadingCallback](./iresourceloadingcallback/) | Implementieren Sie diese Schnittstelle, wenn Sie steuern möchten, wie Aspose.Words externe Ressourcen lädt, wenn ein Dokument importiert und Bilder eingefügt werden[`DocumentBuilder`](../aspose.words/documentbuilder/) . |
## Aufzählung

| Aufzählung | Beschreibung |
