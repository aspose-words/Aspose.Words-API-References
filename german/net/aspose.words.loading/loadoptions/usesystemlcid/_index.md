## LoadOptions.UseSystemLcid property

Ruft ab oder legt fest, ob der aus der Windows-Registrierung abgerufene LCID-Wert zum Bestimmen der Standardränder für die Seiteneinrichtung verwendet werden soll.

```csharp
public bool UseSystemLcid { get; set; }
```

## Bemerkungen

Wenn eingestellt auf`WAHR`, dann wird das Verhalten von MS Word emuliert, das den LCID-Wert aus der Windows-Registrierung übernimmt.

Der Standardwert ist`FALSCH`.
