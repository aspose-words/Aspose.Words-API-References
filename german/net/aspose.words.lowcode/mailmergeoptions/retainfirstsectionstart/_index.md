## MailMergeOptions.RetainFirstSectionStart property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob der Abschnittsanfang des ersten Dokumentabschnitts und seine Kopien für nachfolgende Datenquellenzeilen beim Seriendruck beibehalten oder entsprechend dem Verhalten von MS Word aktualisiert werden.

```csharp
public bool RetainFirstSectionStart { get; set; }
```

## Bemerkungen

Der Standardwert ist`WAHR` .
