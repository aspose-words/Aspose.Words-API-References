## MergeFormatMode enumeration

Gibt an, wie die Formatierung beim Zusammenführen mehrerer Dokumente zusammengeführt wird.

```csharp
public enum MergeFormatMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| MergeFormatting | `0` | Kombinieren Sie die Formatierung der zusammengeführten Dokumente. |
| KeepSourceFormatting | `1` | Bedeutet, dass das Quelldokument seine ursprüngliche Formatierung behält, wie Schriftarten, Größen, Farben, Einzüge und alle anderen auf seinen Inhalt angewendeten Formatierungselemente. |
| KeepSourceLayout | `2` | Behalten Sie das Layout der Originaldokumente im endgültigen Dokument bei. |
