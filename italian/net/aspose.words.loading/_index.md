IL**Aspose.Words.Loading**namespace fornisce classi ed enumerazioni che consentono di specificare opzioni aggiuntive durante il caricamento dei documenti.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ChmLoadOptions](./chmloadoptions/) | Permette di specificare opzioni aggiuntive durante il caricamento del documento CHM in a[`Document`](../aspose.words/document/) oggetto. |
| [DocumentLoadingArgs](./documentloadingargs/) | Un argomento passato[`Notify`](../aspose.words.loading/idocumentloadingcallback/notify/) . |
| [HtmlLoadOptions](./htmlloadoptions/) | Permette di specificare opzioni aggiuntive durante il caricamento del documento HTML in un file[`Document`](../aspose.words/document/) oggetto. |
| [LanguagePreferences](./languagepreferences/) | Permette di impostare le preferenze della lingua. |
| [LoadOptions](./loadoptions/) | Permette di specificare opzioni aggiuntive (come password o URI di base) quando si carica un documento in un[`Document`](../aspose.words/document/) oggetto. |
| [PdfLoadOptions](./pdfloadoptions/) | Permette di specificare opzioni aggiuntive durante il caricamento del documento Pdf in un file[`Document`](../aspose.words/document/) oggetto. |
| [ResourceLoadingArgs](./resourceloadingargs/) | Fornisce i dati per il[`ResourceLoading`](../aspose.words.loading/iresourceloadingcallback/resourceloading/) metodo. |
| [RtfLoadOptions](./rtfloadoptions/) | Permette di specificare opzioni aggiuntive durante il caricamentoRtf documento in a[`Document`](../aspose.words/document/) oggetto. |
| [TxtLoadOptions](./txtloadoptions/) | Permette di specificare opzioni aggiuntive durante il caricamentoText documento in a[`Document`](../aspose.words/document/) oggetto. |
## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IDocumentLoadingCallback](./idocumentloadingcallback/) | Implementa questa interfaccia se desideri che il tuo metodo personalizzato venga chiamato durante il caricamento di un documento. |
| [IResourceLoadingCallback](./iresourceloadingcallback/) | Implementa questa interfaccia se desideri controllare il modo in cui Aspose.Words carica le risorse esterne quando importi un documento e inserisci immagini utilizzando[`DocumentBuilder`](../aspose.words/documentbuilder/) . |
