IL**Aspose.Words.Loading** namespace fornisce classi ed enumerazioni che consentono di specificare opzioni aggiuntive durante il caricamento dei documenti.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ChmLoadOptions](./chmloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamento di un documento CHM in un[`Document`](../aspose.words/document/) oggetto. |
| [DocumentLoadingArgs](./documentloadingargs/) | Un argomento passato in[`Notify`](../aspose.words.loading/idocumentloadingcallback/notify/) . |
| [HtmlLoadOptions](./htmlloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamento di un documento HTML in un[`Document`](../aspose.words/document/) oggetto. |
| [LanguagePreferences](./languagepreferences/) | Consente di impostare le preferenze della lingua. |
| [LoadOptions](./loadoptions/) | Consente di specificare opzioni aggiuntive (come password o URI di base) quando si carica un documento in un[`Document`](../aspose.words/document/) oggetto. |
| [MarkdownLoadOptions](./markdownloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamentoMarkdown documento in un[`Document`](../aspose.words/document/) oggetto. |
| [PdfLoadOptions](./pdfloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamento di un documento Pdf in un[`Document`](../aspose.words/document/) oggetto. |
| [ResourceLoadingArgs](./resourceloadingargs/) | Fornisce dati per il[`ResourceLoading`](../aspose.words.loading/iresourceloadingcallback/resourceloading/) metodo. |
| [RtfLoadOptions](./rtfloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamentoRtf documento in un[`Document`](../aspose.words/document/) oggetto. |
| [TxtLoadOptions](./txtloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamentoText documento in un[`Document`](../aspose.words/document/) oggetto. |
## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IDocumentLoadingCallback](./idocumentloadingcallback/) | Implementa questa interfaccia se vuoi che venga chiamato il tuo metodo personalizzato durante il caricamento di un documento. |
| [IResourceLoadingCallback](./iresourceloadingcallback/) | Implementa questa interfaccia se vuoi controllare come Aspose.Words carica risorse esterne quando importi un documento e inserisci immagini utilizzando[`DocumentBuilder`](../aspose.words/documentbuilder/) . |
