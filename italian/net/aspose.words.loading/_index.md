Il **Aspose.Words.Loading** namespace fornisce classi ed enumerazioni che consentono di specificare opzioni aggiuntive durante il caricamento dei documenti.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ChmLoadOptions](./chmloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamento del documento CHM in a[`Document`](../aspose.words/document/) oggetto. |
| [DocumentLoadingArgs](./documentloadingargs/) | Un argomento passato[`Notify`](../aspose.words.loading/idocumentloadingcallback/notify/) . |
| [HtmlLoadOptions](./htmlloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamento di un documento HTML in un file[`Document`](../aspose.words/document/) oggetto. |
| [LanguagePreferences](./languagepreferences/) | Consente di impostare le preferenze della lingua. |
| [LoadOptions](./loadoptions/) | Consente di specificare opzioni aggiuntive (come password o URI di base) quando carica un documento in un[`Document`](../aspose.words/document/) oggetto. |
| [PdfLoadOptions](./pdfloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamento di un documento Pdf in un file[`Document`](../aspose.words/document/) oggetto. |
| [ResourceLoadingArgs](./resourceloadingargs/) | Fornisce i dati per il[`ResourceLoading`](../aspose.words.loading/iresourceloadingcallback/resourceloading/) metodo. |
| [RtfLoadOptions](./rtfloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamentoRtf documento in a[`Document`](../aspose.words/document/) oggetto. |
| [TxtLoadOptions](./txtloadoptions/) | Consente di specificare opzioni aggiuntive durante il caricamentoText documento in a[`Document`](../aspose.words/document/) oggetto. |
## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IDocumentLoadingCallback](./idocumentloadingcallback/) | Implementa questa interfaccia se vuoi che il tuo metodo personalizzato venga chiamato durante il caricamento di un documento. |
| [IResourceLoadingCallback](./iresourceloadingcallback/) | Implementa questa interfaccia se vuoi controllare come Aspose.Words carica una risorsa esterna quando importa un documento e inserisci immagini usando[`DocumentBuilder`](../aspose.words/documentbuilder/) . |
