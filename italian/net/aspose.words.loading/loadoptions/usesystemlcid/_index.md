## LoadOptions.UseSystemLcid property

Ottiene o imposta se utilizzare il valore LCID ottenuto dal registro di Windows per determinare i margini predefiniti di impostazione della pagina.

```csharp
public bool UseSystemLcid { get; set; }
```

## Osservazioni

Se impostato su`VERO`, viene emulato il comportamento di MS Word che prende il valore LCID dal registro di Windows.

Il valore predefinito è`falso`.
