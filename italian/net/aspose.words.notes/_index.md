IL**Aspose.Words.Notes** namespace fornisce classi per lavorare con le note di chiusura e le note a piè di pagina di Microsoft Word.

## Classi

| Classe | Descrizione |
| --- | --- |
| [EndnoteOptions](./endnoteoptions/) | Rappresenta le opzioni di numerazione delle note di chiusura per un documento o una sezione. |
| [Footnote](./footnote/) | Rappresenta un contenitore per il testo di una nota a piè di pagina o di una nota di chiusura. |
| [FootnoteOptions](./footnoteoptions/) | Rappresenta le opzioni di numerazione delle note a piè di pagina per un documento o una sezione. |
| [FootnoteSeparator](./footnoteseparator/) | Rappresenta un contenitore per il separatore delle note a piè di pagina/note di chiusura e per il contenuto di continuazione di un documento. |
| [FootnoteSeparatorCollection](./footnoteseparatorcollection/) | Fornisce accesso tipizzato a[`FootnoteSeparator`](../aspose.words.notes/footnoteseparator/) nodi di un documento. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [EndnotePosition](./endnoteposition/) | Definisce la posizione della nota di chiusura. |
| [FootnoteNumberingRule](./footnotenumberingrule/) | Determina quando riavviare la numerazione automatica delle note a piè di pagina o delle note di chiusura. |
| [FootnotePosition](./footnoteposition/) | Definisce la posizione della nota a piè di pagina. |
| [FootnoteSeparatorType](./footnoteseparatortype/) | Specifica il tipo di separatore delle note a piè di pagina/note di chiusura. |
| [FootnoteType](./footnotetype/) | Specifica se si tratta di una nota a piè di pagina o di una nota di chiusura. |
