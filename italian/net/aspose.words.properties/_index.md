Il **Aspose.Parole.Proprietà** namespace fornisce classi per lavorare con proprietà del documento personalizzate e integrate come titolo, parole chiave, azienda ecc.

## Classi

| Classe | Descrizione |
| --- | --- |
| [BuiltInDocumentProperties](./builtindocumentproperties/) | Una raccolta di proprietà del documento integrate. |
| [CustomDocumentProperties](./customdocumentproperties/) | Una raccolta di proprietà del documento personalizzate. |
| [DocumentProperty](./documentproperty/) | Rappresenta una proprietà del documento personalizzata o incorporata. |
| [DocumentPropertyCollection](./documentpropertycollection/) | Classe base per[`BuiltInDocumentProperties`](../aspose.words.properties/builtindocumentproperties/) e[`CustomDocumentProperties`](../aspose.words.properties/customdocumentproperties/) raccolte. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [DocumentSecurity](./documentsecurity/) | Usato come valore per il[`Security`](../aspose.words.properties/builtindocumentproperties/security/) proprietà. Specifica il livello di sicurezza di un documento come valore numerico. |
| [PropertyType](./propertytype/) | Specifica il tipo di dati di una proprietà del documento. |
