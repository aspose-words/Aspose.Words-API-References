## StructuredDocumentTagRangeStart.Placeholder property

Ottiene il[`BuildingBlock`](../../../aspose.words.buildingblocks/buildingblock/) contenente un testo segnaposto che dovrebbe essere visualizzato quando questo tag di documento strutturato esegue il contenuto è vuoto, l'elemento XML mappato associato è vuoto come specificato tramite[`XmlMapping`](../xmlmapping/) elemento o il[`IsShowingPlaceholderText`](../isshowingplaceholdertext/) elemento è vero.

```csharp
public BuildingBlock Placeholder { get; }
```

### Osservazioni

Può essere null, il che significa che il segnaposto non è applicabile per questo tag di documento strutturato.
