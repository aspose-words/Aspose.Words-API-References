IL**Aspose.Words.Rendering** namespace fornisce classi che consentono di personalizzare la stampa o il rendering dei documenti.

## Classi

| Classe | Descrizione |
| --- | --- |
| [AsposeWordsPrintDocument](./asposewordsprintdocument/) | Fornisce un'implementazione predefinita per la stampa di un[`Document`](../aspose.words/document/)all'interno del framework di stampa .NET. |
| [NodeRendererBase](./noderendererbase/) | Classe base per[`ShapeRenderer`](../aspose.words.rendering/shaperenderer/) E[`OfficeMathRenderer`](../aspose.words.rendering/officemathrenderer/) . |
| [OfficeMathRenderer](./officemathrenderer/) | Fornisce metodi per il rendering di un individuo[`OfficeMath`](../aspose.words.math/officemath/) a un'immagine raster o vettoriale o a un oggetto grafico. |
| [PageInfo](./pageinfo/) | Rappresenta informazioni su una particolare pagina del documento. |
| [PrinterSettingsContainer](./printersettingscontainer/) | Rappresenta un archivio per alcuni parametri diPrinterSettings oggetto. |
| [ShapeRenderer](./shaperenderer/) | Fornisce metodi per il rendering di un individuo[`Shape`](../aspose.words.drawing/shape/) O[`GroupShape`](../aspose.words.drawing/groupshape/) a un'immagine raster o vettoriale o a un oggetto grafico. |
| [ThumbnailGeneratingOptions](./thumbnailgeneratingoptions/) | Può essere utilizzato per specificare opzioni aggiuntive durante la generazione di miniature per un documento. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [ColorPrintMode](./colorprintmode/) | Specifica come vengono stampate le pagine non colorate se il dispositivo supporta la stampa a colori. |
