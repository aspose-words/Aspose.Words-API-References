IL**Aspose.Words.Drawing.Ole** namespace contiene classi per lavorare con controlli OLE/ActiveX e moduli nei documenti Microsoft Word.

## Classi

| Classe | Descrizione |
| --- | --- |
| [CheckBoxControl](./checkboxcontrol/) | Il controllo CheckBox commuta un valore. |
| [CommandButtonControl](./commandbuttoncontrol/) | Il controllo CommandButton esegue una macro che esegue un'azione quando un utente fa clic su di esso. |
| [Forms2OleControl](./forms2olecontrol/) | Rappresenta il controllo OLE di Microsoft Forms 2.0. |
| [Forms2OleControlCollection](./forms2olecontrolcollection/) | Rappresenta la raccolta di[`Forms2OleControl`](../aspose.words.drawing.ole/forms2olecontrol/) oggetti. |
| [MorphDataControl](./morphdatacontrol/) | La struttura MorphDataControl è un aggregato di sei controlli: CheckBox, ComboBox, ListBox, OptionButton, TextBox e ToggleButton. |
| [OleControl](./olecontrol/) | Rappresenta il controllo ActiveX OLE. |
| [OptionButtonControl](./optionbuttoncontrol/) | Il controllo OptionButton consente una singola scelta in un set limitato di scelte reciprocamente esclusive. |
| [TextBoxControl](./textboxcontrol/) | Il controllo TextBox visualizza il testo da un set organizzato di dati o input utente. |
