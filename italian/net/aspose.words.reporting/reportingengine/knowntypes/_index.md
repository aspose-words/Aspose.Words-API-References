## ReportingEngine.KnownTypes property

Ottiene un set non ordinato (ovvero una raccolta di elementi univoci) contenenteTypeoggetti i cui nomi completi o parzialmente qualificati possono essere utilizzati all'interno dei modelli di report elaborati da questa istanza del motore per richiamare i membri statici dei tipi corrispondenti, eseguire cast di tipi, ecc.

public KnownTypeSet KnownTypes { get; }
