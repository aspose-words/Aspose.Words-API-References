Questo spazio dei nomi contiene classi del nuovo motore di report di Aspose.Words che supporta i modelli di report contrassegnati con un linguaggio basato sulla sintassi del metodo LINQ.

## Classi

| Classe | Descrizione |
| --- | --- |
| [CsvDataLoadOptions](./csvdataloadoptions/) | Rappresenta le opzioni per l'analisi dei dati CSV. |
| [CsvDataSource](./csvdatasource/) | Fornisce l'accesso ai dati di un file CSV o di un flusso da utilizzare all'interno di un report. |
| [JsonDataLoadOptions](./jsondataloadoptions/) | Rappresenta le opzioni per l'analisi dei dati JSON. |
| [JsonDataSource](./jsondatasource/) | Fornisce l'accesso ai dati di un file o flusso JSON da utilizzare all'interno di un report. |
| [KnownTypeSet](./knowntypeset/) | Rappresenta un insieme non ordinato (cioè una raccolta di elementi univoci) contenenteType oggetti i cui nomi completi o parzialmente qualificati possono essere utilizzati all'interno dei modelli di report per richiamare i membri statici dei tipi corrispondenti, eseguire cast di tipi e così via. |
| [ReportingEngine](./reportingengine/) | Fornisce le routine per popolare i documenti modello con i dati e una serie di impostazioni per controllare queste routine. |
| [XmlDataLoadOptions](./xmldataloadoptions/) | Rappresenta le opzioni per il caricamento dei dati XML. |
| [XmlDataSource](./xmldatasource/) | Fornisce l'accesso ai dati di un file o flusso XML da utilizzare all'interno di un report. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [JsonSimpleValueParseMode](./jsonsimplevalueparsemode/) | Specifica una modalità per l'analisi dei valori JSON semplici (null, booleano, numero, intero e stringa) durante il caricamento di JSON. Tale modalità non influisce sull'analisi dei valori di data e ora. |
| [ReportBuildOptions](./reportbuildoptions/) | Specifica le opzioni che controllano il comportamento di[`ReportingEngine`](../aspose.words.reporting/reportingengine/) durante la creazione di un rapporto. |
