IL**Aspose.Words.Reporting** namespace contiene classi del nuovo motore di reporting di Aspose.Words che supporta modelli di report contrassegnati con un linguaggio basato sulla sintassi del metodo LINQ.

## Classi

| Classe | Descrizione |
| --- | --- |
| [CsvDataLoadOptions](./csvdataloadoptions/) | Rappresenta le opzioni per l'analisi dei dati CSV. |
| [CsvDataSource](./csvdatasource/) | Fornisce l'accesso ai dati di un file CSV o di un flusso da utilizzare in un report. |
| [JsonDataLoadOptions](./jsondataloadoptions/) | Rappresenta le opzioni per l'analisi dei dati JSON. |
| [JsonDataSource](./jsondatasource/) | Fornisce l'accesso ai dati di un file o flusso JSON da utilizzare all'interno di un report. |
| [KnownTypeSet](./knowntypeset/) | Rappresenta un insieme non ordinato (ovvero una raccolta di elementi unici) contenenteType oggetti i cui nomi completamente o parzialmente qualificati possono essere utilizzati all'interno dei modelli di report per richiamare i membri statici dei tipi corrispondenti, eseguire cast di tipo, ecc. |
| [ReportingEngine](./reportingengine/) | Fornisce routine per popolare i documenti modello con dati e un set di impostazioni per controllare queste routine. |
| [XmlDataLoadOptions](./xmldataloadoptions/) | Rappresenta le opzioni per il caricamento dei dati XML. |
| [XmlDataSource](./xmldatasource/) | Fornisce l'accesso ai dati di un file XML o di un flusso da utilizzare all'interno di un report. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [JsonSimpleValueParseMode](./jsonsimplevalueparsemode/) | Specifica una modalità per l'analisi di valori JSON semplici (null, booleani, numeri, interi e stringhe) durante il caricamento di JSON. Tale modalità non influisce sull'analisi dei valori di data e ora. |
| [ReportBuildOptions](./reportbuildoptions/) | Specifica le opzioni che controllano il comportamento di[`ReportingEngine`](../aspose.words.reporting/reportingengine/) durante la creazione di un report. |
