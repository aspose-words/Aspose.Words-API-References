## XmlDataLoadOptions.AlwaysGenerateRootObject property

Ottiene o imposta un flag che indica se un'origine dati generata conterrà sempre un oggetto per un elemento XML radice . Se un elemento XML radice non ha attributi e tutti i suoi elementi figlio hanno gli stessi nomi, tale oggetto non viene creato per impostazione predefinita.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

## Osservazioni

Il valore predefinito è`falso` .
