## JsonDataLoadOptions.AlwaysGenerateRootObject property

Ottiene o imposta un flag che indica se un'origine dati generata conterrà sempre un oggetto per un elemento JSON root . Se un elemento radice JSON contiene una singola proprietà complessa, tale oggetto non viene creato per impostazione predefinita.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

## Osservazioni

Il valore predefinito è`falso` .
