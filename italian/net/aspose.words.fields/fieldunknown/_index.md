## Esempi

Mostra come lavorare con il campo "FieldNone" in un documento.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// Inserisci un campo che non denota un tipo di campo obiettivo nel relativo codice di campo.
Field field = builder.InsertField(" NOTAREALFIELD //UN");

// Il tipo di campo "FieldNone" è riservato a campi come questi.
Assert.AreEqual(FieldType.FieldNone, field.Type);

// Possiamo anche continuare a lavorare con questi campi e assegnarli come istanze della classe FieldUnknown.
FieldUnknown fieldUnknown = (FieldUnknown)field;
Assert.AreEqual(" NOTAREALFIELD //a", fieldUnknown.GetFieldCode());
```
