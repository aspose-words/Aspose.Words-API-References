## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IBarcodeGenerator](./ibarcodegenerator/) | Interfaccia pubblica per generatore personalizzato di codici a barre. L'implementazione deve essere fornita dall'utente. |
| [IComparisonExpressionEvaluator](./icomparisonexpressionevaluator/) | Quando implementato, consente di ignorare la valutazione delle espressioni di confronto predefinite per il[`FieldIf`](../aspose.words.fields/fieldif/) e[`FieldCompare`](../aspose.words.fields/fieldcompare/) campi. |
| [IFieldDatabaseProvider](./ifielddatabaseprovider/) | Implementa questa interfaccia per fornire i dati per il[`FieldDatabase`](../aspose.words.fields/fielddatabase/) campo quando viene aggiornato. |
| [IFieldResultFormatter](./ifieldresultformatter/) | Implementa questa interfaccia se vuoi controllare come è formattato il risultato del campo. |
| [IFieldUpdateCultureProvider](./ifieldupdatecultureprovider/) | Quando implementato, fornisce aCultureInfo oggetto da utilizzare durante l'aggiornamento di un determinato campo. |
| [IFieldUpdatingCallback](./ifieldupdatingcallback/) | Implementa questa interfaccia se desideri che i tuoi metodi personalizzati vengano chiamati durante un aggiornamento del campo. |
| [IFieldUserPromptRespondent](./ifielduserpromptrespondent/) | Rappresenta il rispondente alle richieste dell'utente durante l'aggiornamento del campo. |
