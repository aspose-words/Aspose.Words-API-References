## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IBarcodeGenerator](./ibarcodegenerator/) | Interfaccia pubblica per generatore personalizzato di codici a barre. L'implementazione dovrebbe essere fornita dall'utente. |
| [IBibliographyStylesProvider](./ibibliographystylesprovider/) | Implementa questa interfaccia per fornire uno stile bibliografico per il[`FieldBibliography`](../aspose.words.fields/fieldbibliography/) E[`FieldCitation`](../aspose.words.fields/fieldcitation/) campi quando vengono aggiornati. |
| [IComparisonExpressionEvaluator](./icomparisonexpressionevaluator/) | Quando implementato, consente di sovrascrivere la valutazione delle espressioni di confronto predefinite per il[`FieldIf`](../aspose.words.fields/fieldif/) E[`FieldCompare`](../aspose.words.fields/fieldcompare/) campi. |
| [IFieldDatabaseProvider](./ifielddatabaseprovider/) | Implementa questa interfaccia per fornire dati per[`FieldDatabase`](../aspose.words.fields/fielddatabase/) campo quando viene aggiornato. |
| [IFieldResultFormatter](./ifieldresultformatter/) | Implementa questa interfaccia se desideri controllare come viene formattato il risultato del campo. |
| [IFieldUpdateCultureProvider](./ifieldupdatecultureprovider/) | Quando implementato, fornisce aCultureInfo oggetto che dovrebbe essere utilizzato durante l'aggiornamento di un particolare campo. |
| [IFieldUpdatingCallback](./ifieldupdatingcallback/) | Implementa questa interfaccia se desideri che i tuoi metodi personalizzati vengano richiamati durante un aggiornamento del campo. |
| [IFieldUpdatingProgressCallback](./ifieldupdatingprogresscallback/) | Implementa questa interfaccia se desideri monitorare l'avanzamento dell'aggiornamento sul campo. |
| [IFieldUserPromptRespondent](./ifielduserpromptrespondent/) | Rappresenta l'intervistato alle richieste dell'utente durante l'aggiornamento del campo. |
