IL**Aspose.Words.Bibliography**namespace fornisce classi che consentono di accedere a un elenco di fonti bibliografiche disponibili in un documento.

## Classi

| Classe | Descrizione |
| --- | --- |
| [Bibliography](./bibliography/) | Rappresenta l'elenco delle fonti bibliografiche disponibili nel documento. |
| [Contributor](./contributor/) | Rappresenta il contributore di una fonte bibliografica. Può essere un'azienda (un'organizzazione) o un elenco di persone. |
| [ContributorCollection](./contributorcollection/) | Rappresenta i contributori della fonte bibliografica. |
| [Corporate](./corporate/) | Rappresenta un collaboratore di una fonte bibliografica aziendale (un'organizzazione). |
| [Person](./person/) | Rappresenta il collaboratore della fonte bibliografica individuale (una persona). |
| [PersonCollection](./personcollection/) | Rappresenta un elenco di persone che hanno contribuito alla fonte bibliografica. |
| [Source](./source/) | Rappresenta una fonte individuale, come un libro, un articolo di giornale o un'intervista. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [SourceType](./sourcetype/) | Rappresenta i tipi di fonte bibliografica. |
