## CellFormat.HideMark property

Restituisce o imposta la visibilità del contrassegno di cella.

```csharp
public bool HideMark { get; set; }
```

## Osservazioni

Specifica che il contenuto delle celle della tabella viene visualizzato senza altezza se tutte le celle nella riga sono vuote; tuttavia, le celle hanno un'altezza visibile se hanno bordi, margini o spaziatura delle celle diversi da zero.
