IL**Aspose.Words.Tables** Lo spazio dei nomi contiene classi che rappresentano tabelle, righe, celle e la loro formattazione.

## Classi

| Classe | Descrizione |
| --- | --- |
| [Cell](./cell/) | Rappresenta una cella della tabella. |
| [CellCollection](./cellcollection/) | Fornisce accesso tipizzato a una raccolta di[`Cell`](../aspose.words.tables/cell/) nodi. |
| [CellFormat](./cellformat/) | Rappresenta tutta la formattazione per una cella della tabella. |
| [PreferredWidth](./preferredwidth/) | Rappresenta un valore e la sua unità di misura utilizzata per specificare la larghezza preferita di una tabella o di una cella. |
| [Row](./row/) | Rappresenta una riga della tabella. |
| [RowCollection](./rowcollection/) | Fornisce accesso tipizzato a una raccolta di[`Row`](../aspose.words.tables/row/) nodi. |
| [RowFormat](./rowformat/) | Rappresenta tutta la formattazione per una riga della tabella. |
| [Table](./table/) | Rappresenta una tabella in un documento Word. |
| [TableCollection](./tablecollection/) | Fornisce accesso tipizzato a una raccolta di[`Table`](../aspose.words.tables/table/) nodi. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [AutoFitBehavior](./autofitbehavior/) | Determina come Aspose.Words ridimensiona la tabella quando si richiama il[`AutoFit`](../aspose.words.tables/table/autofit/) metodo. |
| [CellMerge](./cellmerge/) | Specifica come una cella in una tabella viene unita ad altre celle. |
| [CellVerticalAlignment](./cellverticalalignment/) | Specifica la giustificazione verticale del testo all'interno di una cella di tabella. |
| [PreferredWidthType](./preferredwidthtype/) | Specifica l'unità di misura per la larghezza preferita di una tabella o di una cella. |
| [TableAlignment](./tablealignment/) | Specifica l'allineamento per una tabella in linea. |
| [TableStyleOptions](./tablestyleoptions/) | Specifica come viene applicato lo stile della tabella a una tabella. |
| [TextWrapping](./textwrapping/) | Specifica come il testo viene disposto attorno alla tabella. |
