IL **Aspose.Words.Tables** lo spazio dei nomi contiene classi che rappresentano tabelle, righe, celle e la loro formattazione.

## Classi

| Classe | Descrizione |
| --- | --- |
| [Cell](./cell/) | Rappresenta una cella di tabella. |
| [CellCollection](./cellcollection/) | Fornisce l'accesso digitato a una raccolta di[`Cell`](../aspose.words.tables/cell/) nodi. |
| [CellFormat](./cellformat/) | Rappresenta tutta la formattazione per una cella di tabella. |
| [PreferredWidth](./preferredwidth/) | Rappresenta un valore e la relativa unità di misura utilizzata per specificare la larghezza preferita di una tabella o di una cella. |
| [Row](./row/) | Rappresenta una riga della tabella. |
| [RowCollection](./rowcollection/) | Fornisce l'accesso digitato a una raccolta di[`Row`](../aspose.words.tables/row/) nodi. |
| [RowFormat](./rowformat/) | Rappresenta tutta la formattazione per una riga di tabella. |
| [Table](./table/) | Rappresenta una tabella in un documento Word. |
| [TableCollection](./tablecollection/) | Fornisce l'accesso digitato a una raccolta di[`Table`](../aspose.words.tables/table/) nodi. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [AutoFitBehavior](./autofitbehavior/) | Determina il modo in cui Aspose.Words ridimensiona la tabella quando si richiama il[`AutoFit`](../aspose.words.tables/table/autofit/) metodo. |
| [CellMerge](./cellmerge/) | Specifica il modo in cui una cella in una tabella viene unita ad altre celle. |
| [CellVerticalAlignment](./cellverticalalignment/) | Specifica la giustificazione verticale del testo all'interno di una cella di tabella. |
| [PreferredWidthType](./preferredwidthtype/) | Specifica l'unità di misura per la larghezza preferita di una tabella o cella. |
| [TableAlignment](./tablealignment/) | Specifica l'allineamento per una tabella in linea. |
| [TableStyleOptions](./tablestyleoptions/) | Specifica la modalità di applicazione dello stile tabella a una tabella. |
| [TextWrapping](./textwrapping/) | Specifica come il testo viene disposto attorno alla tabella. |
