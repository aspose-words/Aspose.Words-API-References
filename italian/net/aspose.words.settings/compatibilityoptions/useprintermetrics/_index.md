## CompatibilityOptions.UsePrinterMetrics property

Utilizza i parametri della stampante per visualizzare i documenti.

```csharp
public bool UsePrinterMetrics { get; set; }
```

## Osservazioni

I parametri della stampante possono variare a seconda dei driver utilizzati. Ad esempio, Windows "Microsoft OpenXPS Class Driver 2" e "Microsoft Print to PDF" forniscono parametri leggermente diversi. Pertanto, il layout del documento finale potrebbe cambiare se questa opzione è abilitata.
