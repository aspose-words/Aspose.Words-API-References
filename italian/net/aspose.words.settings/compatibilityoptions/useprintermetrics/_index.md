## CompatibilityOptions.UsePrinterMetrics property

Usa le metriche della stampante per visualizzare i documenti.

```csharp
public bool UsePrinterMetrics { get; set; }
```

### Osservazioni

Le metriche della stampante possono variare a seconda dei driver utilizzati. Ad esempio, Windows "Microsoft OpenXPS Class Driver 2" e "Microsoft Print to PDF" forniscono metriche leggermente diverse. Pertanto, il layout del documento finale potrebbe cambiare se questa opzione è abilitata.
