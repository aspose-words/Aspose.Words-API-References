## CompatibilityOptions.AutofitToFirstFixedWidthCell property

Consenti alle colonne della tabella di superare la larghezza preferita delle celle costituenti.

```csharp
public bool AutofitToFirstFixedWidthCell { get; set; }
```

### Osservazioni

L'opzione si chiama "Utilizza regole di adattamento automatico delle tabelle di Word 2003" nell'interfaccia utente di MS Word 2013. Influisce effettivamente sul modo in cui viene calcolata la griglia anche per le tabelle a layout fisso (in alcuni casi).
