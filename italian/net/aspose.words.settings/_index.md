IL**Aspose.Words.Settings**namespace contiene classi ed enum che vengono utilizzate per specificare varie impostazioni e opzioni memorizzate in un documento. Si tratta di unione di dati, protezione da scrittura, compatibilità e altre impostazioni.

## Classi

| Classe | Descrizione |
| --- | --- |
| [CompatibilityOptions](./compatibilityoptions/) | Contiene le opzioni di compatibilità (ovvero le preferenze dell'utente immesse nel**Compatibilità** scheda del**Opzioni** dialogo in Microsoft Word). |
| [HyphenationOptions](./hyphenationoptions/) | Consente di configurare le opzioni di sillabazione del documento. |
| [MailMergeSettings](./mailmergesettings/) | Specifica tutte le informazioni di unione di posta per un documento. |
| [Odso](./odso/) | Specifica le impostazioni ODSO (Office Data Source Object) per un'origine dati di stampa unione. |
| [OdsoFieldMapData](./odsofieldmapdata/) | Specifica come una colonna nell'origine dati esterna deve essere mappata ai campi di unione predefiniti all'interno del documento. |
| [OdsoFieldMapDataCollection](./odsofieldmapdatacollection/) | Una raccolta tipizzata di[`OdsoFieldMapData`](../aspose.words.settings/odsofieldmapdata/) oggetti. |
| [OdsoRecipientData](./odsorecipientdata/) | Rappresenta informazioni su un singolo record all'interno di un'origine dati esterna che deve essere escluso dalla stampa unione. |
| [OdsoRecipientDataCollection](./odsorecipientdatacollection/) | Una raccolta tipizzata di[`OdsoRecipientData`](../aspose.words.settings/odsorecipientdata/) |
| [ViewOptions](./viewoptions/) | Fornisce varie opzioni che controllano il modo in cui un documento viene visualizzato in Microsoft Word. |
| [WriteProtection](./writeprotection/) | Specifica le impostazioni di protezione da scrittura per un documento. |
