IL **Aspose.Words.Settings** namespace contiene classi ed enumerazioni che vengono utilizzate quando si specificano varie impostazioni e opzioni archiviate in un documento. Si tratta di stampa unione, protezione da scrittura, compatibilità e altre impostazioni.

## Classi

| Classe | Descrizione |
| --- | --- |
| [CompatibilityOptions](./compatibilityoptions/) | Contiene le opzioni di compatibilità (ovvero le preferenze dell'utente immesse nel file **Compatibilità** scheda del **Opzioni** finestra di dialogo in Microsoft Word). |
| [HyphenationOptions](./hyphenationoptions/) | Permette di configurare le opzioni di sillabazione del documento. |
| [MailMergeSettings](./mailmergesettings/) | Specifica tutte le informazioni sulla stampa unione per un documento. |
| [Odso](./odso/) | Specifica le impostazioni ODSO (Office Data Source Object) per un'origine dati di stampa unione. |
| [OdsoFieldMapData](./odsofieldmapdata/) | Specifica come una colonna nell'origine dati esterna deve essere mappata ai campi di unione predefiniti all'interno del documento. |
| [OdsoFieldMapDataCollection](./odsofieldmapdatacollection/) | Una raccolta digitata di[`OdsoFieldMapData`](../aspose.words.settings/odsofieldmapdata/) oggetti. |
| [OdsoRecipientData](./odsorecipientdata/) | Rappresenta informazioni su un singolo record all'interno di un'origine dati esterna che deve essere esclusa dalla stampa unione. |
| [OdsoRecipientDataCollection](./odsorecipientdatacollection/) | Una raccolta digitata di[`OdsoRecipientData`](../aspose.words.settings/odsorecipientdata/) |
| [ViewOptions](./viewoptions/) | Fornisce varie opzioni che controllano il modo in cui un documento viene visualizzato in Microsoft Word. |
| [WriteProtection](./writeprotection/) | Specifica le impostazioni di protezione da scrittura per un documento. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [Compatibility](./compatibility/) | Specifica i nomi delle opzioni di compatibilità. |
| [JustificationMode](./justificationmode/) | Specifica la regolazione della spaziatura dei caratteri per un documento. Il valore predefinito è`Espandere` . |
| [MailMergeCheckErrors](./mailmergecheckerrors/) | Specifica il modo in cui Microsoft Word segnalerà gli errori rilevati durante la stampa unione. |
| [MailMergeDataType](./mailmergedatatype/) | Specifica il tipo di un'origine dati di stampa unione esterna. |
| [MailMergeDestination](./mailmergedestination/) | Specifica i possibili risultati che possono essere generati quando viene eseguita una stampa unione su un documento. |
| [MailMergeMainDocumentType](./mailmergemaindocumenttype/) | Specifica i tipi possibili per un documento di origine di stampa unione. |
| [MsWordVersion](./mswordversion/) | Consente ad Aspose.Wods di imitare il comportamento dell'applicazione specifico della versione di MS Word. |
| [MultiplePagesType](./multiplepagestype/) | Specifica la modalità di stampa del documento. |
| [OdsoDataSourceType](./odsodatasourcetype/) | Specifica il tipo di origine dati esterna a cui connettersi come parte delle informazioni di connessione ODSO. |
| [OdsoFieldMappingType](./odsofieldmappingtype/) | Specifica i possibili tipi utilizzati per indicare se un determinato campo di stampa unione è stato mappato a una colonna nell'origine dati esterna specificata. |
| [ViewType](./viewtype/) | Valori possibili per la modalità di visualizzazione in Microsoft Word. |
| [ZoomType](./zoomtype/) | Valori possibili per quanto grande o piccolo appare il documento sullo schermo in Microsoft Word. |
