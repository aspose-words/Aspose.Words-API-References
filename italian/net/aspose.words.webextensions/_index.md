Il **Aspose.Words.WebExtensions** namespace fornisce classi che consentono di personalizzare elementi e attributi che estendono il vocabolario XML per rappresentare i componenti aggiuntivi di Office.

## Classi

| Classe | Descrizione |
| --- | --- |
| [BaseWebExtensionCollection&lt;T&gt;](./basewebextensioncollection-1/) | Classe base per[`TaskPaneCollection`](../aspose.words.webextensions/taskpanecollection/) ,[`WebExtensionBindingCollection`](../aspose.words.webextensions/webextensionbindingcollection/) , [`WebExtensionPropertyCollection`](../aspose.words.webextensions/webextensionpropertycollection/) e[`WebExtensionReferenceCollection`](../aspose.words.webextensions/webextensionreferencecollection/) raccolte. |
| [TaskPane](./taskpane/) | Rappresenta un oggetto del riquadro attività aggiuntivo. |
| [TaskPaneCollection](./taskpanecollection/) | Specifica un elenco di oggetti del riquadro attività persistenti. |
| [WebExtension](./webextension/) | Rappresenta un oggetto estensione web. |
| [WebExtensionBinding](./webextensionbinding/) | Specifica una relazione di associazione tra un'estensione Web e i dati nel documento. |
| [WebExtensionBindingCollection](./webextensionbindingcollection/) | Specifica un elenco di associazioni di estensioni Web. |
| [WebExtensionProperty](./webextensionproperty/) | Specifica una proprietà personalizzata dell'estensione Web. |
| [WebExtensionPropertyCollection](./webextensionpropertycollection/) | Specifica un insieme di proprietà personalizzate dell'estensione Web. |
| [WebExtensionReference](./webextensionreference/) | Rappresenta il riferimento a un'estensione Web. Il riferimento viene utilizzato per identificare la posizione del provider e la versione dell'estensione . |
| [WebExtensionReferenceCollection](./webextensionreferencecollection/) | Specifica un elenco di riferimenti alle estensioni Web. |
