Il **Aspose.Parole.Shaping** lo spazio dei nomi contiene classi per fornire il supporto per le funzionalità OpenType utilizzando motori di modellazione del testo esterni.

## Classi

| Classe | Descrizione |
| --- | --- |
| [BasicTextShaperCache](./basictextshapercache/) | Implementa la cache di base per[`ITextShaper`](../aspose.words.shaping/itextshaper/) istanze. Questa classe è thread-safe. |
| [Cluster](./cluster/) | Incapsula punti di codice e glifi che compongono un grafema. |
| [Glyph](./glyph/) | Rappresenta un glifo |
## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [ITextShaper](./itextshaper/) | Fornisce metodi per modellare il testo. |
| [ITextShaperFactory](./itextshaperfactory/) | Un'interfaccia di una fabbrica per costruire[`ITextShaper`](../aspose.words.shaping/itextshaper/) implementazioni. |
## Enumerazione

| Enumerazione | Descrizione |
| --- | --- |
| [Direction](./direction/) | Direzione del testo. |
| [FontFeature](./fontfeature/) | Le funzionalità forniscono informazioni su come vengono utilizzati i glifi in un carattere per eseguire il rendering di uno script. https://docs.microsoft.com/en-us/typography/opentype/spec/featuretags |
| [ScriptShapingLevel](./scriptshapinglevel/) | Descrive i livelli di shaping richiesti da uno script. |
| [UnicodeScript](./unicodescript/) | Proprietà database di caratteri Unicode: Script (sc). |
