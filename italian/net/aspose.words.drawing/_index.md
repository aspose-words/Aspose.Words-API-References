IL**Aspose.Parole.Disegno** namespace fornisce classi che consentono di creare e modificare oggetti di disegno.

| Classe | Descrizione |
| --- | --- |
| [Fill](./fill/) | Rappresenta la formattazione di riempimento per un oggetto. |
| [GradientStop](./gradientstop/) | Rappresenta un'interruzione del gradiente. |
| [GradientStopCollection](./gradientstopcollection/) | Contiene una raccolta di[`GradientStop`](../aspose.words.drawing/gradientstop/) oggetti. |
| [GroupShape](./groupshape/) | Rappresenta un gruppo di forme in un documento. |
| [HorizontalRuleFormat](./horizontalruleformat/) | Rappresenta la formattazione della regola orizzontale. |

| [OleFormat](./oleformat/) | Fornisce l'accesso ai dati di un oggetto OLE o di un controllo ActiveX. |
| [OlePackage](./olepackage/) | Permette di accedere alle proprietà del pacchetto OLE. |
| [ShadowFormat](./shadowformat/) | Rappresenta la formattazione shadow per un oggetto. |
| [Shape](./shape/) | Rappresenta un oggetto nel livello di disegno, ad esempio una forma, una casella di testo, una forma libera, un oggetto OLE, un controllo ActiveX o un'immagine. |
| [ShapeBase](./shapebase/) | Classe base per oggetti nel livello di disegno, ad esempio forme, forme libere, oggetti OLE, controlli ActiveX o immagini. |
| [SignatureLine](./signatureline/) | Fornisce l'accesso alle proprietà della riga della firma. |
| [Stroke](./stroke/) | Definisce un tratto per una forma. |
| [TextBox](./textbox/) | Definisce gli attributi che specificano come viene visualizzato un testo all'interno di una forma. |
| [TextPath](./textpath/) | Definisce il testo e la formattazione del percorso del testo (di un oggetto WordArt). |
