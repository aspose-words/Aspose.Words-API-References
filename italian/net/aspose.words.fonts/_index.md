IL**Aspose.Words.Fonts** namespace fornisce classi ed enumerazioni per accedere alle informazioni sui font utilizzati in un documento.

## Classi

| Classe | Descrizione |
| --- | --- |
| [DefaultFontSubstitutionRule](./defaultfontsubstitutionrule/) | Regola predefinita per la sostituzione dei font. |
| [FileFontSource](./filefontsource/) | Rappresenta il singolo file di font TrueType memorizzato nel file system. |
| [FolderFontSource](./folderfontsource/) | Rappresenta la cartella che contiene i file dei font TrueType. |
| [FontConfigSubstitutionRule](./fontconfigsubstitutionrule/) | Regola di sostituzione della configurazione del font. |
| [FontEmbeddingLicensingRights](./fontembeddinglicensingrights/) | Rappresenta i diritti di licenza di incorporamento per il font. |
| [FontFallbackSettings](./fontfallbacksettings/) | Specifica le impostazioni del meccanismo di fallback del font. |
| [FontInfo](./fontinfo/) | Specifica informazioni su un font utilizzato nel documento. |
| [FontInfoCollection](./fontinfocollection/) | Rappresenta una raccolta di font utilizzati in un documento. |
| [FontInfoSubstitutionRule](./fontinfosubstitutionrule/) | Regola di sostituzione delle informazioni sui font. |
| [FontNameSubstitutionRule](./fontnamesubstitutionrule/) | Regola di sostituzione del font per l'elaborazione del nome del font. |
| [FontSettings](./fontsettings/) | Specifica le impostazioni del font per un documento. |
| [FontSourceBase](./fontsourcebase/) | Questa è una classe base astratta per le classi che consentono all'utente di specificare varie sorgenti di font. |
| [FontSubstitutionRule](./fontsubstitutionrule/) | Questa è una classe base astratta per la regola di sostituzione dei font. |
| [FontSubstitutionSettings](./fontsubstitutionsettings/) | Specifica le impostazioni del meccanismo di sostituzione dei font. |
| [MemoryFontSource](./memoryfontsource/) | Rappresenta il singolo file di font TrueType memorizzato nella memoria. |
| [PhysicalFontInfo](./physicalfontinfo/) | Specifica informazioni sul font fisico disponibile per il motore di font Aspose.Words. |
| [StreamFontSource](./streamfontsource/) | Classe base per la sorgente del font del flusso definito dall'utente. |
| [SystemFontSource](./systemfontsource/) | Rappresenta tutti i font TrueType installati nel sistema. |
| [TableSubstitutionRule](./tablesubstitutionrule/) | Regola di sostituzione dei caratteri della tabella. |
