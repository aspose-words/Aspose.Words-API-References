Il **Aspose.Words.Fonts** namespace fornisce classi ed enumerazioni per accedere alle informazioni sui font utilizzati in un documento.

## Classi

| Classe | Descrizione |
| --- | --- |
| [DefaultFontSubstitutionRule](./defaultfontsubstitutionrule/) | Regola predefinita per la sostituzione dei caratteri. |
| [FileFontSource](./filefontsource/) | Rappresenta il singolo file di font TrueType archiviato nel file system. |
| [FolderFontSource](./folderfontsource/) | Rappresenta la cartella che contiene i file di font TrueType. |
| [FontConfigSubstitutionRule](./fontconfigsubstitutionrule/) | Regola di sostituzione della configurazione dei caratteri. |
| [FontFallbackSettings](./fontfallbacksettings/) | Specifica le impostazioni del meccanismo di fallback dei caratteri. |
| [FontInfo](./fontinfo/) | Specifica le informazioni su un font utilizzato nel documento. |
| [FontInfoCollection](./fontinfocollection/) | Rappresenta una raccolta di caratteri utilizzati in un documento. |
| [FontInfoSubstitutionRule](./fontinfosubstitutionrule/) | Regola di sostituzione delle informazioni sui caratteri. |
| [FontNameSubstitutionRule](./fontnamesubstitutionrule/) | Regola di sostituzione del carattere per l'elaborazione del nome del carattere. |
| [FontSettings](./fontsettings/) | Specifica le impostazioni dei caratteri per un documento. |
| [FontSourceBase](./fontsourcebase/) | Questa è una classe base astratta per le classi che consentono all'utente di specificare varie fonti di font. |
| [FontSubstitutionRule](./fontsubstitutionrule/) | Questa è una classe base astratta per la regola di sostituzione dei caratteri. |
| [FontSubstitutionSettings](./fontsubstitutionsettings/) | Specifica le impostazioni del meccanismo di sostituzione dei caratteri. |
| [MemoryFontSource](./memoryfontsource/) | Rappresenta il singolo file di font TrueType archiviato in memoria. |
| [PhysicalFontInfo](./physicalfontinfo/) | Specifica le informazioni sul carattere fisico disponibile per il motore di caratteri Aspose.Words. |
| [StreamFontSource](./streamfontsource/) | Classe base per l'origine del font stream definita dall'utente. |
| [SystemFontSource](./systemfontsource/) | Rappresenta tutti i font TrueType installati nel sistema. |
| [TableSubstitutionRule](./tablesubstitutionrule/) | Regola di sostituzione dei caratteri della tabella. |
