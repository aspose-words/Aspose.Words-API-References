## OfficeMath.GetMathRenderer method

Crea e restituisce un oggetto che può essere utilizzato per rendere questa equazione in un'immagine.

```csharp
public OfficeMathRenderer GetMathRenderer()
```

### Valore di ritorno

L'oggetto renderer per questa equazione.

### Osservazioni

Questo metodo invoca semplicemente il[`OfficeMathRenderer`](../../../aspose.words.rendering/officemathrenderer)costruttore e passa questo oggetto come parametro.

### Esempi

Mostra come eseguire il rendering di un oggetto Office Math in un file immagine nel file system locale.

```csharp
Document doc = new Document(MyDir + "Office math.docx");

OfficeMath math = (OfficeMath)doc.GetChild(NodeType.OfficeMath, 0, true);

// Crea un oggetto "ImageSaveOptions" da passare al metodo "Save" del renderer del nodo per modificarlo
// come esegue il rendering del nodo OfficeMath in un'immagine.
ImageSaveOptions saveOptions = new ImageSaveOptions(SaveFormat.Png);

// Imposta la proprietà "Scale" su 5 per rendere l'oggetto cinque volte la sua dimensione originale.
saveOptions.Scale = 5;

math.GetMathRenderer().Save(ArtifactsDir + "Shape.RenderOfficeMath.png", saveOptions);
```
