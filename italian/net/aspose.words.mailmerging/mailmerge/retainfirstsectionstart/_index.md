## MailMerge.RetainFirstSectionStart property

Ottiene o imposta un valore che indica se il[`SectionStart`](../../../aspose.words/pagesetup/sectionstart/) della prima sezione del documento e le sue copie per l'origine dati successiva, le righe vengono conservate durante la stampa unione o aggiornate in base al comportamento di MS Word.

```csharp
public bool RetainFirstSectionStart { get; set; }
```

### Osservazioni

Il valore predefinito è **VERO** .
