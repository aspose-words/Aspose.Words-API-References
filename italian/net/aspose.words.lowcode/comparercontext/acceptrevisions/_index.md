## ComparerContext.AcceptRevisions property

Indica se accettare le revisioni nei documenti prima di confrontarli. Se i documenti confrontati contengono revisioni e questo flag è impostato su falso, il processore rifiuterà le revisioni. Il valore predefinito è`VERO` .

```csharp
public bool AcceptRevisions { get; set; }
```
