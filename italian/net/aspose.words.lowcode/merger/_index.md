## Osservazioni

I file o flussi di input e output specificati, insieme alle opzioni di unione e salvataggio desiderate, vengono utilizzati per unire i documenti di input specificati in un unico documento di output.

La funzionalità di fusione supporta oltre 35 formati di file diversi.
