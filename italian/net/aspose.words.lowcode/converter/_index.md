## Osservazioni

I file o flussi di input e output specificati, insieme al formato di salvataggio desiderato, vengono utilizzati per convertire il documento di input specificato di un formato nel documento di output dell'altro formato specificato.

La funzionalità di conversione supporta oltre 35 formati di file diversi.

IL[`ConvertToImages`](./converttoimages/)Un gruppo di metodi è progettato per trasformare i documenti in immagini, , convertendo ogni pagina in un file immagine separato. Questi metodi convertono anche i documenti PDF direttamente in formati a pagina fissa senza caricarli nel modello di documento, il che migliora sia le prestazioni che la precisione.

Con[`PageSet`](../../aspose.words.saving/imagesaveoptions/pageset/), puoi specificare un set specifico di pagine da convertire in immagini.
