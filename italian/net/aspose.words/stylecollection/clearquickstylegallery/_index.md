## StyleCollection.ClearQuickStyleGallery method

Rimuove tutti gli stili dal pannello Galleria stili rapidi.

```csharp
public void ClearQuickStyleGallery()
```

## Esempi

Mostra come rimuovere gli stili dal pannello Galleria stili.

```csharp
Document doc = new Document();
// Nota che per ora la rimozione degli stili funziona solo con il formato DOCX.
doc.Styles.ClearQuickStyleGallery();

doc.Save(ArtifactsDir + "Styles.RemoveStylesFromStyleGallery.docx");
```
