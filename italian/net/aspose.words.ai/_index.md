IL**Aspose.Words.AI** namespace consente un'integrazione fluida con modelli linguistici di grandi dimensioni (LLM), come OpenAI e Gemini, fornendo un solido supporto per l'elaborazione di documenti basata sull'intelligenza artificiale.

## Classi

| Classe | Descrizione |
| --- | --- |
| [AiModel](./aimodel/) | Rappresenta informazioni su un modello linguistico generativo. |
| [AnthropicAiModel](./anthropicaimodel/) | Una classe astratta che rappresenta l'integrazione con i modelli di intelligenza artificiale di Anthropic all'interno di Aspose.Words. |
| [CheckGrammarOptions](./checkgrammaroptions/) | Consente di specificare varie opzioni durante il controllo grammaticale di un documento tramite AI. |
| [GoogleAiModel](./googleaimodel/) | Una classe astratta che rappresenta l'integrazione con i modelli di intelligenza artificiale di Google all'interno di Aspose.Words. |
| [OpenAiModel](./openaimodel/) | Una classe astratta che rappresenta l'integrazione con i grandi modelli linguistici di OpenAI all'interno di Aspose.Words. |
| [SummarizeOptions](./summarizeoptions/) | Consente di specificare diverse opzioni per riassumere il contenuto del documento. |
## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IAiModelText](./iaimodeltext/) | L'interfaccia comune per i modelli di intelligenza artificiale progettati per generare una varietà di contenuti basati su testo. |
