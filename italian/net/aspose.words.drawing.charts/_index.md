Il **Aspose.Words.Drawing.Grafici** namespace fornisce classi che consentono di accedere ai grafici nei documenti di Microsoft Word.

## Classi

| Classe | Descrizione |
| --- | --- |
| [AxisBound](./axisbound/) | Rappresenta il limite minimo o massimo dei valori dell'asse. |
| [AxisDisplayUnit](./axisdisplayunit/) | Fornisce l'accesso alle opzioni di scala delle unità di visualizzazione per l'asse dei valori. |
| [AxisScaling](./axisscaling/) | Rappresenta le opzioni di ridimensionamento dell'asse. |
| [Chart](./chart/) | Fornisce l'accesso alle proprietà della forma del grafico. |
| [ChartAxis](./chartaxis/) | Rappresenta le opzioni dell'asse del grafico. |
| [ChartDataLabel](./chartdatalabel/) | Rappresenta l'etichetta dei dati su un punto del grafico o una linea di tendenza. |
| [ChartDataLabelCollection](./chartdatalabelcollection/) | Rappresenta una raccolta di[`ChartDataLabel`](../aspose.words.drawing.charts/chartdatalabel/) . |
| [ChartDataPoint](./chartdatapoint/) | Consente di specificare la formattazione di un singolo punto dati sul grafico. |
| [ChartDataPointCollection](./chartdatapointcollection/) | Rappresenta la raccolta di a[`ChartDataPoint`](../aspose.words.drawing.charts/chartdatapoint/) . |
| [ChartFormat](./chartformat/) | Rappresenta la formattazione di un elemento del grafico. |
| [ChartLegend](./chartlegend/) | Rappresenta le proprietà della legenda del grafico. |
| [ChartLegendEntry](./chartlegendentry/) | Rappresenta una voce della legenda del grafico. |
| [ChartLegendEntryCollection](./chartlegendentrycollection/) | Rappresenta una raccolta di voci della legenda del grafico. |
| [ChartMarker](./chartmarker/) | Rappresenta un indicatore di dati del grafico. |
| [ChartNumberFormat](./chartnumberformat/) | Rappresenta la formattazione del numero dell'elemento padre. |
| [ChartSeries](./chartseries/) | Rappresenta le proprietà della serie di grafici. |
| [ChartSeriesCollection](./chartseriescollection/) | Rappresenta la raccolta di a[`ChartSeries`](../aspose.words.drawing.charts/chartseries/) . |
| [ChartTitle](./charttitle/) | Fornisce l'accesso alle proprietà del titolo del grafico. |
