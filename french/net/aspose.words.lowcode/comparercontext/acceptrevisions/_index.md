## ComparerContext.AcceptRevisions property

Indique s'il faut accepter les révisions dans les documents avant de les comparer. Si les documents comparés contiennent des révisions et que cet indicateur est défini sur faux, le processeur rejettera les révisions. La valeur par défaut est`vrai` .

```csharp
public bool AcceptRevisions { get; set; }
```
