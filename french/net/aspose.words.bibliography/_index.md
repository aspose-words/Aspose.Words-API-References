Le**Aspose.Words.Bibliographie**namespace fournit des classes qui permettent d'accéder à une liste de sources bibliographiques disponibles dans un document.

## Des classes

| Classer | La description |
| --- | --- |
| [Bibliography](./bibliography/) | Représente la liste des sources bibliographiques disponibles dans le document. |
| [Contributor](./contributor/) | Représente un contributeur à la source bibliographique. Il peut s'agir d'une entreprise (une organisation) ou d'une liste de personnes. |
| [ContributorCollection](./contributorcollection/) | Représente les contributeurs de la source bibliographique. |
| [Corporate](./corporate/) | Représente un contributeur de source de bibliographie d'entreprise (une organisation). |
| [Person](./person/) | Représente le contributeur individuel (une personne) à la source de la bibliographie. |
| [PersonCollection](./personcollection/) | Représente une liste de personnes qui sont des contributeurs de sources bibliographiques. |
| [Source](./source/) | Représente une source individuelle, telle qu'un livre, un article de journal ou une interview. |
