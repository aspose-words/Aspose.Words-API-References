## ChartDataLabelCollection.ShowDataLabelsRange property

Permet de spécifier si les valeurs de la plage des étiquettes de données doivent être affichées dans les étiquettes de données de la série entière. La valeur par défaut est`FAUX` .

```csharp
public bool ShowDataLabelsRange { get; set; }
```

### Remarques

La valeur définie pour cette propriété peut être remplacée pour une étiquette de données individuelle en utilisant the [`ShowDataLabelsRange`](../../chartdatalabel/showdatalabelsrange/) propriété.
