## ChartXValueType enumeration

Permet de spécifier le type d'une valeur X d'une série de graphiques.

```csharp
public enum ChartXValueType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| String | `0` | Spécifie qu'une valeur X est une catégorie de chaîne. |
| Double | `1` | Spécifie qu'une valeur X est un nombre à virgule flottante double précision. |
| DateTime | `2` | Spécifie qu'une valeur X est une date et une heure du jour. |
| Time | `3` | Spécifie qu'une valeur X est une heure de la journée. |
| Multilevel | `4` | Spécifie qu'une valeur X est une valeur à plusieurs niveaux. |
