## Insert(*int, [ChartXValue](../../chartxvalue/)*) {#insert}

Insère la valeur X spécifiée dans la série de graphiques à l'index spécifié. Si la série prend en charge les valeurs Y et les tailles de bulles, elles seront vides pour la valeur X.

```csharp
public void Insert(int index, ChartXValue xValue)
```

## Remarques

Le point de données correspondant avec le formatage par défaut sera inséré dans la collection de points de données. Et, si des étiquettes de données sont affichées, l'étiquette de données correspondante avec le formatage par défaut sera également insérée.
