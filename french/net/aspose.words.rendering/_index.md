Le**Aspose.Words.Rendering** L'espace de noms fournit des classes qui permettent de personnaliser l'impression ou le rendu des documents.

## Des classes

| Classer | La description |
| --- | --- |
| [AsposeWordsPrintDocument](./asposewordsprintdocument/) | Fournit une implémentation par défaut pour l'impression d'un[`Document`](../aspose.words/document/)dans le framework d'impression .NET. |
| [NodeRendererBase](./noderendererbase/) | Classe de base pour[`ShapeRenderer`](../aspose.words.rendering/shaperenderer/) et[`OfficeMathRenderer`](../aspose.words.rendering/officemathrenderer/) . |
| [OfficeMathRenderer](./officemathrenderer/) | Fournit des méthodes pour restituer un individu[`OfficeMath`](../aspose.words.math/officemath/) à une image raster ou vectorielle ou à un objet graphique. |
| [PageInfo](./pageinfo/) | Représente des informations sur une page de document particulière. |
| [PrinterSettingsContainer](./printersettingscontainer/) | Représente un stockage pour certains paramètres dePrinterSettings objet. |
| [ShapeRenderer](./shaperenderer/) | Fournit des méthodes pour restituer un individu[`Shape`](../aspose.words.drawing/shape/) ou[`GroupShape`](../aspose.words.drawing/groupshape/) à une image raster ou vectorielle ou à un objet graphique. |
| [ThumbnailGeneratingOptions](./thumbnailgeneratingoptions/) | Peut être utilisé pour spécifier des options supplémentaires lors de la génération d'une vignette pour un document. |
