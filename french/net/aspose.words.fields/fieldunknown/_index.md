## Exemples

Montre comment travailler avec le champ « FieldNone » dans un document.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// Insérer un champ qui ne désigne pas un type de champ objectif dans son code de champ.
Field field = builder.InsertField(" NOTAREALFIELD //un");

// Le type de champ « FieldNone » est réservé à des champs tels que ceux-ci.
Assert.AreEqual(FieldType.FieldNone, field.Type);

// Nous pouvons également toujours travailler avec ces champs et les attribuer comme instances de la classe FieldUnknown.
FieldUnknown fieldUnknown = (FieldUnknown)field;
Assert.AreEqual(" NOTAREALFIELD //a", fieldUnknown.GetFieldCode());
```
