## PdfAttachmentsEmbeddingMode enumeration

Spécifie comment les pièces jointes sont intégrées au document PDF.

```csharp
public enum PdfAttachmentsEmbeddingMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Les pièces jointes ne sont pas intégrées. |
| Annotations | `1` | Les pièces jointes sont intégrées sous forme d'annotations avec l'emplacement sur la page. |
| DocumentEmbeddedFiles | `2` | Les pièces jointes sont intégrées sous forme de fichiers intégrés au document sans annotations. |
