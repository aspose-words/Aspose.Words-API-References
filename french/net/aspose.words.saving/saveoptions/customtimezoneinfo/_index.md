## SaveOptions.CustomTimeZoneInfo property

Obtient ou définit le fuseau horaire local personnalisé utilisé pour les champs de date/heure.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

## Remarques

Cette option est disponible dans .Net Framework à partir de la version 3.5 ou .Net Standard.

Par défaut, Aspose.Words utilise le fuseau horaire local du système lors de l'écriture des champs de date/heure, cette option permet de définir une valeur personnalisée.
