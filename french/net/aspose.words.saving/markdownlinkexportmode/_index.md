## MarkdownLinkExportMode enumeration

Le mode d'export des liens vers un document cible.

```csharp
public enum MarkdownLinkExportMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Auto | `0` | Un lien est exporté comme bloc de référence s'il contient des informations aller-retour ou s'il est mentionné plus d'une fois dans un document. Dans tous les autres cas, un lien est exporté sous forme de bloc en ligne. |
| Inline | `1` | Les liens sont exportés sous forme de blocs en ligne. |
| Reference | `2` | Les liens sont exportés sous forme de blocs de référence. |
