Le**Aspose.Words.Listes** L'espace de noms contient des classes pour working avec des listes à puces et numérotées définies dans un document.

## Des classes

| Classer | La description |
| --- | --- |
| [List](./list/) | Représente le formatage d'une liste. |
| [ListCollection](./listcollection/) | Stocke et gère le formatage des listes à puces et numérotées utilisées dans un document. |
| [ListFormat](./listformat/) | Permet de contrôler quel formatage de liste est appliqué à un paragraphe. |
| [ListLabel](./listlabel/) | Définit les propriétés spécifiques à une étiquette de liste. |
| [ListLevel](./listlevel/) | Définit le formatage pour un niveau de liste. |
| [ListLevelCollection](./listlevelcollection/) | Une collection de formats de liste pour chaque niveau d'une liste. |
## Énumération

| Énumération | La description |
| --- | --- |
| [ListLevelAlignment](./listlevelalignment/) | Spécifie l'alignement du numéro de liste ou de la puce. |
| [ListTemplate](./listtemplate/) | Spécifie l'un des formats de liste prédéfinis disponibles dans Microsoft Word. |
| [ListTrailingCharacter](./listtrailingcharacter/) | Spécifie le caractère qui sépare l'étiquette de la liste du texte du paragraphe. |
