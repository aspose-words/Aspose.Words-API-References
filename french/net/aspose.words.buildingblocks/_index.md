Le**Aspose.Words.BuildingBlocks**L'espace de noms fournit des classes qui permettent d'accéder à et d'utiliser les entrées AutoText, AutoCorrect et Building Blocks dans un document.

## Des classes

| Classer | La description |
| --- | --- |
| [BuildingBlock](./buildingblock/) | Représente une entrée de document de glossaire telle qu'un bloc de construction, un texte automatique ou une entrée de correction automatique. |
| [BuildingBlockCollection](./buildingblockcollection/) | Une collection de[`BuildingBlock`](../aspose.words.buildingblocks/buildingblock/) objets dans le document. |
| [GlossaryDocument](./glossarydocument/) | Représente l'élément racine d'un document de glossaire dans un document Word. Un document de glossaire est un espace de stockage pour les entrées de texte automatique, de correction automatique et les blocs de construction. |
## Énumération

| Énumération | La description |
| --- | --- |
| [BuildingBlockBehavior](./buildingblockbehavior/) | Spécifie le comportement qui doit être appliqué au contenu du bloc de construction lorsqu'il est inséré dans le document principal. |
| [BuildingBlockGallery](./buildingblockgallery/) | Spécifie la galerie prédéfinie dans laquelle un bloc de construction est classé. |
| [BuildingBlockType](./buildingblocktype/) | Spécifie un type de bloc de construction. Ce type peut affecter la visibilité et le comportement du bloc de construction dans Microsoft Word. |
