Le**Aspose.Words.AI** L'espace de noms permet une intégration transparente avec de grands modèles de langage (LLM), tels qu'OpenAI et Gemini, offrant un support robuste pour le traitement de documents alimenté par l'IA.

## Des classes

| Classer | La description |
| --- | --- |
| [AiModel](./aimodel/) | Représente des informations sur un modèle de langage génératif. |
| [AnthropicAiModel](./anthropicaimodel/) | Une classe abstraite représentant l'intégration avec les modèles d'IA d'Anthropic dans Aspose.Words. |
| [CheckGrammarOptions](./checkgrammaroptions/) | Permet de spécifier diverses options lors de la vérification de la grammaire d'un document à l'aide de l'IA. |
| [GoogleAiModel](./googleaimodel/) | Une classe abstraite représentant l'intégration avec les modèles d'IA de Google dans Aspose.Words. |
| [OpenAiModel](./openaimodel/) | Une classe abstraite représentant l'intégration avec les grands modèles de langage d'OpenAI dans Aspose.Words. |
| [SummarizeOptions](./summarizeoptions/) | Permet de spécifier différentes options pour résumer le contenu du document. |
## Interfaces

| --- | --- |
