Le**Aspose.Words.Tables** L'espace de noms contient des classes qui représentent des tableaux, des lignes, des cellules et leur mise en forme.

| Classer | La description |
| --- | --- |
| [Cell](./cell/) | Représente une cellule de tableau. |
| [CellCollection](./cellcollection/) | Fournit un accès typé à une collection de[`Cell`](../aspose.words.tables/cell/) nœuds. |
| [CellFormat](./cellformat/) | Représente toute la mise en forme d'une cellule de tableau. |
| [PreferredWidth](./preferredwidth/) | Représente une valeur et son unité de mesure qui est utilisée pour spécifier la largeur préférée d'un tableau ou d'une cellule. |
| [Row](./row/) | Représente une ligne de tableau. |
| [RowCollection](./rowcollection/) | Fournit un accès typé à une collection de[`Row`](../aspose.words.tables/row/) nœuds. |
| [RowFormat](./rowformat/) | Représente tout le formatage d'une ligne de tableau. |
| [Table](./table/) | Représente un tableau dans un document Word. |
| [TableCollection](./tablecollection/) | Fournit un accès typé à une collection de[`Table`](../aspose.words.tables/table/) nœuds. |
