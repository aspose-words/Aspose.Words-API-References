Le**Aspose.Words.Reporting** L'espace de noms contient les classes du nouveau moteur de reporting d'Aspose.Words qui prend en charge les modèles de rapport balisés avec un langage basé sur la syntaxe de méthode LINQ.

## Des classes

| Classer | La description |
| --- | --- |
| [CsvDataLoadOptions](./csvdataloadoptions/) | Représente les options d'analyse des données CSV. |
| [CsvDataSource](./csvdatasource/) | Fournit l'accès aux données d'un fichier CSV ou d'un flux à utiliser dans un rapport. |
| [JsonDataLoadOptions](./jsondataloadoptions/) | Représente les options d'analyse des données JSON. |
| [JsonDataSource](./jsondatasource/) | Fournit l'accès aux données d'un fichier ou d'un flux JSON à utiliser dans un rapport. |
| [KnownTypeSet](./knowntypeset/) | Représente un ensemble non ordonné (c'est-à-dire une collection d'éléments uniques) contenantType objets dont les noms entièrement ou partiellement qualifiés peuvent être utilisés dans les modèles de rapport pour appeler les membres statiques des types correspondants, effectuer des conversions de type, etc. |
| [ReportingEngine](./reportingengine/) | Fournit des routines pour remplir les documents modèles avec des données et un ensemble de paramètres pour contrôler ces routines. |
| [XmlDataLoadOptions](./xmldataloadoptions/) | Représente les options de chargement de données XML. |
| [XmlDataSource](./xmldatasource/) | Fournit l'accès aux données d'un fichier ou d'un flux XML à utiliser dans un rapport. |
