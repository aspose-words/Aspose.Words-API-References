## ReportingEngine.GetRestrictedTypes method

Renvoie les types, les membres ainsi que les membres des types dérivés qui doivent être inaccessibles par le moteur via la syntaxe du modèle.

```csharp
public static Type[] GetRestrictedTypes()
```

### Return_Value

Types, quels membres ainsi que quels membres de types dérivés doivent être inaccessibles par le moteur via la syntaxe du modèle.

## Remarques

Le tableau renvoyé contient des éléments précédemment définis à l'aide[`SetRestrictedTypes`](../setrestrictedtypes/) .

La modification des éléments du tableau renvoyé n'a aucun effet sur les types restreints. Pour modifier les types restreints, utilisez [`SetRestrictedTypes`](../setrestrictedtypes/) à la place.
