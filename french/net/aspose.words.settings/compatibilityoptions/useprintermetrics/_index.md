## CompatibilityOptions.UsePrinterMetrics property

Utiliser les mesures de l'imprimante pour afficher les documents.

```csharp
public bool UsePrinterMetrics { get; set; }
```

### Remarques

Les mesures de l'imprimante peuvent différer selon les pilotes utilisés. Par exemple, Windows "Microsoft OpenXPS Class Driver 2" et "Microsoft Print to PDF" fournissent des mesures légèrement différentes. Par conséquent, la mise en page du document final peut changer si cette option est activée.
