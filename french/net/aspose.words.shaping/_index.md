Le**Aspose.Mots.Formation** L'espace de noms contient des classes pour fournir la prise en charge des fonctionnalités OpenType à l'aide de moteurs de mise en forme de texte externes.

| Classer | La description |
| --- | --- |

| [Cluster](./cluster/) | Encapsule les points de code et les glyphes composant un graphème. |
| [Glyph](./glyph/) | Représente un glyphe |
| [VariationAxisCoordinate](./variationaxiscoordinate/) | Représente une coordonnée d'axe. |
