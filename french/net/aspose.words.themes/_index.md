La **Aspose.Words.Themes** namespace fournit des classes qui permettent d'accéder aux informations de thème dans les documents Microsoft Word.

## Des classes

| Classer | La description |
| --- | --- |
| [Theme](./theme/) | Représente le thème du document et donne accès aux principales parties du thème, y compris[`MajorFonts`](../aspose.words.themes/theme/majorfonts/) ,[`MinorFonts`](../aspose.words.themes/theme/minorfonts/) et[`Colors`](../aspose.words.themes/theme/colors/) |
| [ThemeColors](./themecolors/) | Représente le jeu de couleurs du thème du document qui contient douze couleurs. |
| [ThemeFonts](./themefonts/) | Représente une collection de polices dans le jeu de polices, permettant de spécifier différentes polices pour différentes langues[`Latin`](../aspose.words.themes/themefonts/latin/) ,[`EastAsian`](../aspose.words.themes/themefonts/eastasian/) et[`ComplexScript`](../aspose.words.themes/themefonts/complexscript/) . |
## Énumération

| Énumération | La description |
| --- | --- |
| [ThemeColor](./themecolor/) | Spécifie les couleurs de thème pour les thèmes de document. |
| [ThemeFont](./themefont/) | Spécifie les types de noms de police de thème pour les thèmes de document. |
