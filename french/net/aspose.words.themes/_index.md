Le**Aspose.Words.Thèmes** namespace fournit des classes qui permettent d'accéder aux informations thématiques dans les documents Microsoft Word.

## Des classes

| Classer | La description |
| --- | --- |
| [Theme](./theme/) | Représente le thème du document et donne accès aux principales parties du thème, notamment[`MajorFonts`](../aspose.words.themes/theme/majorfonts/) ,[`MinorFonts`](../aspose.words.themes/theme/minorfonts/) et[`Colors`](../aspose.words.themes/theme/colors/) |
| [ThemeColors](./themecolors/) | Représente le jeu de couleurs du thème du document qui contient douze couleurs. |
| [ThemeFonts](./themefonts/) | Représente une collection de polices dans le schéma de polices, permettant de spécifier différentes polices pour différentes langues[`Latin`](../aspose.words.themes/themefonts/latin/) ,[`EastAsian`](../aspose.words.themes/themefonts/eastasian/) et[`ComplexScript`](../aspose.words.themes/themefonts/complexscript/) . |
## Énumération

| Énumération | La description |
| --- | --- |
| [ThemeColor](./themecolor/) | Spécifie les couleurs de thème pour les thèmes de document. |
| [ThemeFont](./themefont/) | Spécifie les types de noms de polices de thème pour les thèmes de document. |
