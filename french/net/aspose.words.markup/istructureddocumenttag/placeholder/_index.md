## IStructuredDocumentTag.Placeholder property

Obtient le[`BuildingBlock`](../../../aspose.words.buildingblocks/buildingblock/)contenant du texte d'espace réservé qui doit être affiché lorsque le contenu de cette exécution SDT est vide, l'élément XML mappé associé est vide comme spécifié via le[`XmlMapping`](../xmlmapping/) element ou le[`IsShowingPlaceholderText`](../isshowingplaceholdertext/) l'élément est vrai.

```csharp
public BuildingBlock Placeholder { get; }
```

## Remarques

Peut être nul, ce qui signifie que l'espace réservé n'est pas applicable pour ce Sdt.
