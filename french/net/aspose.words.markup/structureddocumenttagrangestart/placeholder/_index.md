## StructuredDocumentTagRangeStart.Placeholder property

Obtient le[`BuildingBlock`](../../../aspose.words.buildingblocks/buildingblock/)contenant un texte d'espace réservé qui doit être affiché lorsque le contenu de cette balise de document structuré est vide, l'élément XML mappé associé est vide comme spécifié via le[`XmlMapping`](../xmlmapping/) élément ou le[`IsShowingPlaceholderText`](../isshowingplaceholdertext/) l'élément est`vrai` .

```csharp
public BuildingBlock Placeholder { get; }
```

## Remarques

Peut être`nul`, ce qui signifie que l'espace réservé n'est pas applicable à cette balise de document structuré.
