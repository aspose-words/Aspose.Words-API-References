## LineSpacingRule enumeration

Spécifie les valeurs d'espacement des lignes pour un paragraphe.

```csharp
public enum LineSpacingRule
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| AtLeast | `0` | L'espacement des lignes peut être supérieur ou égal, mais jamais inférieur, à la valeur spécifiée dans le[`LineSpacing`](../paragraphformat/linespacing/) propriété. |
| Exactly | `1` | L'espacement des lignes ne change jamais par rapport à la valeur spécifiée dans the [`LineSpacing`](../paragraphformat/linespacing/) propriété, même si une police plus grande est utilisée dans le paragraphe. |
| Multiple | `2` | L'interligne est spécifié dans le[`LineSpacing`](../paragraphformat/linespacing/) propriété comme nombre de lignes. Une ligne équivaut à 12 points. |
