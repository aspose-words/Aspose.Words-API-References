## LoadOptions.UseSystemLcid property

Obtient ou définit s'il faut utiliser la valeur LCID obtenue à partir du registre Windows pour déterminer les marges par défaut de la configuration de la page.

```csharp
public bool UseSystemLcid { get; set; }
```

## Remarques

Si défini sur`vrai`, puis le comportement de MS Word est émulé, ce qui prend la valeur LCID du registre Windows.

La valeur par défaut est`FAUX`.
