## Shape.UpdateSmartArtDrawing method

Met à jour le dessin pré-rendu SmartArt en utilisant le moteur de rendu à froid SmartArt d'Aspose.Words.

```csharp
public void UpdateSmartArtDrawing()
```

### Remarques

Microsoft Word génère et enregistre le dessin pré-rendu avec l'objet SmartArt. Cependant, si le document est enregistré par d'autres applications, le dessin SmartArt pré-rendu peut être manquant ou incorrect. Si un dessin pré-rendu est disponible, Aspose.Words l'utilise pour restituer l'objet SmartArt. Si un dessin pré-rendu est manquant alors Aspose.Words utilise son propre moteur de rendu à froid SmartArt pour rendre l'objet SmartArt . Si le dessin pré-rendu est incorrect, il est nécessaire d'appeler cette méthode pour appeler le moteur de rendu à froid SmartArt .
