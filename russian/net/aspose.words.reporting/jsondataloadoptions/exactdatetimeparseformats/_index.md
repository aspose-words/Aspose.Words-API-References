## JsonDataLoadOptions.ExactDateTimeParseFormats property

Получает или задает точные форматы для анализа значений даты и времени JSON при загрузке JSON. Значение по умолчанию: **null** .

```csharp
public IEnumerable<string> ExactDateTimeParseFormats { get; set; }
```

### Примечания

Строки, закодированные с использованием формата даты и времени Microsoft® JSON (для Например, "/Date(1224043200000)/") всегда распознаются как значения даты и времени независимо от значения этого свойства. Свойство определяет дополнительные форматы для использования при анализе значений даты и времени из строк следующим образом:

* Когда`ExactDateTimeParseFormats`is **null** , формат ISO-8601 и все форматы даты и времени поддерживаемые для текущего языка, английский язык США и английский язык Новой Зеландии используются дополнительно в упомянутый порядок.
* Когда`ExactDateTimeParseFormats`содержит строки, они используются как дополнительные форматы даты и времени , использующие текущий язык и региональные параметры.
* Когда`ExactDateTimeParseFormats`пуст, дополнительные форматы даты и времени не используются.
