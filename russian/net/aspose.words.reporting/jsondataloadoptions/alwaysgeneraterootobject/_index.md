## JsonDataLoadOptions.AlwaysGenerateRootObject property

Получает или задает флаг, указывающий, всегда ли сгенерированный источник данных будет содержать объект для корневого элемента JSON . Если корневой элемент JSON содержит одно сложное свойство, такой объект по умолчанию не создается.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

### Примечания

Значение по умолчанию: **false** .
