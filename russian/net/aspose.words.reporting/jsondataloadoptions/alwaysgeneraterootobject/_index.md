## JsonDataLoadOptions.AlwaysGenerateRootObject property

Получает или задает флаг, указывающий, будет ли сгенерированный источник данных всегда содержать объект для элемента JSON root . Если корневой элемент JSON содержит одно сложное свойство, такой объект по умолчанию не создается.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

### Примечания

Значение по умолчанию:`ЛОЖЬ` .
