## Характеристики

| Имя | Описание |

| [AlwaysGenerateRootObject](../../aspose.words.reporting/xmldataloadoptions/alwaysgeneraterootobject/) { get; set; } | Возвращает или задает флаг, указывающий, будет ли сгенерированный источник данных всегда содержать объект для корневого элемента XML . Если корневой элемент XML не имеет атрибутов и все его дочерние элементы имеют одинаковые имена, такой объект по умолчанию не создается. |
