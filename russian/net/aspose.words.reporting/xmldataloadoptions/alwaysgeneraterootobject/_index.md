## XmlDataLoadOptions.AlwaysGenerateRootObject property

Возвращает или задает флаг, указывающий, будет ли сгенерированный источник данных всегда содержать объект для корневого элемента XML . Если корневой элемент XML не имеет атрибутов и все его дочерние элементы имеют одинаковые имена, такой объект по умолчанию не создается.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

## Примечания

Значение по умолчанию:`ЛОЖЬ` .
