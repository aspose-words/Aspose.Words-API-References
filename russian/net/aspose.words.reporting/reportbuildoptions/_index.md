## ReportBuildOptions enumeration

Указывает параметры, управляющие поведением[`ReportingEngine`](../reportingengine/) при построении отчета.

```csharp
[Flags]
public enum ReportBuildOptions
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Указывает параметры по умолчанию. |
| AllowMissingMembers | `1` | Указывает, что отсутствующие члены объекта должны рассматриваться движком как нулевые литералы. Этот параметр влияет только на доступ к членам объекта экземпляра (то есть нестатическим) и методам расширения. Если этот параметр не установлен, движок выдает исключение при обнаружении отсутствующего члена объекта. |
| RemoveEmptyParagraphs | `2` | Указывает, что движок должен удалять абзацы, которые становятся пустыми после того, как теги синтаксиса шаблона удаляются или заменяются пустыми значениями. |
| InlineErrorMessages | `4` | Указывает, что движок должен встраивать сообщения об ошибках синтаксиса шаблона в выходные документы. Если этот параметр не установлен, движок выдает исключение при обнаружении ошибки синтаксиса. |
| UseLegacyHeaderFooterVisiting | `8` | Указывает, что движок должен посещать дочерние узлы раздела (заголовки, нижние колонтитулы, тела) в порядке совместимом с версиями Aspose.Words до 21.9. |
| RespectJpegExifOrientation | `10` | Указывает, что движок должен использовать значения ориентации изображения EXIF ​​для соответствующего поворота вставленных изображений JPEG. |
| UpdateFieldsSyntaxAware | `20` | Указывает, что движок должен игнорировать синтаксис шаблона в результатах полей и обновлять поля после построения отчета . |
