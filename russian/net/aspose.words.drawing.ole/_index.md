The**Aspose.Words.Рисунок.Ole** Пространство имен содержит классы для работы с элементами управления OLE/ActiveX и формами в документах Microsoft Word.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [CheckBoxControl](./checkboxcontrol/) | Элемент управления CheckBox переключает значение. |
| [CommandButtonControl](./commandbuttoncontrol/) | Элемент управления CommandButton запускает макрос, который выполняет действие, когда пользователь нажимает на него. |
| [Forms2OleControl](./forms2olecontrol/) | Представляет элемент управления Microsoft Forms 2.0 OLE. |
| [Forms2OleControlCollection](./forms2olecontrolcollection/) | Представляет собой коллекцию[`Forms2OleControl`](../aspose.words.drawing.ole/forms2olecontrol/) объекты. |
| [MorphDataControl](./morphdatacontrol/) | Структура MorphDataControl представляет собой совокупность шести элементов управления: CheckBox, ComboBox, ListBox, OptionButton, TextBox и ToggleButton. |
| [OleControl](./olecontrol/) | Представляет элемент управления OLE ActiveX. |
| [OptionButtonControl](./optionbuttoncontrol/) | Элемент управления OptionButton позволяет сделать один выбор из ограниченного набора взаимоисключающих вариантов. |
| [TextBoxControl](./textboxcontrol/) | Элемент управления TextBox отображает текст из организованного набора данных или пользовательского ввода. |
