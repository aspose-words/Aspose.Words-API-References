The**Aspose.Words.AI** Пространство имен обеспечивает бесшовную интеграцию с большими языковыми моделями (LLM), такими как OpenAI и Gemini, , обеспечивая надежную поддержку обработки документов с помощью ИИ.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [AiModel](./aimodel/) | Представляет информацию о генеративной языковой модели. |
| [AnthropicAiModel](./anthropicaimodel/) | Абстрактный класс, представляющий интеграцию с моделями искусственного интеллекта Anthropic в Aspose.Words. |
| [CheckGrammarOptions](./checkgrammaroptions/) | Позволяет указать различные параметры при проверке грамматики документа с использованием ИИ. |
| [GoogleAiModel](./googleaimodel/) | Абстрактный класс, представляющий интеграцию с моделями ИИ Google в Aspose.Words. |
| [OpenAiModel](./openaimodel/) | Абстрактный класс, представляющий интеграцию с большими языковыми моделями OpenAI в Aspose.Words. |
| [SummarizeOptions](./summarizeoptions/) | Позволяет указать различные параметры для резюмирования содержимого документа. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IAiModelText](./iaimodeltext/) | Общий интерфейс для моделей ИИ, предназначенный для генерации разнообразного текстового контента. |
