## MailMerge.RetainFirstSectionStart property

Возвращает или задает значение, указывающее, является ли[`SectionStart`](../../../aspose.words/pagesetup/sectionstart/) первого раздела документа и его копии для последующих строк источника данных сохраняются во время слияния почты или обновляются в соответствии с поведением MS Word.

```csharp
public bool RetainFirstSectionStart { get; set; }
```

## Примечания

Значение по умолчанию:`истинный` .
