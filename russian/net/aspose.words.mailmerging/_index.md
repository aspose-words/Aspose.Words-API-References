**Aspose.Words.MailMerging** namespace содержит классы «исходной» системы создания отчетов о слиянии почты.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [FieldMergingArgs](./fieldmergingargs/) | Предоставляет данные для **MergeField** событие. |
| [FieldMergingArgsBase](./fieldmergingargsbase/) | Базовый класс для[`FieldMergingArgs`](../aspose.words.mailmerging/fieldmergingargs/) а также[`ImageFieldMergingArgs`](../aspose.words.mailmerging/imagefieldmergingargs/) . |
| [ImageFieldMergingArgs](./imagefieldmergingargs/) | Предоставляет данные для[`ImageFieldMerging`](../aspose.words.mailmerging/ifieldmergingcallback/imagefieldmerging/) событие. |
| [MailMerge](./mailmerge/) | Представляет функцию слияния почты. |
| [MailMergeRegionInfo](./mailmergeregioninfo/) | Содержит информацию о регионе слияния. |
| [MappedDataFieldCollection](./mappeddatafieldcollection/) | Позволяет автоматически сопоставлять имена полей в источнике данных и имена полей слияния в документе. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IFieldMergingCallback](./ifieldmergingcallback/) | Реализуйте этот интерфейс, если вы хотите управлять тем, как данные вставляются в поля слияния во время операции слияния. |
| [IMailMergeCallback](./imailmergecallback/) | Реализуйте этот интерфейс, если хотите получать уведомления во время слияния почты. |
| [IMailMergeDataSource](./imailmergedatasource/) | Реализуйте этот интерфейс, чтобы разрешить слияние почты из пользовательского источника данных, например списка объектов. Данные Master-Detail также поддерживаются. |
| [IMailMergeDataSourceRoot](./imailmergedatasourceroot/) | Реализуйте этот интерфейс, чтобы разрешить слияние почты из пользовательского источника данных с основными и подробными данными. |
