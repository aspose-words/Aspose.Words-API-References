## ComparerContext.AcceptRevisions property

Указывает, следует ли принимать изменения в документах перед их сравнением. Если сравниваемые документы содержат изменения и этот флаг установлен в значение false, процессор отклонит изменения. Значение по умолчанию:`истинный` .

```csharp
public bool AcceptRevisions { get; set; }
```
