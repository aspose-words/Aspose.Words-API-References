## Range.NormalizeFieldTypes method

Изменяет значения типов полей.[`FieldType`](../../../aspose.words.fields/fieldchar/fieldtype/) из[`FieldStart`](../../../aspose.words.fields/fieldstart/) ,[`FieldSeparator`](../../../aspose.words.fields/fieldseparator/) ,[`FieldEnd`](../../../aspose.words.fields/fieldend/) в этом диапазоне, чтобы они соответствовали типам полей, содержащимся в кодах полей.

```csharp
public void NormalizeFieldTypes()
```

## Примечания

Используйте этот метод после изменений документа, которые влияют на типы полей.

Чтобы изменить значения типов полей во всем документе, используйте[`NormalizeFieldTypes`](../../document/normalizefieldtypes/).
