## Document.UnlinkFields method

Разъединяет поля во всем документе.

```csharp
public void UnlinkFields()
```

### Примечания

Заменяет все поля во всем документе самыми последними результатами.

Чтобы разъединить поля в определенной части документа, используйте[`UnlinkFields`](../../range/unlinkfields).

### Примеры

Показывает, как отменить связь всех полей в документе.

```csharp
Document doc = new Document(MyDir + "Linked fields.docx");

doc.UnlinkFields();
```
