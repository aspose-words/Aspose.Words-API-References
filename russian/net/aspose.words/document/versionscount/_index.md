## Document.VersionsCount property

Получает количество версий документа, которые были сохранены в документе DOC.

```csharp
public int VersionsCount { get; }
```

### Примечания

Версии в Microsoft Word доступны через меню File/Versions. Microsoft Word поддерживает версии только для файлов DOC.

Это свойство позволяет определить, были ли в этом документе сохранены версии до того, как он был открыт в Aspose.Words. Aspose.Words не предоставляет никакой другой поддержки версий документа. Если вы сохраните этот документ с помощью Aspose.Words, документ будет сохранен без версий.

### Примеры

Показывает, как работать с функцией подсчета версий старых документов Microsoft Word.

```csharp
Document doc = new Document(MyDir + "Versions.doc");

// Мы можем прочитать это свойство документа, но не можем сохранить его при сохранении.
Assert.AreEqual(4, doc.VersionsCount);

doc.Save(ArtifactsDir + "Document.VersionsCount.doc");      
doc = new Document(ArtifactsDir + "Document.VersionsCount.doc");

Assert.AreEqual(0, doc.VersionsCount);
```
