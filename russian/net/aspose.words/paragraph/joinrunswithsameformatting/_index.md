## Paragraph.JoinRunsWithSameFormatting method

Соединения выполняются с одинаковым форматированием в абзаце.

```csharp
public int JoinRunsWithSameFormatting()
```

### Возвращаемое значение

Количество выполненных объединений. Когда соединяются **N** смежных трасс, они считаются как **N - 1** .

### Примеры

Показывает, как упростить абзацы путем объединения лишних строк.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

 // Вставляем четыре фрагмента текста в абзац.
builder.Write("Run 1. ");
builder.Write("Run 2. ");
builder.Write("Run 3. ");
builder.Write("Run 4. ");

 // Если мы откроем этот документ в Microsoft Word, абзац будет выглядеть как одно сплошное текстовое тело.
 // Однако он будет состоять из четырех отдельных прогонов с одинаковым форматированием. Фрагментированные абзацы, подобные this
 // может произойти, когда мы много раз вручную редактируем части одного абзаца в Microsoft Word.
Paragraph para = builder.CurrentParagraph;

Assert.AreEqual(4, para.Runs.Count);

 // Изменяем стиль последнего запуска, чтобы он отличался от первых трех.
para.Runs[3].Font.StyleIdentifier = StyleIdentifier.Emphasis;

 // Мы можем запустить метод "JoinRunsWithSameFormatting" для оптимизации содержимого документа
// объединяя похожие прогоны в один, уменьшая их общее количество.
 // Этот метод также возвращает количество прогонов, объединенных этим методом.
 // Эти два слияния произошли для объединения запусков №1, №2 и №3, 
 // при этом пропускаем Run #4, потому что он имеет несовместимый стиль.
Assert.AreEqual(2, para.JoinRunsWithSameFormatting());

 // Количество оставшихся прогонов будет равно исходному count
 // минус количество прогонов слияний, выполненных методом "JoinRunsWithSameFormatting".
Assert.AreEqual(2, para.Runs.Count);
Assert.AreEqual("Run 1. Run 2. Run 3. ", para.Runs[0].Text);
Assert.AreEqual("Run 4. ", para.Runs[1].Text);
```
