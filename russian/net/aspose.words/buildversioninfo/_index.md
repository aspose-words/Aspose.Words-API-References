## BuildVersionInfo class

Предоставляет информацию о текущем названии продукта и версии.

Чтобы узнать больше, посетите[Имя генератора или производителя включено в выходные документы](https://docs.aspose.com/words/net/generator-or-producer-name-included-in-output-documents/) документальная статья.

```csharp
public static class BuildVersionInfo
```

## Характеристики

| Имя | Описание |
| --- | --- |
| static [Product](../../aspose.words/buildversioninfo/product/) { get; } | Получает полное название продукта. |
| static [Version](../../aspose.words/buildversioninfo/version/) { get; } | Получает версию продукта. |

## Примеры

Показывает, как отобразить информацию об установленной версии Aspose.Words.

```csharp
Console.WriteLine($"I am currently using {BuildVersionInfo.Product}, version number {BuildVersionInfo.Version}!");
```
