**Aspose.Words.Shaping** Пространство имен содержит классы для обеспечения поддержки функций OpenType с использованием внешних механизмов формирования текста.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [BasicTextShaperCache](./basictextshapercache/) | Реализует базовый кэш для[`ITextShaper`](../aspose.words.shaping/itextshaper/) экземпляры. Этот класс потокобезопасен. |
| [Cluster](./cluster/) | Инкапсулирует кодовые точки и глифы, составляющие графему. |
| [Glyph](./glyph/) | Представляет глиф |
| [VariationAxisCoordinate](./variationaxiscoordinate/) | Представляет координату оси. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [ITextShaper](./itextshaper/) | Предоставляет методы для формирования текста. |
| [ITextShaperFactory](./itextshaperfactory/) | Интерфейс фабрики для строительства[`ITextShaper`](../aspose.words.shaping/itextshaper/) реализации. |
