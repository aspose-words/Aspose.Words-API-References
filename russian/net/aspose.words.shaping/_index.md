**Aspose.Words.Shaping** Пространство имен содержит классы для обеспечения поддержки функций OpenType с использованием внешних механизмов формирования текста.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [BasicTextShaperCache](./basictextshapercache/) | Реализует базовый кеш для[`ITextShaper`](../aspose.words.shaping/itextshaper/) экземпляры. Этот класс является потокобезопасным. |
| [Cluster](./cluster/) | Инкапсулирует кодовые точки и глифы, составляющие графему. |
| [Glyph](./glyph/) | Представляет собой глиф . |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [ITextShaper](./itextshaper/) | Предоставляет методы для формирования текста. |
| [ITextShaperFactory](./itextshaperfactory/) | Интерфейс фабрики для построения[`ITextShaper`](../aspose.words.shaping/itextshaper/) реализации. |
