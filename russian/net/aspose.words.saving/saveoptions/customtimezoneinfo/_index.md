## SaveOptions.CustomTimeZoneInfo property

Получает или задает пользовательский местный часовой пояс, используемый для полей даты/времени.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

### Примечания

Эта опция доступна либо в .Net framework, начиная с версии 3.5, либо в .Net Standard.

По умолчанию Aspose.Words использует локальный системный часовой пояс при записи полей даты/времени, эта опция позволяет установить пользовательское значение.
