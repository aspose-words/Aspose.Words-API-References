## SaveOptions.CustomTimeZoneInfo property

Получает или задает пользовательский часовой пояс, используемый для полей даты и времени.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

## Примечания

Эта опция доступна как в .Net Framework, начиная с версии 3.5, так и в .Net Standard.

По умолчанию Aspose.Words использует локальный часовой пояс системы при записи полей даты/времени, эта опция позволяет установить собственное значение.
