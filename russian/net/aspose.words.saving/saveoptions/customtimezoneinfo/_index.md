## SaveOptions.CustomTimeZoneInfo property

Возвращает или задает пользовательский локальный часовой пояс, используемый для полей даты/времени.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

## Примечания

Эта опция доступна либо в .Net Framework, начиная с версии 3.5, либо в .Net Standard.

По умолчанию Aspose.Words использует локальный часовой пояс системы при записи полей даты/времени; эта опция позволяет задать пользовательское значение.
