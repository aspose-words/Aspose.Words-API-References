## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [ICssSavingCallback](./icsssavingcallback/) | Реализуйте этот интерфейс, если хотите контролировать, как Aspose.Words сохраняет CSS (каскадную таблицу стилей) при сохранении документа в HTML. |
| [IDocumentPartSavingCallback](./idocumentpartsavingcallback/) | Реализуйте этот интерфейс, если хотите получать уведомления и контролировать, как Aspose.Words сохраняет части документа при экспорте документа вHtml илиEpub формат. |
| [IDocumentSavingCallback](./idocumentsavingcallback/) | Реализуйте этот интерфейс, если вы хотите, чтобы во время сохранения документа вызывался собственный метод. |
| [IFontSavingCallback](./ifontsavingcallback/) | Реализуйте этот интерфейс, если хотите получать уведомления и контролировать, как Aspose.Words сохраняет шрифты при экспорте документа в формат HTML. |
| [IImageSavingCallback](./iimagesavingcallback/) | Реализуйте этот интерфейс, если хотите контролировать, как Aspose.Words сохраняет изображения при сохранении документа в HTML. Может использоваться другими форматами. |
| [IPageSavingCallback](./ipagesavingcallback/) | Реализуйте этот интерфейс, если хотите контролировать, как Aspose.Words сохраняет отдельные страницы при сохранении документа в фиксированных форматах страниц. |
| [IResourceSavingCallback](./iresourcesavingcallback/) | Реализуйте этот интерфейс, если хотите контролировать, как Aspose.Words сохраняет внешние ресурсы (изображения, шрифты и CSS) при сохранении документа в HTML или SVG с фиксированной страницей. |
