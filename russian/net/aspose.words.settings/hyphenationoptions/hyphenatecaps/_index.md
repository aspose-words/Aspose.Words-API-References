## HyphenationOptions.HyphenateCaps property

Получает или задает значение, определяющее, переносятся ли слова, написанные заглавными буквами. Значение по умолчанию для этого свойства: **истинный** .

```csharp
public bool HyphenateCaps { get; set; }
```
