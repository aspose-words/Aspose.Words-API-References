## ZoomType enumeration

Возможные значения того, насколько большим или маленьким документ будет отображаться на экране в Microsoft Word.

```csharp
public enum ZoomType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Custom | `0` | Процент масштабирования задается явно. Он не пересчитывается автоматически при изменении размера элемента управления. |
| None | `0` | Указывает на использование явного процента масштабирования. Такой же какCustom . |
| FullPage | `1` | Процент масштабирования автоматически пересчитывается для размещения одной полной страницы. |
| PageWidth | `2` | Процент масштабирования автоматически пересчитывается в соответствии с шириной страницы. |
| TextFit | `3` | Процент масштабирования автоматически пересчитывается в соответствии с размером текста. |
