## CompatibilityOptions.UICompat97To2003 property

**True** для отключения функций пользовательского интерфейса, несовместимых с Word97-2003. Значение по умолчанию: **false** .

```csharp
public bool UICompat97To2003 { get; set; }
```

### Примечания

Управляет настройкой совместимости Word97-2003, которая отключает функциональность пользовательского интерфейса, которая не совместим с Word97-2003. Когда **true** , XML-элемент 'w:uiCompat97To2003' записывается в '\word\settings.xml' документ часть пакета. Значение по умолчанию: **false** . При значении **false** этот элемент не записывается. Технически это свойство не является частью параметров совместимости, но мы разместили его здесь для удобства API.
