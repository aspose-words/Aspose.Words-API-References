## CompatibilityOptions.UICompat97To2003 property

**Истинный** для отключения функций пользовательского интерфейса, несовместимых с Word97-2003. Значение по умолчанию: **ЛОЖЬ** .

```csharp
public bool UICompat97To2003 { get; set; }
```

### Примечания

Управляет параметром совместимости Word97-2003, который отключает функции пользовательского интерфейса, которые несовместимы с Word97-2003. Когда **истинный** , 'w:uiCompat97To2003' XML-элемент записывается в '\word\settings.xml' часть пакета документа. Значение по умолчанию: **ЛОЖЬ** . При установке на **ЛОЖЬ** , этот элемент не прописан. Технически это свойство не является частью параметров совместимости, но мы разместили его здесь для удобства API.
