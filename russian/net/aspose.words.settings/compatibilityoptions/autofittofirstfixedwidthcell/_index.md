## CompatibilityOptions.AutofitToFirstFixedWidthCell property

Разрешить столбцам таблицы превышать предпочтительную ширину составляющих ячеек.

```csharp
public bool AutofitToFirstFixedWidthCell { get; set; }
```

### Примечания

В пользовательском интерфейсе MS Word 2013 этот параметр называется «Использовать правила автоподбора таблиц Word 2003». Фактически он влияет на то, как рассчитывается сетка и для таблиц с фиксированным макетом (в некоторых случаях).
