The**Aspose.Words.Lists** Пространство имен содержит классы для working с маркированными и нумерованными списками, определенными в документе.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [List](./list/) | Представляет форматирование списка. |
| [ListCollection](./listcollection/) | Сохраняет и управляет форматированием маркированных и нумерованных списков, используемых в документе. |
| [ListFormat](./listformat/) | Позволяет контролировать, какое форматирование списка применяется к абзацу. |
| [ListLabel](./listlabel/) | Определяет свойства, специфичные для метки списка. |
| [ListLevel](./listlevel/) | Определяет форматирование для уровня списка. |
| [ListLevelCollection](./listlevelcollection/) | Коллекция форматирования списка для каждого уровня в списке. |
## перечисление

| перечисление | Описание |
| --- | --- |
| [ListLevelAlignment](./listlevelalignment/) | Задает выравнивание для номера списка или маркера. |
| [ListTemplate](./listtemplate/) | Указывает один из предопределенных форматов списка, доступных в Microsoft Word. |
| [ListTrailingCharacter](./listtrailingcharacter/) | Указывает символ, который отделяет метку списка от текста абзаца. |
