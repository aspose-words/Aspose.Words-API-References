**Aspose.Words.Replaceing** пространство имен предоставляет классы для манипулирования с помощью операций поиска/замены в дереве документа.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [FindReplaceOptions](./findreplaceoptions/) | Определяет параметры для операций поиска/замены. |
| [ReplacingArgs](./replacingargs/) | Предоставляет данные для пользовательской операции замены. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IReplacingCallback](./ireplacingcallback/) | Реализуйте этот интерфейс, если вы хотите, чтобы во время операции поиска и замены вызывался собственный метод. |
## перечисление

| перечисление | Описание |
| --- | --- |
| [FindReplaceDirection](./findreplacedirection/) | Указывает направление операций замены. |
| [ReplaceAction](./replaceaction/) | Позволяет пользователю указать, что происходит с текущим совпадением во время операции замены. |
