The**Aspose.Words.Библиография**Пространство имен предоставляет классы, которые позволяют получить доступ к списку библиографических источников, доступных в документе.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Bibliography](./bibliography/) | Представляет список источников библиографии, доступных в документе. |
| [Contributor](./contributor/) | Представляет автора источника библиографии. Может быть как корпорацией (организацией), так и списком лиц. |
| [ContributorCollection](./contributorcollection/) | Представляет авторов источника библиографии. |
| [Corporate](./corporate/) | Представляет корпоративного (организационного) автора библиографического источника. |
| [Person](./person/) | Представляет индивидуального (человека) автора источника библиографии. |
| [PersonCollection](./personcollection/) | Представляет список лиц, которые являются авторами библиографических источников. |
| [Source](./source/) | Представляет отдельный источник, например книгу, журнальную статью или интервью. |
