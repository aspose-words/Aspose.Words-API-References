**Aspose.Words.Themes** namespace предоставляет классы, которые позволяют получить доступ к информации о теме в документах Microsoft Word.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Theme](./theme/) | Представляет тему документа и обеспечивает доступ к основным частям темы, включая[`MajorFonts`](../aspose.words.themes/theme/majorfonts/) ,[`MinorFonts`](../aspose.words.themes/theme/minorfonts/) а также[`Colors`](../aspose.words.themes/theme/colors/) |
| [ThemeColors](./themecolors/) | Представляет цветовую схему темы документа, которая содержит двенадцать цветов. |
| [ThemeFonts](./themefonts/) | Представляет набор шрифтов в схеме шрифтов, позволяя указывать разные шрифты для разных языков.[`Latin`](../aspose.words.themes/themefonts/latin/) ,[`EastAsian`](../aspose.words.themes/themefonts/eastasian/) а также[`ComplexScript`](../aspose.words.themes/themefonts/complexscript/) . |
## перечисление

| перечисление | Описание |
| --- | --- |
| [ThemeColor](./themecolor/) | Определяет цвета темы для тем документа. |
| [ThemeFont](./themefont/) | Определяет типы названий шрифтов тем для тем документов. |
