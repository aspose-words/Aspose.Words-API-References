The**Aspose.Words.Themes** Пространство имен предоставляет классы, позволяющие получить доступ к информации о теме в документах Microsoft Word.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Theme](./theme/) | Представляет тему документа и обеспечивает доступ к основным частям темы, включая[`MajorFonts`](../aspose.words.themes/theme/majorfonts/) ,[`MinorFonts`](../aspose.words.themes/theme/minorfonts/) и[`Colors`](../aspose.words.themes/theme/colors/) |
| [ThemeColors](./themecolors/) | Представляет цветовую схему темы документа, содержащую двенадцать цветов. |
| [ThemeFonts](./themefonts/) | Представляет коллекцию шрифтов в схеме шрифтов, позволяющую указывать разные шрифты для разных языков.[`Latin`](../aspose.words.themes/themefonts/latin/) ,[`EastAsian`](../aspose.words.themes/themefonts/eastasian/) и[`ComplexScript`](../aspose.words.themes/themefonts/complexscript/) . |
## перечисление

| перечисление | Описание |
| --- | --- |
| [ThemeColor](./themecolor/) | Задает цвета темы для тем документа. |
| [ThemeFont](./themefont/) | Указывает типы названий шрифтов темы для тем документа. |
