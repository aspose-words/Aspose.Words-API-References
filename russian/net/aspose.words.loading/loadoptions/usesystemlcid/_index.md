## LoadOptions.UseSystemLcid property

Возвращает или задает, следует ли использовать значение LCID, полученное из реестра Windows, для определения полей страницы по умолчанию.

```csharp
public bool UseSystemLcid { get; set; }
```

## Примечания

Если установлено значение`истинный`, то эмулируется поведение MS Word, который берет значение LCID из реестра Windows.

Значение по умолчанию:`ЛОЖЬ`.
