## LoadOptions.FlatOpcXmlMappingOnly property

Получает или задает значение, определяющее, какие форматы документов разрешено отображать с помощью[`XmlMapping`](../../../aspose.words.markup/structureddocumenttag/xmlmapping). По умолчанию только формат документаFlatOpcразрешен для сопоставления.

```csharp
public bool FlatOpcXmlMappingOnly { get; set; }
```

### Примечания

Этот параметр работает в паре с[`FlatOpcXmlMappingOnly`](../../../aspose.words.saving/saveoptions/flatopcxmlmappingonly). Обычно вам нужно установить оба значения в false, чтобы разрешить сопоставление произвольного формата документа.
