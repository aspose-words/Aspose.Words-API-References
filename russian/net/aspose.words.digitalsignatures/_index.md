The**Aspose.Words.DigitalSignatures** Пространство имен предоставляет классы для подписи документов и проверки подписей.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [CertificateHolder](./certificateholder/) | Представляет держателя**X509Сертификат2** экземпляр. |
| [DigitalSignature](./digitalsignature/) | Представляет цифровую подпись на документе и результат ее проверки. |
| [DigitalSignatureCollection](./digitalsignaturecollection/) | Предоставляет коллекцию цифровых подписей, прикрепленных к документу, доступную только для чтения. |
| [DigitalSignatureUtil](./digitalsignatureutil/) | Предоставляет методы для подписания документа. |
| [SignOptions](./signoptions/) | Позволяет указать параметры подписания документа. |
## перечисление
