## LayoutEnumerator indexer

Получает именованное свойство сущности.

```csharp
public object this[string key] { get; }
```

| Параметр | Описание |
| --- | --- |
| key | Имя свойства (с учетом регистра). |

### Возвращаемое значение

Null, если свойство недоступно, в противном случае — значение свойства.

## Примечания

В настоящее время это используется для получения свойств шрифта spans. См.[`Font`](../../../aspose.words/font/) класс для возможных имен свойств. Не все свойства поддерживаются.
