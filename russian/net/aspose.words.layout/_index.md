**Aspose.Words.Layout** Пространство имен предоставляет классы, которые позволяют получить доступ к информации , например, на какой странице и где на странице расположены определенные элементы документа , когда документ форматируется в страницы.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [LayoutCollector](./layoutcollector/) | Этот класс позволяет вычислять номера страниц узлов документа. |
| [LayoutEnumerator](./layoutenumerator/) | Перечисляет объекты макета страницы документа. Этот класс можно использовать для обхода модели макета страницы. Доступными свойствами являются тип, геометрия, текст и индекс страницы, на которой отображается объект, , а также общая структура и связи. Используйте комбинацию[`GetEntity`](../aspose.words.layout/layoutcollector/getentity/) и[`Current`](../aspose.words.layout/layoutenumerator/current/) перейти к объекту, который соответствует узлу документа. |
| [LayoutOptions](./layoutoptions/) | Содержит параметры, позволяющие управлять процессом макетирования документа. |
| [PageLayoutCallbackArgs](./pagelayoutcallbackargs/) | Аргумент, переданный в[`Notify`](../aspose.words.layout/ipagelayoutcallback/notify/) |
| [RevisionOptions](./revisionoptions/) | Позволяет контролировать обработку изменений документа в процессе макетирования. |
## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IPageLayoutCallback](./ipagelayoutcallback/) | Реализуйте этот интерфейс, если вы хотите, чтобы во время построения и рендеринга модели макета страницы вызывался собственный метод. |
