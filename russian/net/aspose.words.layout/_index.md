The**Aspose.Words.Layout** Пространство имен предоставляет классы, которые позволяют получить доступ к информации , например, на какой странице и где на странице расположены определенные элементы документа , когда документ форматируется на страницы.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [LayoutCollector](./layoutcollector/) | Этот класс позволяет вычислять номера страниц узлов документа. |
| [LayoutEnumerator](./layoutenumerator/) | Перечисляет сущности макета страницы документа. Вы можете использовать этот класс для обхода модели макета страницы. Доступные свойства: тип, геометрия, текст и индекс страницы, где отображается сущность, а также общая структура и отношения. Используйте комбинацию[`GetEntity`](../aspose.words.layout/layoutcollector/getentity/) и[`Current`](../aspose.words.layout/layoutenumerator/current/) перейти к сущности, которая соответствует узлу документа. |
| [LayoutOptions](./layoutoptions/) | Содержит параметры, позволяющие управлять процессом макетирования документа. |
| [PageLayoutCallbackArgs](./pagelayoutcallbackargs/) | Аргумент передан в[`Notify`](../aspose.words.layout/ipagelayoutcallback/notify/) |
| [RevisionOptions](./revisionoptions/) | Позволяет контролировать обработку изменений документа в процессе макетирования. |
## Интерфейсы

| Интерфейс | Описание |
