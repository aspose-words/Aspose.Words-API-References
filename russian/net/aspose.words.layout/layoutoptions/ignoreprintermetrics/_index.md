## LayoutOptions.IgnorePrinterMetrics property

Получает или задает указание на то, игнорируется ли параметр совместимости "Использовать метрики принтера для компоновки документа". Значение по умолчанию — True.

```csharp
public bool IgnorePrinterMetrics { get; set; }
```

### Примеры

Показывает, как игнорировать параметр «Использовать метрики принтера для компоновки документа».

```csharp
Document doc = new Document(MyDir + "Rendering.docx");

doc.LayoutOptions.IgnorePrinterMetrics = false;

doc.Save(ArtifactsDir + "Document.IgnorePrinterMetrics.docx");
```
