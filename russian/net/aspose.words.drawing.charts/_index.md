**Aspose.Words.Drawing.Charts** namespace предоставляет классы, позволяющие получать доступ к диаграммам в документах Microsoft Word.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [AxisBound](./axisbound/) | Представляет минимальную или максимальную границу значений оси. |
| [AxisDisplayUnit](./axisdisplayunit/) | Предоставляет доступ к параметрам масштабирования единиц отображения для оси значений. |
| [AxisScaling](./axisscaling/) | Представляет параметры масштабирования оси. |
| [Chart](./chart/) | Предоставляет доступ к свойствам формы диаграммы. |
| [ChartAxis](./chartaxis/) | Представляет параметры оси диаграммы. |
| [ChartDataLabel](./chartdatalabel/) | Представляет метку данных в точке диаграммы или на линии тренда. |
| [ChartDataLabelCollection](./chartdatalabelcollection/) | Представляет набор[`ChartDataLabel`](../aspose.words.drawing.charts/chartdatalabel/) . |
| [ChartDataPoint](./chartdatapoint/) | Позволяет задать форматирование одной точки данных на диаграмме. |
| [ChartDataPointCollection](./chartdatapointcollection/) | Представляет набор[`ChartDataPoint`](../aspose.words.drawing.charts/chartdatapoint/) . |
| [ChartFormat](./chartformat/) | Представляет форматирование элемента диаграммы. |
| [ChartLegend](./chartlegend/) | Представляет свойства легенды диаграммы. |
| [ChartLegendEntry](./chartlegendentry/) | Представляет запись легенды диаграммы. |
| [ChartLegendEntryCollection](./chartlegendentrycollection/) | Представляет набор записей легенды диаграммы. |
| [ChartMarker](./chartmarker/) | Представляет маркер данных диаграммы. |
| [ChartNumberFormat](./chartnumberformat/) | Представляет числовое форматирование родительского элемента. |
| [ChartSeries](./chartseries/) | Представляет свойства ряда диаграммы. |
| [ChartSeriesCollection](./chartseriescollection/) | Представляет набор[`ChartSeries`](../aspose.words.drawing.charts/chartseries/) . |
| [ChartTitle](./charttitle/) | Предоставляет доступ к свойствам заголовка диаграммы. |
