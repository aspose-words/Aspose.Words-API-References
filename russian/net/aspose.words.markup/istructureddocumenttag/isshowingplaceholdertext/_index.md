## IStructuredDocumentTag.IsShowingPlaceholderText property

Указывает, должно ли содержимое этого **SDT** интерпретироваться как содержать замещающий текст (в отличие от обычного текстового содержимого в SDT).

если установлено значение true, это состояние должно быть возобновлено (с текстом-заполнителем) при открытии этого документа.

```csharp
public bool IsShowingPlaceholderText { get; set; }
```
