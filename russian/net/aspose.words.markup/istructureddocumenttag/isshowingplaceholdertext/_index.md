## IStructuredDocumentTag.IsShowingPlaceholderText property

Указывает, будет ли содержимое этого**СДТ** должно интерпретироваться как содержащее заполнитель text (в отличие от обычного текстового содержимого в SDT).

, если установлено значение true, это состояние будет возобновлено (с отображением текста-заполнителя) при открытии этого документа.

```csharp
public bool IsShowingPlaceholderText { get; set; }
```
