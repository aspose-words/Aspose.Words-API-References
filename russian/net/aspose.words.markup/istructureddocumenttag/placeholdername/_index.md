## IStructuredDocumentTag.PlaceholderName property

Получает или задает Имя[`BuildingBlock`](../../../aspose.words.buildingblocks/buildingblock/) содержащий текст-заполнитель.

Строительный блок с таким именем[`Name`](../../../aspose.words.buildingblocks/buildingblock/name/) должен присутствовать в[`GlossaryDocument`](../../../aspose.words/document/glossarydocument/) иначеInvalidOperationException произойдет.

public string PlaceholderName { get; set; }
