## IStructuredDocumentTag.Placeholder property

Получает[`BuildingBlock`](../../../aspose.words.buildingblocks/buildingblock/)содержащий текст-заполнитель, который должен отображаться, когда содержимое этого запуска SDT пусто, связанный сопоставленный XML-элемент пуст, как указано через[`XmlMapping`](../xmlmapping/) element или[`IsShowingPlaceholderText`](../isshowingplaceholdertext/) элемент верен.

```csharp
public BuildingBlock Placeholder { get; }
```

### Примечания

Может иметь значение null, что означает, что заполнитель неприменим для этого Sdt.
