## SdtCalendarType enumeration

Указывает возможные типы календарей, которые можно использовать для указания[`CalendarType`](../structureddocumenttag/calendartype/) в документе Office Open XML.

```csharp
public enum SdtCalendarType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Default | `0` | Используется как значение по умолчанию в OOXML. РавноGregorian . |
| Gregorian | `0` | Указывает, что должен использоваться григорианский календарь, определенный в ISO 8601. Этот календарь должен быть переведен на соответствующий язык. |
| GregorianArabic | `1` | Указывает, что должен использоваться григорианский календарь, определенный в ISO 8601. Значения для этого календаря должны быть представлены на арабском языке. |
| GregorianMeFrench | `2` | Указывает, что должен использоваться григорианский календарь, определенный в ISO 8601. Значения для этого календаря должны быть представлены на ближневосточном французском языке. |
| GregorianUs | `3` | Указывает, что должен использоваться григорианский календарь, определенный в ISO 8601. Значения для этого календаря должны быть представлены на английском языке. |
| GregorianXlitEnglish | `4` | Указывает, что должен использоваться григорианский календарь, как определено в ISO 8601. Значения для этого календаря должны быть представлением английских строк в соответствующих арабских символах (арабская транслитерация английского языка для григорианского календаря). |
| GregorianXlitFrench | `5` | Указывает, что должен использоваться григорианский календарь, как определено в ISO 8601. Значения для этого календаря должны быть представлением французских строк в соответствующих арабских символах (арабская транслитерация французского языка для григорианского календаря). |
| Hebrew | `6` | Указывает, что должен использоваться еврейский лунный календарь, как описано в формуле Гаусса для Песаха [CITATION] и в Полном изложении устного закона (Мишне Тора). |
| Hijri | `7` | Указывает, что должен использоваться лунный календарь Хиджры, как описано Королевством Саудовская Аравия, Министерством по делам ислама, пожертвований, призывов и руководства. |
| Japan | `8` | Указывает, что должен использоваться календарь эпохи японского императора, как описано в Японском промышленном стандарте JIS X 0301. |
| Korea | `9` | Указывает, что должен использоваться корейский календарь эпохи Тангун, , как описано в Законодательном акте Кореи № 4. |
| None | `10` | Указывает, что календарь не должен использоваться. |
| Saka | `11` | Указывает, что должен использоваться календарь эпохи саков, описанный Комитетом по реформе календаря Индии, как часть Индийских эфемерид и Морского альманаха. |
| Taiwan | `12` | Указывает, что должен использоваться тайваньский календарь, определенный китайским национальным стандартом CNS 7648. |
