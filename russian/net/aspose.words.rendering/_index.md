The**Aspose.Words.Rendering** Пространство имен предоставляет классы, которые позволяют to настраивать печать или рендеринг документов.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [AsposeWordsPrintDocument](./asposewordsprintdocument/) | Предоставляет реализацию по умолчанию для печати[`Document`](../aspose.words/document/)within фреймворк печати .NET. |
| [NodeRendererBase](./noderendererbase/) | Базовый класс для[`ShapeRenderer`](../aspose.words.rendering/shaperenderer/) и[`OfficeMathRenderer`](../aspose.words.rendering/officemathrenderer/) . |
| [OfficeMathRenderer](./officemathrenderer/) | Предоставляет методы для визуализации отдельного[`OfficeMath`](../aspose.words.math/officemath/) в растровое или векторное изображение или в графический объект. |
| [PageInfo](./pageinfo/) | Представляет информацию о конкретной странице документа. |
| [PrinterSettingsContainer](./printersettingscontainer/) | Представляет хранилище для некоторых параметровPrinterSettings объект. |
| [ShapeRenderer](./shaperenderer/) | Предоставляет методы для визуализации отдельного[`Shape`](../aspose.words.drawing/shape/) или[`GroupShape`](../aspose.words.drawing/groupshape/) в растровое или векторное изображение или в графический объект. |
| [ThumbnailGeneratingOptions](./thumbnailgeneratingoptions/) | Может использоваться для указания дополнительных параметров при создании миниатюры документа. |
## перечисление
