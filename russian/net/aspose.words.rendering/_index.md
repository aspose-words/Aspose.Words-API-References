**Aspose.Words.Rendering** пространство имен предоставляет классы, которые позволяют to настраивать печать или отображение документов.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [AsposeWordsPrintDocument](./asposewordsprintdocument/) | Предоставляет реализацию по умолчанию для печати[`Document`](../aspose.words/document/) внутри среды печати .NET. |
| [NodeRendererBase](./noderendererbase/) | Базовый класс для[`ShapeRenderer`](../aspose.words.rendering/shaperenderer/) и[`OfficeMathRenderer`](../aspose.words.rendering/officemathrenderer/) . |
| [OfficeMathRenderer](./officemathrenderer/) | Предоставляет методы для визуализации человека[`OfficeMath`](../aspose.words.math/officemath/) в растровое или векторное изображение или в графический объект. |
| [PageInfo](./pageinfo/) | Представляет информацию об определенной странице документа. |
| [PrinterSettingsContainer](./printersettingscontainer/) | Представляет собой хранилище для некоторых параметровPrinterSettings объект. |
| [ShapeRenderer](./shaperenderer/) | Предоставляет методы для визуализации человека[`Shape`](../aspose.words.drawing/shape/) или[`GroupShape`](../aspose.words.drawing/groupshape/) в растровое или векторное изображение или в графический объект. |
| [ThumbnailGeneratingOptions](./thumbnailgeneratingoptions/) | Может использоваться для указания дополнительных параметров при создании миниатюры документа. |
## перечисление

| перечисление | Описание |
| --- | --- |
| [ColorPrintMode](./colorprintmode/) | Указывает, как печатаются нецветные страницы, если устройство поддерживает цветную печать. |
