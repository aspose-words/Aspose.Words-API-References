## CellFormat.HideMark property

Возвращает или задает видимость метки ячейки.

```csharp
public bool HideMark { get; set; }
```

## Примечания

Указывает, что содержимое ячейки таблицы отображается без высоты, если все ячейки в строке пустые; Однако ячейки имеют видимую высоту, если у них ненулевые границы ячеек, поля ячеек или интервалы между ячейками.
