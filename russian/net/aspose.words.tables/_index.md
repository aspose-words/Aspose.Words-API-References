**Aspose.Words.Tables** пространство имен содержит классы, которые представляют таблицы, строки, ячейки и их форматирование.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Cell](./cell/) | Представляет ячейку таблицы. |
| [CellCollection](./cellcollection/) | Обеспечивает типизированный доступ к коллекции[`Cell`](../aspose.words.tables/cell/) узлы. |
| [CellFormat](./cellformat/) | Представляет все форматирование ячейки таблицы. |
| [PreferredWidth](./preferredwidth/) | Представляет значение и его единицу измерения, которая используется для указания предпочтительной ширины таблицы или ячейки. |
| [Row](./row/) | Представляет строку таблицы. |
| [RowCollection](./rowcollection/) | Обеспечивает типизированный доступ к коллекции[`Row`](../aspose.words.tables/row/) узлы. |
| [RowFormat](./rowformat/) | Представляет все форматирование строки таблицы. |
| [Table](./table/) | Представляет таблицу в документе Word. |
| [TableCollection](./tablecollection/) | Обеспечивает типизированный доступ к коллекции[`Table`](../aspose.words.tables/table/) узлы. |
