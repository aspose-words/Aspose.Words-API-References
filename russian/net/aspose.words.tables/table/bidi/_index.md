## Table.Bidi property

Получает или задает, является ли это таблицей с письмом справа налево.

```csharp
public bool Bidi { get; set; }
```

### Примечания

Когда`истинный`, ячейки в этом ряду располагаются справа налево.

Значение по умолчанию`ЛОЖЬ`.
