## Shape.UpdateSmartArtDrawing method

Обновляет предварительно обработанный рисунок SmartArt с помощью механизма холодного рендеринга Aspose.Words SmartArt.

```csharp
public void UpdateSmartArtDrawing()
```

### Примечания

Microsoft Word создает и сохраняет предварительно визуализированный рисунок вместе с объектом SmartArt. Однако, , если документ сохранен другими приложениями, предварительно визуализированный рисунок SmartArt может отсутствовать или быть неверным. Если доступен предварительно визуализированный рисунок, Aspose.Words использует его для визуализации объекта SmartArt. отсутствует, то Aspose.Words использует собственный механизм холодного рендеринга SmartArt для рендеринга объекта SmartArt . Если предварительно обработанный рисунок неверен, необходимо вызвать этот метод для вызова механизма рендеринга SmartArt cold .
