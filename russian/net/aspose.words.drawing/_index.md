**Aspose.Words.Drawing** пространство имен предоставляет классы, которые позволяют создавать и изменять объекты рисования.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [Fill](./fill/) | Представляет форматирование заливки объекта. |
| [GradientStop](./gradientstop/) | Представляет одну остановку градиента. |
| [GradientStopCollection](./gradientstopcollection/) | Содержит коллекцию[`GradientStop`](../aspose.words.drawing/gradientstop/) объекты. |
| [GroupShape](./groupshape/) | Представляет группу фигур в документе. |
| [HorizontalRuleFormat](./horizontalruleformat/) | Представляет горизонтальное форматирование правил. |
| [ImageData](./imagedata/) | Определяет изображение для фигуры. |
| [ImageSize](./imagesize/) | Содержит информацию о размере и разрешении изображения. |
| [OleFormat](./oleformat/) | Обеспечивает доступ к данным объекта OLE или элемента управления ActiveX. |
| [OlePackage](./olepackage/) | Разрешает доступ к свойствам пакета OLE. |
| [ShadowFormat](./shadowformat/) | Представляет теневое форматирование объекта. |
| [Shape](./shape/) | Представляет объект в слое рисования, например автофигуру, текстовое поле, произвольную форму, объект OLE, элемент управления ActiveX или изображение. |
| [ShapeBase](./shapebase/) | Базовый класс для объектов в слое рисования, таких как автофигура, произвольная форма, объект OLE, элемент управления ActiveX или изображение. |
| [SignatureLine](./signatureline/) | Обеспечивает доступ к свойствам строки подписи. |
| [Stroke](./stroke/) | Определяет обводку фигуры. |
| [TextBox](./textbox/) | Определяет атрибуты, которые определяют, как текст отображается внутри фигуры. |
| [TextPath](./textpath/) | Определяет текст и форматирование текстового пути (объекта WordArt). |
