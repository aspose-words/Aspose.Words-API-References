## IBibliographyStylesProvider.GetStyle method

Возвращает стиль библиографии.

```csharp
public Stream GetStyle(string styleFileName)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| styleFileName | String | Имя файла в стиле библиографии. |

### Возвращаемое значение

Stream с таблицей стилей XSLT библиографического стиля.

### Примечания

Реализация должна вернуть результат`нулевой` чтобы указать, что следует использовать версию MS Word указанного стиля.
