### Примечания

Извлекает имя пользователя, который последним изменил и сохранил текущий документ, как записано в **Последний модифицированный** свойство встроенных свойств документа.

### Примеры

Показывает, как использовать поле LASTSAVEDBY.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// Если мы создаем документ в Microsoft Word, во встроенном свойстве «Последний сохраненный» у него будет имя пользователя.
 // Если мы создаем документ программно, это свойство будет иметь значение null, и нам нужно будет присвоить значение.
doc.BuiltInDocumentProperties.LastSavedBy = "John Doe";

// Мы можем использовать поле LASTSAVEDBY для отображения значения этого свойства в документе.
FieldLastSavedBy field = (FieldLastSavedBy)builder.InsertField(FieldType.FieldLastSavedBy, true);

Assert.AreEqual(" LASTSAVEDBY ", field.GetFieldCode());
Assert.AreEqual("John Doe", field.Result);

doc.Save(ArtifactsDir + "Field.LASTSAVEDBY.docx");
```
