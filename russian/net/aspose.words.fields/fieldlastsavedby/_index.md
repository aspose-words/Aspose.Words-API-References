### Примечания

Получает имя пользователя, который последним изменил и сохранил текущий документ, как записано в **LastModifiedBy** свойство встроенных свойств документа.

### Примеры

Показывает, как использовать поле LASTSAVEDBY.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// Если мы создадим документ в Microsoft Word, он будет иметь имя пользователя во встроенном свойстве «Последнее сохранение».
 // Если мы создадим документ программно, это свойство будет нулевым, и нам нужно будет присвоить значение.
doc.BuiltInDocumentProperties.LastSavedBy = "John Doe";

// Мы можем использовать поле LASTSAVEDBY для отображения значения этого свойства в документе.
FieldLastSavedBy field = (FieldLastSavedBy)builder.InsertField(FieldType.FieldLastSavedBy, true);

Assert.AreEqual(" LASTSAVEDBY ", field.GetFieldCode());
Assert.AreEqual("John Doe", field.Result);

doc.Save(ArtifactsDir + "Field.LASTSAVEDBY.docx");
```
