The**Aspose.Words.Fonts** Пространство имен предоставляет классы и перечисления для доступа к информации о шрифтах , используемых в документе.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [DefaultFontSubstitutionRule](./defaultfontsubstitutionrule/) | Правило замены шрифта по умолчанию. |
| [FileFontSource](./filefontsource/) | Представляет один файл шрифта TrueType, хранящийся в файловой системе. |
| [FolderFontSource](./folderfontsource/) | Представляет папку, содержащую файлы шрифтов TrueType. |
| [FontConfigSubstitutionRule](./fontconfigsubstitutionrule/) | Правило подстановки конфигурации шрифта. |
| [FontEmbeddingLicensingRights](./fontembeddinglicensingrights/) | Представляет собой лицензионные права на внедрение шрифта. |
| [FontFallbackSettings](./fontfallbacksettings/) | Задает настройки механизма резервного копирования шрифтов. |
| [FontInfo](./fontinfo/) | Указывает информацию о шрифте, используемом в документе. |
| [FontInfoCollection](./fontinfocollection/) | Представляет коллекцию шрифтов, используемых в документе. |
| [FontInfoSubstitutionRule](./fontinfosubstitutionrule/) | Правило подстановки информации о шрифте. |
| [FontNameSubstitutionRule](./fontnamesubstitutionrule/) | Правило подстановки шрифта для обработки имени шрифта. |
| [FontSettings](./fontsettings/) | Задает настройки шрифта для документа. |
| [FontSourceBase](./fontsourcebase/) | Это абстрактный базовый класс для классов, которые позволяют пользователю указывать различные источники шрифтов. |
| [FontSubstitutionRule](./fontsubstitutionrule/) | Это абстрактный базовый класс для правила замены шрифта. |
| [FontSubstitutionSettings](./fontsubstitutionsettings/) | Задает настройки механизма подстановки шрифтов. |
| [MemoryFontSource](./memoryfontsource/) | Представляет один файл шрифта TrueType, хранящийся в памяти. |
| [PhysicalFontInfo](./physicalfontinfo/) | Указывает информацию о физическом шрифте, доступном для движка шрифтов Aspose.Words. |
| [StreamFontSource](./streamfontsource/) | Базовый класс для определяемого пользователем источника потокового шрифта. |
| [SystemFontSource](./systemfontsource/) | Представляет все шрифты TrueType, установленные в системе. |
| [TableSubstitutionRule](./tablesubstitutionrule/) | Правило замены шрифта таблицы. |
