**Aspose.Words.WebExtensions**Пространство имен предоставляет классы, которые позволяют to настраивать элементы и атрибуты, расширяющие словарь XML для представления надстроек Office.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [BaseWebExtensionCollection&lt;T&gt;](./basewebextensioncollection-1/) | Базовый класс для[`TaskPaneCollection`](../aspose.words.webextensions/taskpanecollection/) ,[`WebExtensionBindingCollection`](../aspose.words.webextensions/webextensionbindingcollection/) , [`WebExtensionPropertyCollection`](../aspose.words.webextensions/webextensionpropertycollection/) и[`WebExtensionReferenceCollection`](../aspose.words.webextensions/webextensionreferencecollection/) коллекции. |
| [TaskPane](./taskpane/) | Представляет объект области задач надстройки. |
| [TaskPaneCollection](./taskpanecollection/) | Указывает список сохраненных объектов панели задач. |
| [WebExtension](./webextension/) | Представляет объект веб-расширения. |
| [WebExtensionBinding](./webextensionbinding/) | Указывает связь привязки между веб-расширением и данными в документе. |
| [WebExtensionBindingCollection](./webextensionbindingcollection/) | Указывает список привязок веб-расширений. |
| [WebExtensionProperty](./webextensionproperty/) | Указывает пользовательское свойство веб-расширения. |
| [WebExtensionPropertyCollection](./webextensionpropertycollection/) | Указывает набор пользовательских свойств веб-расширения. |
| [WebExtensionReference](./webextensionreference/) | Представляет ссылку на веб-расширение. Ссылка используется для определения местоположения поставщика и версии расширения . |
| [WebExtensionReferenceCollection](./webextensionreferencecollection/) | Указывает список ссылок на веб-расширения. |
