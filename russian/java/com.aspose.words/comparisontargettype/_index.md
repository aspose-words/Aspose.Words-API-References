 Позволяет указать базовый документ, который будет использоваться при сравнении. Значение по умолчанию[CURRENT](../../com.aspose.words/comparisontargettype\#CURRENT). Относится к параметру Microsoft Word «Показать изменения в» в диалоговом окне «Сравнить документы».
## Поля

| Поле | Описание |
| --- | --- |
| [CURRENT](#CURRENT) | Этот документ используется в качестве базы при сравнении. |
| [NEW](#NEW) | В качестве базы при сравнении используется другой документ. |
