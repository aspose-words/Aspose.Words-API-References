java.lang.Object
```
public class ThemeColor
```

Указывает цвета темы для тем документа.

 Чтобы узнать больше, посетите**Working with Styles and Themes** документальная статья.

Указанный цвет темы — это ссылка на один из предопределенных цветов темы, расположенных в разделе «Тема» документа, что позволяет централизованно задавать информацию о цвете в документе.
