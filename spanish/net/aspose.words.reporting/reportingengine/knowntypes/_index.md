## ReportingEngine.KnownTypes property

Obtiene un conjunto desordenado (es decir, una colección de elementos únicos) que contieneType objetos cuyos nombres total o parcialmente calificados se pueden usar dentro de las plantillas de informes procesadas por esta instancia de engine para invocar los miembros estáticos de los tipos correspondientes, realizar conversiones de tipos, etc.

```csharp
public KnownTypeSet KnownTypes { get; }
```
