## JsonDataLoadOptions.AlwaysGenerateRootObject property

Obtiene o establece un indicador que indica si una fuente de datos generada siempre contendrá un objeto para un elemento JSON root . Si un elemento raíz JSON contiene una única propiedad compleja, dicho objeto no se crea de forma predeterminada.

```csharp
public bool AlwaysGenerateRootObject { get; set; }
```

## Observaciones

El valor predeterminado es`FALSO` .
