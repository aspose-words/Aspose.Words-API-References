## MailMerge.RetainFirstSectionStart property

Obtiene o establece un valor que indica si el[`SectionStart`](../../../aspose.words/pagesetup/sectionstart/) de la primera sección del documento y sus copias para las filas de fuente de datos subsiguientes se conservan durante la combinación de correspondencia o se actualizan de acuerdo con el comportamiento de MS Word.

```csharp
public bool RetainFirstSectionStart { get; set; }
```

### Observaciones

El valor predeterminado es **verdadero** .
