## MailMergeOptions.RetainFirstSectionStart property

Obtiene o establece un valor que indica si el inicio de la sección de la primera sección del documento y sus copias para las filas de origen de datos posteriores se conservan durante la combinación de correspondencia o se actualizan según el comportamiento de MS Word.

```csharp
public bool RetainFirstSectionStart { get; set; }
```

## Observaciones

El valor predeterminado es`verdadero` .
