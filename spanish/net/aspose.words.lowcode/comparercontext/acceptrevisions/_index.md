## ComparerContext.AcceptRevisions property

Indica si se deben aceptar revisiones en los documentos antes de compararlos. Si los documentos comparados contienen revisiones y este indicador se establece en falso, el procesador rechazará las revisiones. El valor predeterminado es`verdadero` .

```csharp
public bool AcceptRevisions { get; set; }
```
