## PageSetup.HeadingLevelForChapter property

Obtiene o establece el estilo de nivel de título que se aplica a los títulos de los capítulos del documento.

```csharp
public int HeadingLevelForChapter { get; set; }
```

### Observaciones

Puede ser un número del 0 al 9. 0 significa que no hay número de capítulo si se aplica al número de página.

Antes de que pueda crear números de página que incluyan números de capítulo, los encabezados del documento deben tener aplicado un formato de esquema numerado.
