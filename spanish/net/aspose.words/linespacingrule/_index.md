## LineSpacingRule enumeration

Especifica valores de interlineado para un párrafo.

```csharp
public enum LineSpacingRule
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| AtLeast | `0` | El interlineado puede ser mayor o igual, pero nunca menor, el valor especificado en el[`LineSpacing`](../paragraphformat/linespacing) propiedad. |
| Exactly | `1` | El interlineado nunca cambia del valor especificado en el [`LineSpacing`](../paragraphformat/linespacing) propiedad, incluso si se usa una fuente más grande dentro del párrafo. |
| Multiple | `2` | El interlineado se especifica en el[`LineSpacing`](../paragraphformat/linespacing) propiedad como el número de líneas. Una línea equivale a 12 puntos. |
