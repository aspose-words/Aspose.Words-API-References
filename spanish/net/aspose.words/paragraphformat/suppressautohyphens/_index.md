## ParagraphFormat.SuppressAutoHyphens property

Especifica si el párrafo actual debe estar exento de cualquier separación silábica que se aplica en la configuración del documento.

```csharp
public bool SuppressAutoHyphens { get; set; }
```

### Ejemplos

Muestra cómo suprimir la separación silábica de un párrafo.

```csharp
Hyphenation.RegisterDictionary("de-CH", MyDir + "hyph_de_CH.dic");

Assert.True(Hyphenation.IsDictionaryRegistered("de-CH"));

// Abra un documento que contenga texto con una configuración regional que coincida con la de nuestro diccionario.
// Cuando guardamos este documento en un formato de guardado de página fijo, su texto tendrá guiones.
Document doc = new Document(MyDir + "German text.docx");

// Podemos establecer la propiedad "SuppressAutoHyphens" en "true" para deshabilitar la partición
// para un párrafo específico mientras lo mantiene habilitado para el resto del documento.
// El valor por defecto de esta propiedad es "falso",
// lo que significa que cada párrafo por defecto usa guiones si hay alguno disponible.
doc.FirstSection.Body.FirstParagraph.ParagraphFormat.SuppressAutoHyphens = suppressAutoHyphens;

doc.Save(ArtifactsDir + "ParagraphFormat.SuppressHyphens.pdf");
```
