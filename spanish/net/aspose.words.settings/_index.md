los **Aspose.Words.Configuración** El espacio de nombres contiene clases y enumeraciones que se utilizan al especificar varias configuraciones y opciones almacenadas en un documento. Estas son combinación de correspondencia, protección contra escritura, compatibilidad y otras configuraciones.

## Clases

| Clase | Descripción |
| --- | --- |
| [CompatibilityOptions](./compatibilityoptions/) | Contiene opciones de compatibilidad (es decir, las preferencias de usuario ingresadas en el **Compatibilidad** pestaña de la **Opciones** diálogo en Microsoft Word). |
| [HyphenationOptions](./hyphenationoptions/) | Permite configurar las opciones de partición de documentos. |
| [MailMergeSettings](./mailmergesettings/) | Especifica toda la información de combinación de correspondencia para un documento. |
| [Odso](./odso/) | Especifica la configuración del objeto de origen de datos de Office (ODSO) para un origen de datos de combinación de correspondencia. |
| [OdsoFieldMapData](./odsofieldmapdata/) | Especifica cómo se asignará una columna en la fuente de datos externa a los campos de combinación predefinidos dentro del documento. |
| [OdsoFieldMapDataCollection](./odsofieldmapdatacollection/) | Una colección tipeada de los[`OdsoFieldMapData`](../aspose.words.settings/odsofieldmapdata/) objetos. |
| [OdsoRecipientData](./odsorecipientdata/) | Representa información sobre un solo registro dentro de una fuente de datos externa que se excluirá de la combinación de correspondencia. |
| [OdsoRecipientDataCollection](./odsorecipientdatacollection/) | Una colección escrita de[`OdsoRecipientData`](../aspose.words.settings/odsorecipientdata/) |
| [ViewOptions](./viewoptions/) | Proporciona varias opciones que controlan cómo se muestra un documento en Microsoft Word. |
| [WriteProtection](./writeprotection/) | Especifica la configuración de protección contra escritura para un documento. |
## Enumeración

| Enumeración | Descripción |
| --- | --- |
| [Compatibility](./compatibility/) | Especifica los nombres de las opciones de compatibilidad. |
| [MailMergeCheckErrors](./mailmergecheckerrors/) | Especifica cómo Microsoft Word informará de los errores detectados durante la combinación de correspondencia. |
| [MailMergeDataType](./mailmergedatatype/) | Especifica el tipo de un origen de datos de combinación de correo externo. |
| [MailMergeDestination](./mailmergedestination/) | Especifica los posibles resultados que se pueden generar cuando se realiza una fusión de correspondencia sobre un documento. |
| [MailMergeMainDocumentType](./mailmergemaindocumenttype/) | Especifica los tipos posibles para un documento de origen de combinación de correspondencia. |
| [MsWordVersion](./mswordversion/) | Permite que Aspose.Wods imite el comportamiento de la aplicación específica de la versión de MS Word. |
| [MultiplePagesType](./multiplepagestype/) | Especifica cómo se imprime el documento. |
| [OdsoDataSourceType](./odsodatasourcetype/) | Especifica el tipo de fuente de datos externa a la que se conectará como parte de la información de conexión ODSO. |
| [OdsoFieldMappingType](./odsofieldmappingtype/) | Especifica los posibles tipos usados para indicar si un campo de combinación de correo dado se ha asignado a una columna en la fuente de datos externa dada. |
| [ViewType](./viewtype/) | Valores posibles para el modo de vista en Microsoft Word. |
| [ZoomType](./zoomtype/) | Valores posibles de cuán grande o pequeño aparece el documento en la pantalla en Microsoft Word. |
