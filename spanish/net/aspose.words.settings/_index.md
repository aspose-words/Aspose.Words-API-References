El**Aspose.Palabras.Configuración**El espacio de nombres contiene clases y enumeraciones que se utilizan al especificar varias configuraciones y opciones almacenadas en un documento. Estas son la combinación de correspondencia, la protección contra escritura, la compatibilidad y otras configuraciones.

## Clases

| Clase | Descripción |
| --- | --- |
| [CompatibilityOptions](./compatibilityoptions/) | Contiene opciones de compatibilidad (es decir, las preferencias del usuario ingresadas en el**Compatibilidad** Pestaña de la**Opciones** diálogo en Microsoft Word). |
| [HyphenationOptions](./hyphenationoptions/) | Permite configurar las opciones de separación de palabras del documento. |
| [MailMergeSettings](./mailmergesettings/) | Especifica toda la información de combinación de correspondencia para un documento. |
| [Odso](./odso/) | Especifica la configuración del objeto de origen de datos de Office (ODSO) para un origen de datos de combinación de correspondencia. |
| [OdsoFieldMapData](./odsofieldmapdata/) | Especifica cómo se asignará una columna en la fuente de datos externa a los campos de combinación predefinidos dentro del documento. |
| [OdsoFieldMapDataCollection](./odsofieldmapdatacollection/) | Una colección tipificada de[`OdsoFieldMapData`](../aspose.words.settings/odsofieldmapdata/) objetos. |
| [OdsoRecipientData](./odsorecipientdata/) | Representa información sobre un solo registro dentro de una fuente de datos externa que se excluirá de la combinación de correspondencia. |
| [OdsoRecipientDataCollection](./odsorecipientdatacollection/) | Una colección tipificada de[`OdsoRecipientData`](../aspose.words.settings/odsorecipientdata/) |
| [ViewOptions](./viewoptions/) | Proporciona varias opciones que controlan cómo se muestra un documento en Microsoft Word. |
| [WriteProtection](./writeprotection/) | Especifica la configuración de protección contra escritura para un documento. |
## Enumeración

| Enumeración | Descripción |
| --- | --- |

| [MailMergeCheckErrors](./mailmergecheckerrors/) | Especifica cómo Microsoft Word informará los errores detectados durante la combinación de correspondencia. |
| [MailMergeDataType](./mailmergedatatype/) | Especifica el tipo de una fuente de datos de combinación de correspondencia externa. |
| [MailMergeDestination](./mailmergedestination/) | Especifica los posibles resultados que se pueden generar cuando se realiza una combinación de correspondencia en un documento. |
| [MailMergeMainDocumentType](./mailmergemaindocumenttype/) | Especifica los tipos posibles para un documento fuente de combinación de correspondencia. |
| [MsWordVersion](./mswordversion/) | Permite que Aspose.Wods imite el comportamiento de la aplicación específica de la versión de MS Word. |
| [MultiplePagesType](./multiplepagestype/) | Especifica cómo se imprime el documento. |
| [OdsoDataSourceType](./odsodatasourcetype/) | Especifica el tipo de fuente de datos externa a la que se conectará como parte de la información de conexión de ODSO. |
| [OdsoFieldMappingType](./odsofieldmappingtype/) | Especifica los tipos posibles utilizados para indicar si un campo de combinación de correspondencia determinado se ha asignado a una columna en la fuente de datos externa dada. |
| [ViewType](./viewtype/) | Valores posibles para el modo de visualización en Microsoft Word. |
| [ZoomType](./zoomtype/) | Valores posibles para determinar qué tan grande o pequeño debe ser el documento que aparece en la pantalla en Microsoft Word. |
