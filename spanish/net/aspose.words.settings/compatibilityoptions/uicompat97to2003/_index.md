## CompatibilityOptions.UICompat97To2003 property

**Verdadero** para deshabilitar la funcionalidad de la interfaz de usuario que no es compatible con Word97-2003. El valor predeterminado es **falso** .

```csharp
public bool UICompat97To2003 { get; set; }
```

### Observaciones

Controla la configuración de compatibilidad de Word97-2003 que desactiva la funcionalidad de la interfaz de usuario que no es compatible con Word97-2003. Cuando **verdadero** , el elemento XML 'w:uiCompat97To2003' se escribe en '\word\settings.xml' parte del paquete de documentos. El valor predeterminado es **falso** . cuando se establece en **falso** , este elemento no está escrito. Técnicamente, esta propiedad no forma parte de las opciones de compatibilidad, pero la hemos puesto aquí para la comodidad de la API.
