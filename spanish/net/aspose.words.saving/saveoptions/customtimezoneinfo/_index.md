## SaveOptions.CustomTimeZoneInfo property

Obtiene o establece la zona horaria local personalizada utilizada para los campos de fecha y hora.

```csharp
public TimeZoneInfo CustomTimeZoneInfo { get; set; }
```

## Observaciones

Esta opción está disponible tanto en .Net Framework a partir de la versión 3.5 como en .Net Standard.

De forma predeterminada, Aspose.Words utiliza la zona horaria local del sistema cuando escribe campos de fecha/hora; esta opción permite establecer un valor personalizado.
