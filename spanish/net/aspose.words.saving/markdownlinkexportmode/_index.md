## MarkdownLinkExportMode enumeration

El modo de exportar enlaces a un documento de destino.

```csharp
public enum MarkdownLinkExportMode
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Auto | `0` | Un enlace se exporta como bloque de referencia si tiene información de ida y vuelta o se menciona más de una vez en un documento. En todos los demás casos, un enlace se exporta como un bloque en línea. |
| Inline | `1` | Los enlaces se exportan como bloques en línea. |
| Reference | `2` | Los enlaces se exportan como bloques de referencia. |
