## Observaciones

`FootnoteSeparator` puede contener[`Paragraph`](../../aspose.words/paragraph/) y[`Mesa`](../../aspose.words.tables/table/) nodos secundarios.

Sólo puede haber uno`FootnoteSeparator` de cada uno[`FootnoteSeparatorType`](../footnoteseparatortype/) en un documento.

## Ejemplos

Muestra cómo administrar el formato del separador de notas al pie.

```csharp
Document doc = new Document(MyDir + "Footnotes and endnotes.docx");

FootnoteSeparator footnoteSeparator = doc.FootnoteSeparators[FootnoteSeparatorType.FootnoteSeparator];
// Alinear el separador de notas al pie.
footnoteSeparator.FirstParagraph.ParagraphFormat.Alignment = ParagraphAlignment.Center;
```
