El**Aspose.Palabras.Notas** El espacio de nombres proporciona clases para trabajar con notas finales y notas al pie de Microsoft Word.

## Clases

| Clase | Descripción |
| --- | --- |
| [EndnoteOptions](./endnoteoptions/) | Representa las opciones de numeración de notas finales para un documento o sección. |
| [Footnote](./footnote/) | Representa un contenedor para el texto de una nota al pie o una nota final. |
| [FootnoteOptions](./footnoteoptions/) | Representa las opciones de numeración de notas al pie para un documento o sección. |
| [FootnoteSeparator](./footnoteseparator/) | Representa un contenedor para el separador de notas al pie/notas finales y el contenido de continuación de un documento. |
| [FootnoteSeparatorCollection](./footnoteseparatorcollection/) | Proporciona acceso tipificado a[`FootnoteSeparator`](../aspose.words.notes/footnoteseparator/) nodos de un documento. |

| [FootnotePosition](./footnoteposition/) | Define la posición de la nota al pie. |
| [FootnoteSeparatorType](./footnoteseparatortype/) | Especifica el tipo de separador de notas al pie/notas finales. |
