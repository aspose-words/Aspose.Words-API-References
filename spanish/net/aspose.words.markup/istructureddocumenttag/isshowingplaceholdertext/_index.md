## IStructuredDocumentTag.IsShowingPlaceholderText property

Especifica si el contenido de este **TED** se interpretará como que contiene el marcador de posición text (a diferencia del contenido de texto normal dentro del SDT).

si se establece en verdadero, este estado se reanudará (mostrando texto de marcador de posición) al abrir este documento.

```csharp
public bool IsShowingPlaceholderText { get; set; }
```
