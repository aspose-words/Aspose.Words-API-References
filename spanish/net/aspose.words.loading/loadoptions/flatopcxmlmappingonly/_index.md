## LoadOptions.FlatOpcXmlMappingOnly property

Obtiene o establece el valor que determina qué formatos de documento pueden ser asignados por[`XmlMapping`](../../../aspose.words.markup/structureddocumenttag/xmlmapping) . Solo por defectoFlatOpc se permite mapear el formato del documento.

```csharp
public bool FlatOpcXmlMappingOnly { get; set; }
```

### Observaciones

Esta opción se combina con[`FlatOpcXmlMappingOnly`](../../../aspose.words.saving/saveoptions/flatopcxmlmappingonly) . Por lo general, debe establecer ambos en falso para permitir que se asignen formatos de documentos arbitrarios.
