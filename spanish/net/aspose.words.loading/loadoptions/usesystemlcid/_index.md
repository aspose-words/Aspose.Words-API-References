## LoadOptions.UseSystemLcid property

Obtiene o establece si se debe utilizar el valor LCID obtenido del registro de Windows para determinar los márgenes predeterminados de configuración de página.

```csharp
public bool UseSystemLcid { get; set; }
```

## Observaciones

Si se establece en`verdadero`, luego se emula el comportamiento de MS Word, que toma el valor LCID del registro de Windows.

El valor predeterminado es`FALSO`.
