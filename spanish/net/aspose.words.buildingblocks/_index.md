El**Aspose.Words.BuildingBlocks**El espacio de nombres proporciona clases que permiten acceder a y utilizar Autotexto, entradas de Autocorrección y bloques de creación en un documento.

## Clases

| Clase | Descripción |
| --- | --- |
| [BuildingBlock](./buildingblock/) | Representa una entrada de documento de glosario, como un bloque de creación, un autotexto o una entrada de autocorrección. |
| [BuildingBlockCollection](./buildingblockcollection/) | Una colección de[`BuildingBlock`](../aspose.words.buildingblocks/buildingblock/) objetos en el documento. |
| [GlossaryDocument](./glossarydocument/) | Representa el elemento raíz de un documento de glosario dentro de un documento de Word. Un documento de glosario es un almacenamiento para entradas de Autotexto, Autocorrección y Bloques de creación. |
## Enumeración

| Enumeración | Descripción |
| --- | --- |
| [BuildingBlockBehavior](./buildingblockbehavior/) | Especifica el comportamiento que se aplicará al contenido del bloque de construcción cuando se inserta en el documento principal. |
| [BuildingBlockGallery](./buildingblockgallery/) | Especifica la galería predefinida en la que se clasifica un bloque de construcción. |
| [BuildingBlockType](./buildingblocktype/) | Especifica un tipo de bloque de creación. El tipo puede afectar la visibilidad y el comportamiento del bloque de creación en Microsoft Word. |
