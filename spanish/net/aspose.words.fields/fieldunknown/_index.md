## Ejemplos

Muestra cómo trabajar con el campo 'FieldNone' en un documento.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// Insertar un campo que no indique un tipo de campo objetivo en su código de campo.
Field field = builder.InsertField(" NOTAREALFIELD //a");

// El tipo de campo "FieldNone" está reservado para campos como estos.
Assert.AreEqual(FieldType.FieldNone, field.Type);

// También podemos seguir trabajando con estos campos y asignarlos como instancias de la clase FieldUnknown.
FieldUnknown fieldUnknown = (FieldUnknown)field;
Assert.AreEqual(" NOTAREALFIELD //a", fieldUnknown.GetFieldCode());
```
