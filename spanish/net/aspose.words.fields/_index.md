## Interfaces

| Interfaz | Descripción |
| --- | --- |
| [IBarcodeGenerator](./ibarcodegenerator/) | Interfaz pública para generador personalizado de códigos de barras. La implementación debe ser proporcionada por el usuario. |
| [IBibliographyStylesProvider](./ibibliographystylesprovider/) | Implemente esta interfaz para proporcionar estilo de bibliografía para el[`FieldBibliography`](../aspose.words.fields/fieldbibliography/) y[`FieldCitation`](../aspose.words.fields/fieldcitation/) campos cuando se actualizan. |
| [IComparisonExpressionEvaluator](./icomparisonexpressionevaluator/) | Cuando se implementa, permite anular la evaluación de expresiones de comparación predeterminadas para el[`FieldIf`](../aspose.words.fields/fieldif/) y[`FieldCompare`](../aspose.words.fields/fieldcompare/) campos. |
| [IFieldDatabaseProvider](./ifielddatabaseprovider/) | Implemente esta interfaz para proporcionar datos para el[`FieldDatabase`](../aspose.words.fields/fielddatabase/) campo cuando se actualiza. |
| [IFieldResultFormatter](./ifieldresultformatter/) | Implemente esta interfaz si desea controlar cómo se formatea el resultado del campo. |
| [IFieldUpdateCultureProvider](./ifieldupdatecultureprovider/) | Cuando se implementa, proporciona unaCultureInfo objeto que debe usarse durante la actualización de un campo en particular. |
| [IFieldUpdatingCallback](./ifieldupdatingcallback/) | Implemente esta interfaz si desea que se llamen a sus propios métodos personalizados durante una actualización de campo. |
| [IFieldUpdatingProgressCallback](./ifieldupdatingprogresscallback/) | Implemente esta interfaz si desea realizar un seguimiento del progreso de la actualización del campo. |
| [IFieldUserPromptRespondent](./ifielduserpromptrespondent/) | Representa al encuestado ante las indicaciones del usuario durante la actualización del campo. |
