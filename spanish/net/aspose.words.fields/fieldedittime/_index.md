### Observaciones

Recupera el tiempo total de edición, en minutos, desde que se creó el documento.

### Ejemplos

Muestra cómo utilizar el campo EDITTIME.

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);

// El campo EDITTIME mostrará, en minutos,
// el tiempo pasado con el documento abierto en una ventana de Microsoft Word.
builder.MoveToHeaderFooter(HeaderFooterType.HeaderPrimary);
builder.Write("You've been editing this document for ");
FieldEditTime field = (FieldEditTime)builder.InsertField(FieldType.FieldEditTime, true);
builder.Writeln(" minutes.");

// Esta propiedad de documento incorporada realiza un seguimiento de los minutos. Microsoft Word usa esta propiedad
// para realizar un seguimiento del tiempo pasado con el documento abierto. También podemos editarlo nosotros mismos.
doc.BuiltInDocumentProperties.TotalEditingTime = 10;
field.Update();

Assert.AreEqual(" EDITTIME ", field.GetFieldCode());
Assert.AreEqual("10", field.Result);

// El campo no se actualiza en tiempo real y también tendrá que ser
// actualizado manualmente en Microsoft Word cada vez que necesitamos un valor preciso.
doc.UpdateFields();
doc.Save(ArtifactsDir + "Field.EDITTIME.docx");
```
