## Shape.UpdateSmartArtDrawing method

Actualiza el dibujo pre-renderizado de SmartArt mediante el motor de renderizado en frío de SmartArt de Aspose.Words.

```csharp
public void UpdateSmartArtDrawing()
```

## Observaciones

Microsoft Word genera y guarda el dibujo pre-renderizado junto con el objeto SmartArt. Sin embargo, si el documento se guarda con otras aplicaciones, el dibujo pre-renderizado podría faltar o ser incorrecto. Si el dibujo pre-renderizado está disponible, Aspose.Words lo utiliza para renderizar el objeto SmartArt. Si el dibujo pre-renderizado no está disponible, Aspose.Words utiliza su propio motor de renderizado en frío de SmartArt para renderizar el objeto SmartArt. Si el dibujo pre-renderizado es incorrecto, se requiere llamar a este método para invocar el motor de renderizado en frío de SmartArt.
