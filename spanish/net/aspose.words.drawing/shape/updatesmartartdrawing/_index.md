## Shape.UpdateSmartArtDrawing method

Actualiza el dibujo renderizado previamente de SmartArt mediante el uso del motor de renderizado en frío SmartArt de Aspose.Words.

```csharp
public void UpdateSmartArtDrawing()
```

### Observaciones

Microsoft Word genera y guarda el dibujo renderizado previamente junto con el objeto SmartArt. Sin embargo, si otras aplicaciones guardan el documento, es posible que falte el dibujo SmartArt renderizado previamente o sea incorrecto. Si el dibujo renderizado previamente está disponible, Aspose.Words lo usa para representar el objeto SmartArt. falta, entonces Aspose.Words usa su propio motor de renderizado en frío SmartArt para renderizar el objeto SmartArt. Si el dibujo renderizado previamente es incorrecto, entonces es necesario llamar a este método para invocar el motor de renderizado en frío SmartArt .
