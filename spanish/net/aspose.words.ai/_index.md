El**Aspose.Words.AI** El espacio de nombres permite una integración perfecta con modelos de lenguaje grandes (LLM), como OpenAI y Gemini, y proporciona un soporte sólido para el procesamiento de documentos impulsado por IA.

| Clase | Descripción |
| --- | --- |
| [AiModel](./aimodel/) | Representa información sobre un modelo de lenguaje generativo. |
| [AnthropicAiModel](./anthropicaimodel/) | Una clase abstracta que representa la integración con los modelos de IA de Anthropic dentro de Aspose.Words. |
| [CheckGrammarOptions](./checkgrammaroptions/) | Permite especificar varias opciones mientras se verifica la gramática de un documento usando IA. |
| [GoogleAiModel](./googleaimodel/) | Una clase abstracta que representa la integración con los modelos de IA de Google dentro de Aspose.Words. |
| [OpenAiModel](./openaimodel/) | Una clase abstracta que representa la integración con los grandes modelos de lenguaje de OpenAI dentro de Aspose.Words. |
| [SummarizeOptions](./summarizeoptions/) | Permite especificar varias opciones para resumir el contenido del documento. |
